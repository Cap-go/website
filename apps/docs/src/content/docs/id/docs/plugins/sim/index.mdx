Gunakan SIM Capgo untuk menanyakan data operator, indeks slot, dan status izin sambil tetap berada di dalam Capacitor.

<CardGrid stagger>
  <Card title="Dukungan multi-SIM" icon="rocket">
    Hitung setiap SIM yang terpasang beserta indeks slot dan ID langganan.
  </Card>
  <Card title="Pembantu izin" icon="approve-check">
    Periksa dan minta izin telepon yang diperlukan di Android.
  </Card>
  <Card title="Skema yang konsisten" icon="star">
    Bekerja dengan bidang operator yang dinormalisasi seperti MCC, MNC, negara ISO, dan nomor telepon.
  </Card>
  <Card title="iOS-aware" icon="pencil">
    Tangani nuansa platform seperti data operator yang terbatas di iOS 16.4+.
  </Card>
</CardGrid>

Dapatkan panduan memulai untuk mengintegrasikan izin dan pola konsumsi.
