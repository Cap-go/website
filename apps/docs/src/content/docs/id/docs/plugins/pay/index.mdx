Capgo Pembayaran menjembatani Apple Pembayaran terbaru dan Google Pembayaran SDKs sehingga Anda dapat memberikan satu pengalaman pembayaran yang lancar di seluruh perangkat.

<CardGrid stagger>
  <Card title="Terpadu API" icon="pencil">
    Panggil `isPayAvailable` dan `requestPayment` satu kali dan biarkan plugin menangani setiap platform.
  </Card>
  <Card title="Pengaturan cepat" icon="rocket">
    Kirimkan lembar pembayaran yang sepenuhnya asli tanpa memasang kode khusus platform secara manual.
  </Card>
  <Card title="Pembayaran yang diberi token" icon="approve-check">
    Terima token pembayaran yang siap untuk integrasi gateway atau PSP Anda.
  </Card>
  <Card title="Aliran yang dapat disesuaikan" icon="star">
    Konfigurasikan item baris, bidang kontak, dan persyaratan penagihan agar sesuai dengan pembayaran Anda.
  </Card>
</CardGrid>

Pelajari panduan memulai untuk daftar periksa penyiapan dan contoh payload untuk setiap platform.
