Capgo Pay 连接了最新的 Apple Pay 和 Google Pay SDKs，因此您可以跨设备提供一种无缝结账体验。

<CardGrid stagger>
  <Card title="统一API" icon="pencil">
    调用 `isPayAvailable` 和 `requestPayment` 一次，让插件处理每个平台。
  </Card>
  <Card title="快速设置" icon="rocket">
    发送完全原生的支付表，无需手动连接特定于平台的代码。
  </Card>
  <Card title="代币化支付" icon="approve-check">
    接收准备用于网关或 PSP 集成的支付令牌。
  </Card>
  <Card title="可定制的流程" icon="star">
    配置行项目、联系人字段和计费要求以适合您的结帐。
  </Card>
</CardGrid>

深入了解每个平台的设置清单和示例有效负载的入门指南。
