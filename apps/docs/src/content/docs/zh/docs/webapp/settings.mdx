## 如何进入设置页面

首先点击**您的姓名和图片**。然后点击**设置**。

<figure><img src="/settings-go.webp" alt="打开设置" /><figcaption></figcaption></figure>

## 更改用户设置

要更改任何用户设置，您只需填写帐户中的表格，然后单击“**更新**”。

<figure><img src="/account-save.webp" alt="保存帐户更改" /><figcaption></figcaption></figure>

然后应该会出现确认信息。

<figure><img src="/account-updated.webp" alt="帐户已更新" /><figcaption></figcaption></figure>

## 修改密码

要更改密码，请转到设置页面并单击“**密码**”。然后填写表格并单击“**更新**”。

<figure><img src="/update-passwd.webp" alt="更新密码" /><figcaption></figcaption></figure>

当密码不符合 capgo 密码安全规则时，您将收到错误消息。

<figure><img src="/passwd-error.webp" alt="密码错误" /><figcaption></figcaption></figure>
