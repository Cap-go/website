## 显示所有捆绑包

在 Capgo 中，捆绑包代表应用程序代码和资产的特定版本，可以分发到设备。首先，让我们看一下捆绑页面。您可以通过[单击您的应用程序](/docs/webapp/main-page)，然后[单击捆绑包选项卡](/docs/webapp/main-app-page) 来访问它。

<figure><img style="margin-left: auto; margin-right: auto" src="/bundles.webp" alt="捆绑清单" /><figcaption></figcaption></figure>

捆绑包列表显示：
- **名称**：捆绑版本号
- **创建于**：上传捆绑包时
- **频道**：捆绑包当前链接到哪个频道（如果有）
- **大小**：捆绑文件大小
- **操作**：快速操作，包括链接到频道（齿轮图标）和删除（垃圾桶图标）

您可以使用 **重新加载** 按钮刷新列表，**+ 添加** 上传新捆绑包，并使用 **过滤器** 缩小显示的捆绑包范围。搜索框允许您按名称查找捆绑包。

## 删除一个包

有两种方法可以删除捆绑包：

 - 通常情况下
 - 不安全

删除捆绑包的不安全方式已于 2024 年 8 月 12 日添加到 Capgo 中。

两种方式的区别在于删除后能否重新使用版本号。

例如，如果您以正常方式删除版本 `1.0.0`，然后尝试上传 `1.0.0` 版本，则会失败。
如果您通过不安全删除删除此版本，您将能够上传 `1.0.0` 版本。

:::danger
不安全地删除版本并重新上传是非常危险的。
它可能会导致插件中出现各种错误和不可预测的行为。
It should NEVER be used for bundles that have been used in a public channel.
正是这个原因，不安全地删除版本需要“super_admin”权限
:::

## 管理特定包

看到所有捆绑包的列表后，单击您要管理的捆绑包。执行此操作后，您应该会看到如下内容：

<figure><img style="margin-left: auto; margin-right: auto" src="/bundle-info.webp" alt="捆绑信息" /><figcaption></figcaption></figure>

捆绑包详细信息页面包含三个选项卡：

1. **信息**：显示所有包元数据和属性
2. **依赖项**：列出此捆绑包中包含的依赖项
3. **历史**：显示此捆绑包的更改历史记录

### 捆绑信息

信息选项卡显示以下详细信息：

- **捆绑包编号**：版本标识符（例如 12.87.1）
- **ID**：此捆绑包的唯一内部标识符
- **创建于**：首次上传捆绑包时
- **更新于**：上次修改捆绑包时
- **校验和**：捆绑包的完整性哈希（点击复制）
- **频道**：此捆绑包链接到的频道（单击以打开频道或更改它）
- **加密**：捆绑包是否加密
- **CLI 版本**：用于上传此捆绑包的 Capgo CLI 的版本
- **Zip应用程序包**：压缩包大小（点击下载）
- **清单**：清单文件大小
- **状态**：捆绑包是活动还是非活动（单击垃圾桶图标进行删除）

### 将捆绑包链接到频道

单击频道行旁边或操作列中的齿轮图标以打开频道链接模式：<figure><img style="margin-left: auto; margin-right: auto" src="/bundle-change.webp" alt="捆绑改变" /><figcaption></figcaption></figure>

该模式提供：
1. **搜索频道**：按名称过滤可用频道
2. **当前捆绑包**：显示正在链接的捆绑包版本
3. **可用频道**：所有频道及其详细信息的列表（ID、可见性、平台、创建日期）。标记为“当前”的频道表示该捆绑包已链接到的位置。公共频道显示链接图标。
4. **将捆绑包设置为频道**：确认选择并将该捆绑包分配给所选频道

**将捆绑包设置为通道：** 将此捆绑包分配为所选通道的活动版本。订阅该频道的设备将收到该捆绑包。

要打开频道的专用页面，请单击频道行中的频道名称链接。

要取消捆绑包与频道的链接，请导航至频道的配置页面。

### 下载捆绑包

要下载捆绑包，请单击“信息”选项卡中的 **Zip 应用程序捆绑包** 大小值。将出现一个确认对话框，确认后，捆绑包将直接下载到您的设备。
