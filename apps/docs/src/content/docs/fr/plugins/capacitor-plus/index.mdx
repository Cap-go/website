## Philosophie

L'équipe Ionic maintient Capacitor avec ses propres priorités et calendrier de sortie. Les contributions communautaires peuvent attendre des mois, voire des années, pour être fusionnées. Capacitor+ adopte une approche différente :

1. **Fusionner les PR de Forks** - Les PR précieux bloqués dans la file d'attente en amont sont activement fusionnés
2. **Synchronisation continue** - Chaque modification en amont est automatiquement extraite et testée
3. **Publications rapides** - Les modifications sont publiées sur npm dès qu'elles passent CI
4. **La communauté d'abord** - Vos contributions comptent et sont prioritaires
5. **Transparence** – Toute l'automatisation est open source et visible
