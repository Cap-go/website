Le placement de l’appel `CapacitorUpdater.notifyAppReady()` est crucial pour le bon déroulement de la mise à jour automatique.


Si ce n'est pas fait correctement, l'application ne pourra pas se mettre à jour vers la dernière version. Chaque version qui n'appelle pas `notifyAppReady()` dans les 10 secondes sera marquée comme invalide et sera remplacée par la version valide précédente ou la version par défaut (intégrée).


Dans ce guide, nous allons vous montrer comment placer correctement l'appel `notifyAppReady()` dans votre application pour garantir un flux de mise à jour automatique correct.
