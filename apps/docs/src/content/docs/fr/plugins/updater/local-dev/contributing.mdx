## Pourquoi contribuer ?

Tout d'abord, merci d'envisager de contribuer aux projets open source capgo ! Ce sont des gens comme vous qui font des projets open source Capgo de si bons outils.

Voici quelques raisons pour lesquelles vous pourriez vouloir contribuer :

* Contribuer aux projets open source de capgo est un excellent moyen de gagner de l'argent grâce aux [innombrables primes](https://console.algora.io/org/Capgo) proposées par l'équipe de capgo.
* Contribuer aux projets open source capgo est un excellent moyen d'ajouter une fonctionnalité que vous aimeriez voir
* Contribuer aux projets open source capgo est un excellent moyen de corriger un bug que vous avez rencontré
* [Licence principale Capgo](https://github.com/Cap-go/capgo/blob/main/LICENSE) vous oblige à ouvrir la source de toutes les modifications que vous y apportez. En contribuant votre code, vous conservez vos modifications open source et permettez aux autres de les utiliser

## Comment contribuer

 * Tout d'abord, vous devez forker le référentiel auquel vous souhaitez contribuer
 * Deuxièmement, vous devez valider et transmettre vos modifications à ce référentiel
 * Enfin, vous devez ouvrir une pull request
 *C'est ça ! Il ne vous reste plus qu'à attendre que l'équipe capgo examine votre PR
