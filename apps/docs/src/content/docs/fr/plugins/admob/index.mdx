import { Card, CardGrid } from '@astrojs/starlight/components';

Le plug-in AdMob de Capgo offre un pont de première classe vers les annonces mobiles Google SDK afin que vous puissiez monétiser les expériences natives sans sacrifier la simplicité de Capacitor.

<CardGrid stagger>
  <Card title="Formats d'annonces flexibles" icon="star">
    Diffusez des bannières, des interstitiels, des publicités avec récompense et des annonces ouvertes dans une application avec un API unifié.
  </Card>
  <Card title="Configuration de la demande" icon="setting">
    Affinez les demandes d'annonces avec des indicateurs destinés aux enfants, des évaluations de contenu et des paramètres réseau supplémentaires.
  </Card>
  <Card title="Gestion du cycle de vie" icon="rocket">
    Créez, chargez, affichez, masquez et détruisez des publicités à la demande tout en suivant l'état de chargement.
  </Card>
  <Card title="Privacy-ready" icon="approve-check">
    Gérez les invites ATT et surveillez l’état de l’autorisation de suivi pour rester conforme.
  </Card>
</CardGrid>

Utilisez le guide de démarrage pour connecter vos identifiants d'application AdMob, configurer le consentement et gérer l'inventaire publicitaire directement à partir de votre projet Capacitor.
