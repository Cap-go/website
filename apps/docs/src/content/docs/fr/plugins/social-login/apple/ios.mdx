Décomposons ce dont vous aurez besoin pour configurer la connexion Apple sur IOS.

<Steps>
1. Configurez les capacités de votre application.

    Pour ce faire, veuillez ouvrir Xcode, cliquer sur `App`
    ![Application XCode](/social-login-assets/app-xcode.png)

2. Assurez-vous de sélectionner la bonne cible.

    ![Cible de l'application XCode](/social-login-assets/xcode-app-target.png)

3. Assurez-vous d'ajouter la fonctionnalité `Sign in with Apple`.

    ![App XCode Ajouter une capacité](/social-login-assets/app-xcode-add-cap.png)

    ![App XCode Ajouter une capacité](/social-login-assets/sign-with-apple-cap.png)

    :::attention
    Si vous ne voyez pas la fonctionnalité `Sign in with Apple`, configurez le [Partage des données de compte et d'organisation](https://developer.apple.com/account/resources/services/cwa/configure/)
    :::

4. Initialisez la connexion Apple dans votre application.

    :::remarque
    J'utilise Vue comme framework, l'implémentation exacte variera en fonction du framework de votre choix
    :::

    ```ts
    //  onMounted is vue specific
    onMounted(() => {
    SocialLogin.initialize({
        apple: {}
    })
    });
    ```

5. Créez un bouton qui lancera le processus de connexion.

    Ce bouton doit appeler la fonction suivante :

    ```ts
    async function loginApple() {
    const res = await SocialLogin.login({
        provider: 'apple',
        options: {}
    })
    ```

6. Exécutez votre application sur un appareil **<u>PHYSICAL</u>** et testez-la. 

    Si vous avez suivi attentivement les étapes, vous verrez l'écran suivant après avoir cliqué sur votre bouton.

    ![Apple Invite de connexion sur l'appareil iOS](/social-login-assets/apple-sign-in-ios-final.png)
</Steps>

C'est tout ! Vous êtes prêt.
