import { Card, CardGrid } from '@astrojs/starlight/components';

Le plugin Media Session relie Capacitor à Media Session API afin que vos lecteurs audio et vidéo se sentent natifs partout.

<CardGrid stagger>
  <Card title="Des métadonnées riches" icon="star">
    Fournissez la pochette de l'album, l'artiste et le titre pour les commandes de verrouillage de l'écran et de notification.
  </Card>
  <Card title="Synchronisation de l'état de lecture" icon="rocket">
    Gardez les boutons lecture/pause et les indicateurs de progression alignés avec votre lecteur.
  </Card>
  <Card title="Rappels d'actions" icon="pencil">
    Gérez les boutons du casque et les actions multimédias du système d'exploitation à partir de votre code Capacitor.
  </Card>
  <Card title="Rapport de position" icon="download">
    Appuyez sur la durée, la position et le débit pour que le gommage fonctionne de manière fluide.
  </Card>
</CardGrid>

Accédez au guide de démarrage pour intégrer les mises à jour des métadonnées dans le cycle de vie de votre lecteur.
