## Utilisation

Importez le plugin et utilisez ses méthodes pour contrôler la lampe de poche :

```typescript
import { CapacitorFlash } from '@capgo/capacitor-flash';

// Check if flashlight is available
const checkFlashlight = async () => {
  const { value } = await CapacitorFlash.isAvailable();
  console.log('Flashlight available:', value);
};

// Turn on flashlight
const turnOn = async () => {
  await CapacitorFlash.switchOn();
};

// Turn off flashlight
const turnOff = async () => {
  await CapacitorFlash.switchOff();
};

// Check if flashlight is on
const checkStatus = async () => {
  const { value } = await CapacitorFlash.isSwitchedOn();
  console.log('Flashlight is on:', value);
};

// Toggle flashlight
const toggle = async () => {
  await CapacitorFlash.toggle();
};
```

## API Référence

### estDisponible()

Vérifie si la lampe de poche est disponible sur l'appareil.

```typescript
const result = await CapacitorFlash.isAvailable();
// Returns: { value: boolean }
```

### allumer (options ?)

Allume la lampe de poche.

```typescript
interface SwitchOnOptions {
  intensity?: number; // 0.0 to 1.0 (iOS only)
}

await CapacitorFlash.switchOn({ intensity: 0.5 });
```

### switchOff()

Éteint la lampe de poche.

```typescript
await CapacitorFlash.switchOff();
```

### estSwitchedOn()

Vérifie si la lampe de poche est actuellement allumée.

```typescript
const result = await CapacitorFlash.isSwitchedOn();
// Returns: { value: boolean }
```

### bascule()

Active/désactive la lampe de poche.

```typescript
await CapacitorFlash.toggle();
```

## Exemple complet

```typescript
import { CapacitorFlash } from '@capgo/capacitor-flash';

export class FlashlightService {
  private isOn = false;

  async init() {
    const { value } = await CapacitorFlash.isAvailable();
    if (!value) {
      throw new Error('Flashlight not available on this device');
    }
  }

  async toggle() {
    if (this.isOn) {
      await CapacitorFlash.switchOff();
    } else {
      await CapacitorFlash.switchOn();
    }
    this.isOn = !this.isOn;
  }

  async turnOn(intensity?: number) {
    await CapacitorFlash.switchOn({ intensity });
    this.isOn = true;
  }

  async turnOff() {
    await CapacitorFlash.switchOff();
    this.isOn = false;
  }

  async strobe(intervalMs: number = 100, duration: number = 3000) {
    const endTime = Date.now() + duration;
    
    while (Date.now() < endTime) {
      await CapacitorFlash.toggle();
      await new Promise(resolve => setTimeout(resolve, intervalMs));
    }
    
    // Ensure flashlight is off after strobe
    await CapacitorFlash.switchOff();
    this.isOn = false;
  }
}
```

## meilleures pratiques

1. **Vérifiez d'abord la disponibilité**
   Vérifiez toujours la disponibilité de la lampe de poche avant de l'utiliser :
   ```typescript
   const { value } = await CapacitorFlash.isAvailable();
   if (!value) {
     // Show alternative UI or disable flashlight features
   }
   ```

2. **Gérer les erreurs avec élégance**
   ```typescript
   try {
     await CapacitorFlash.switchOn();
   } catch (error) {
     console.error('Failed to turn on flashlight:', error);
   }
   ```

3. **Éteignez lorsque vous n'en avez pas besoin**
   Éteignez toujours la lampe de poche lorsque votre application passe en arrière-plan ou lorsque la fonctionnalité n'est plus nécessaire pour économiser la batterie.

4. **Évitez les basculements rapides**
   Mettez en œuvre une fonction anti-rebond pour éviter une commutation marche/arrêt rapide qui pourrait endommager la LED ou vider la batterie.

## Remarques sur la plate-forme

### iOS
- Nécessite iOS 10.0+
- Utilise le mode torche `AVCaptureDevice`
- Prend en charge le contrôle de l'intensité (0,0 à 1,0)

### Android
- Nécessite Android 6.0 (API 23)+
- Utilise Camera2 API
- Contrôle de l'intensité non pris en charge (utilise la luminosité par défaut)

###Web
- Pris en charge sur les navigateurs avec MediaDevices API et capacité de torche
- Utilise le flux de la caméra pour accéder à la lampe de poche de l'appareil
- Nécessite l'autorisation de l'utilisateur pour accéder à la caméra
- Fonctionne mieux sur les navigateurs mobiles (Chrome sur Android)
- Les navigateurs de bureau renvoient généralement `isAvailable: false` (pas de matériel de torche)
- Contrôle d'intensité non pris en charge sur le Web
