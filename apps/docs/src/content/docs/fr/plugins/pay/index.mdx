import { Card, CardGrid } from '@astrojs/starlight/components';

Capgo Pay relie les dernières versions de Apple Pay et Google Pay SDKs afin que vous puissiez offrir une expérience de paiement transparente sur tous les appareils.

<CardGrid stagger>
  <Card title="API unifié" icon="pencil">
    Appelez `isPayAvailable` et `requestPayment` une fois et laissez le plugin gérer chaque plate-forme.
  </Card>
  <Card title="Installation rapide" icon="rocket">
    Expédiez des feuilles de paiement entièrement natives sans câbler manuellement le code spécifique à la plateforme.
  </Card>
  <Card title="Paiements tokenisés" icon="approve-check">
    Recevez des jetons de paiement prêts pour votre passerelle ou vos intégrations PSP.
  </Card>
  <Card title="Flux personnalisables" icon="star">
    Configurez les éléments de campagne, les champs de contact et les exigences de facturation en fonction de votre paiement.
  </Card>
</CardGrid>

Plongez dans le guide de démarrage pour découvrir les listes de contrôle de configuration et des exemples de charges utiles pour chaque plate-forme.
