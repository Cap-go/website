Maintenant que vous avez terminé le guide de démarrage rapide, vous devriez avoir une compréhension de base des concepts clés de Capgo ! Les concepts clés que vous avez appris dans ce guide sont :

<Steps>
1. Ajouter une application à votre compte Capgo

2. Intégration de Capgo à votre pipeline CI/CD 

3. Déclenchement des téléchargements de bundles vers Capgo lors de nouveaux commits

4. Configuration de votre application pour activer les mises à jour en direct avec le Capgo SDK

5. Déployer des mises à jour en direct sur votre application à partir du tableau de bord Capgo
</Steps>
