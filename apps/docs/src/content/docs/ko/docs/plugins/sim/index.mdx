import { Card, CardGrid } from '@astrojs/starlight/components';

Capgo SIM을 사용하여 Capacitor 내부에 있는 동안 이동통신사 데이터, 슬롯 인덱스 및 권한 상태를 쿼리합니다.

<CardGrid stagger>
  <Card title="다중 SIM 지원" icon="rocket">
    슬롯 색인 및 구독 ID와 함께 설치된 모든 SIM을 열거합니다.
  </Card>
  <Card title="권한 도우미" icon="approve-check">
    Android에 필요한 전화 통신 권한을 확인하고 요청하세요.
  </Card>
  <Card title="일관된 스키마" icon="star">
    MCC, MNC, ISO 국가 및 전화번호와 같은 정규화된 이동통신사 필드로 작업합니다.
  </Card>
  <Card title="iOS-aware" icon="pencil">
    iOS 16.4+에서 제한된 이동통신사 데이터와 같은 플랫폼 미묘한 차이를 처리합니다.
  </Card>
</CardGrid>

권한과 소비 패턴을 통합하려면 시작 가이드를 확인하세요.
