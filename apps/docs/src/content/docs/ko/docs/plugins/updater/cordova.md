이 플러그인을 Cordova에서 사용할 수 있을지 궁금하셨을 겁니다.

우리는 이를 위해 R\&D 저장소를 시작했지만 엄청난 양의 작업이 필요합니다.

## 문제

우리는 그렇게 할 수 있다는 것을 알고 있지만 이를 위해서는 Capacitor에서 했던 것처럼 Cordova 코드베이스의 모든 코드를 읽어서 ap Capgo 기능과 함께 작동하도록 만드는 방법을 이해해야 합니다.

Android 버전은 둘 다 Java를 사용하므로 수행하기가 더 쉽지만 iOS은 Swift가 아직 Cordova에서 잘 지원되지 않기 때문에 완전히 다시 작성해야 합니다.
