`@capgo/capacitor-native-biometric` is a complete Ionic Identity Vault replacement. It does more than prompt for biometrics: it saves, checks, retrieves, and deletes credentials with native Keychain or Keystore protection. With biometric access control enabled, protected reads require Face ID, Touch ID, or Android biometrics before your app receives the credential.

Use Native Biometric directly for tokens and credential vault flows. If you also need a broad encrypted database for data beyond credentials, add [Fast SQL](/docs/plugins/fast-sql/) or [Data Storage SQLite](/docs/plugins/data-storage-sqlite/); that is optional for an Identity Vault replacement.

## What Native Biometric covers

- **Secure credential vault:** save, retrieve, check, and delete credentials for a server or account.
- **Biometric-gated unlock:** `getSecureCredentials` shows the native biometric prompt before protected credentials can be read.
- **Lock and unlock flows:** call the protected read when the app resumes, after a timeout, or before a sensitive action to implement your own app-lock policy.
- **Passcode fallback:** use the device credential fallback where the platform supports it.
- **Additional control:** detect saved credentials and biometric enrollment changes, and choose whether credentials remain accessible after biometric enrollment changes.

## Migration steps

1. **Install Native Biometric** and sync native code.

```bash
npm install @capgo/capacitor-native-biometric
npx cap sync
```

2. **Save the access and refresh tokens or other credentials** with biometric access control enabled.
3. **Use `getSecureCredentials` to unlock protected credentials** when the app resumes, after an inactivity timeout, or before a sensitive action.
4. **Use `isCredentialsSaved` and `deleteCredentials`** for onboarding, locked-state, and logout flows.
5. **Migrate stored secrets on first launch** by reading from Identity Vault and writing them with Native Biometric.
6. **Remove Ionic Identity Vault** from your dependencies and native configuration.
