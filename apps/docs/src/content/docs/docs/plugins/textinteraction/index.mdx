## Overview

Toggle text interaction in Capacitor based iOS apps.

## Core Capabilities

- `toggle` - Toggle text interaction (selection) on the Capacitor WebView.

## Public API

| Method | Description |
| --- | --- |
| `toggle` | Toggle text interaction (selection) on the Capacitor WebView. |
| `getPluginVersion` | Get the native Capacitor plugin version. |
