## Overview

Capacitor Mute Plugin for detecting device mute status.

## Core Capabilities

- `isMuted` - Check if the device mute switch is enabled.

## Public API

| Method | Description |
| --- | --- |
| `isMuted` | Check if the device mute switch is enabled. |
| `getPluginVersion` | Get the native Capacitor plugin version. |
