## Overview

Capacitor Share Target Plugin interface.

## Core Capabilities

- `addListener` - Listen for shareReceived event.
- `removeAllListeners` - Remove all listeners for this plugin.
- `getPluginVersion` - Get the native Capacitor plugin version.
