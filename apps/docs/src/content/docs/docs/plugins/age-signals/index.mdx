## Overview

Capacitor interface for retrieving Play Age Signals.

## Core Capabilities

- `checkAgeSignals` - Request the current Play Age Signals for the active user.

## Public API

| Method | Description |
| --- | --- |
| `checkAgeSignals` | Request the current Play Age Signals for the active user. |
| `getPluginVersion` | Get the native Capacitor plugin version. |
