## Overview

Capacitor API for inspecting SSL pinning configuration.

## Core Capabilities

- `getConfiguration` - Returns the active native configuration visible to the plugin.

## Public API

| Method | Description |
| --- | --- |
| `getConfiguration` | Returns the active native configuration visible to the plugin. |
| `getPluginVersion` | Returns the native implementation version marker. |
