import { Card, CardGrid } from '@astrojs/starlight/components';

Utilizza Capgo SIM per interrogare i dati dell'operatore, gli indici degli slot e gli stati delle autorizzazioni rimanendo all'interno di Capacitor.

<CardGrid stagger>
  <Card title="Supporto multi-SIM" icon="rocket">
    Enumerare ogni SIM installata insieme agli indici degli slot e agli ID di abbonamento.
  </Card>
  <Card title="Aiutanti per i permessi" icon="approve-check">
    Controlla e richiedi l'autorizzazione telefonica richiesta su Android.
  </Card>
  <Card title="Schema coerente" icon="star">
    Utilizza campi dell'operatore normalizzati come MCC, MNC, paese ISO e numeri di telefono.
  </Card>
  <Card title="iOS-aware" icon="pencil">
    Gestisci le sfumature della piattaforma come i dati limitati dell'operatore su iOS 16.4+.
  </Card>
</CardGrid>

Ottieni la guida introduttiva per integrare autorizzazioni e modelli di consumo.
