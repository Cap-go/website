## Utilizzo di base

```typescript
import { CapacitorVideoPlayer } from '@capgo/capacitor-video-player';

// Initialize a video player
const playVideo = async () => {
  const result = await CapacitorVideoPlayer.initPlayer({
    mode: 'fullscreen',
    url: 'https://example.com/video.mp4',
    playerId: 'myPlayer',
    componentTag: 'div'
  });

  console.log('Player initialized:', result);
};

// Play the video
await CapacitorVideoPlayer.play({ playerId: 'myPlayer' });

// Pause the video
await CapacitorVideoPlayer.pause({ playerId: 'myPlayer' });

// Get current time
const { value } = await CapacitorVideoPlayer.getCurrentTime({ playerId: 'myPlayer' });
console.log('Current time:', value);

// Seek to position
await CapacitorVideoPlayer.setCurrentTime({
  playerId: 'myPlayer',
  seektime: 30 // seconds
});

// Set volume (0-1)
await CapacitorVideoPlayer.setVolume({
  playerId: 'myPlayer',
  volume: 0.5
});
```

## API Riferimento

### initPlayer(opzioni)

Inizializza un'istanza del lettore video.

```typescript
await CapacitorVideoPlayer.initPlayer({
  mode: 'fullscreen', // or 'embedded'
  url: 'https://example.com/video.mp4',
  playerId: 'player1',
  subtitle: 'https://example.com/subtitles.vtt',
  language: 'en',
  rate: 1.0,
  exitOnEnd: true,
  loopOnEnd: false,
  pipEnabled: true,
  showControls: true
});
```

### gioca(opzioni)

Riproduci il video.

```typescript
await CapacitorVideoPlayer.play({ playerId: 'player1' });
```

### pausa(opzioni)

Metti in pausa il video.

```typescript
await CapacitorVideoPlayer.pause({ playerId: 'player1' });
```

### getDuration(opzioni)

Ottieni la durata del video in secondi.

```typescript
const { value } = await CapacitorVideoPlayer.getDuration({ playerId: 'player1' });
console.log('Duration:', value, 'seconds');
```

### getCurrentTime(opzioni)

Ottieni la posizione di riproduzione corrente in pochi secondi.

```typescript
const { value } = await CapacitorVideoPlayer.getCurrentTime({ playerId: 'player1' });
```

### setCurrentTime(opzioni)

Cerca un momento specifico.

```typescript
await CapacitorVideoPlayer.setCurrentTime({
  playerId: 'player1',
  seektime: 60 // seconds
});
```

### impostaVolume(opzioni)

Imposta il volume (da 0,0 a 1,0).

```typescript
await CapacitorVideoPlayer.setVolume({
  playerId: 'player1',
  volume: 0.8
});
```

### getVolume(opzioni)

Ottieni il volume corrente.

```typescript
const { value } = await CapacitorVideoPlayer.getVolume({ playerId: 'player1' });
```

### setMuted(opzioni)

Disattiva o attiva l'audio del video.

```typescript
await CapacitorVideoPlayer.setMuted({
  playerId: 'player1',
  muted: true
});
```

### setRate(opzioni)

Imposta la velocità di riproduzione.

```typescript
await CapacitorVideoPlayer.setRate({
  playerId: 'player1',
  rate: 1.5 // 1.5x speed
});
```

### stopAllPlayers()

Ferma tutti i giocatori attivi.

```typescript
await CapacitorVideoPlayer.stopAllPlayers();
```

### esciGiocatore()

Esci dal lettore video.

```typescript
await CapacitorVideoPlayer.exitPlayer();
```

## Esempio completo

```typescript
import { CapacitorVideoPlayer } from '@capgo/capacitor-video-player';

export class VideoPlayerService {
  private playerId = 'mainPlayer';

  async initializePlayer(videoUrl: string, subtitleUrl?: string) {
    try {
      const result = await CapacitorVideoPlayer.initPlayer({
        mode: 'fullscreen',
        url: videoUrl,
        playerId: this.playerId,
        subtitle: subtitleUrl,
        language: 'en',
        rate: 1.0,
        exitOnEnd: true,
        loopOnEnd: false,
        pipEnabled: true,
        bkmodeEnabled: true,
        showControls: true,
        displayMode: 'all'
      });

      console.log('Player initialized:', result);
      return result;
    } catch (error) {
      console.error('Failed to initialize player:', error);
      throw error;
    }
  }

  async togglePlayPause() {
    const { value: isPlaying } = await CapacitorVideoPlayer.isPlaying({
      playerId: this.playerId
    });

    if (isPlaying) {
      await CapacitorVideoPlayer.pause({ playerId: this.playerId });
    } else {
      await CapacitorVideoPlayer.play({ playerId: this.playerId });
    }
  }

  async seekForward(seconds: number = 10) {
    const { value: currentTime } = await CapacitorVideoPlayer.getCurrentTime({
      playerId: this.playerId
    });

    await CapacitorVideoPlayer.setCurrentTime({
      playerId: this.playerId,
      seektime: currentTime + seconds
    });
  }

  async seekBackward(seconds: number = 10) {
    const { value: currentTime } = await CapacitorVideoPlayer.getCurrentTime({
      playerId: this.playerId
    });

    await CapacitorVideoPlayer.setCurrentTime({
      playerId: this.playerId,
      seektime: Math.max(0, currentTime - seconds)
    });
  }

  async setPlaybackSpeed(speed: number) {
    await CapacitorVideoPlayer.setRate({
      playerId: this.playerId,
      rate: speed
    });
  }

  async getProgress() {
    const { value: currentTime } = await CapacitorVideoPlayer.getCurrentTime({
      playerId: this.playerId
    });
    const { value: duration } = await CapacitorVideoPlayer.getDuration({
      playerId: this.playerId
    });

    return {
      currentTime,
      duration,
      percentage: (currentTime / duration) * 100
    };
  }

  async cleanup() {
    await CapacitorVideoPlayer.stopAllPlayers();
  }
}
```

## Note sulla piattaforma

### iOS
- Richiede iOS 10.0+
- Utilizza AVPlayer nativo
- Supporta l'immagine nell'immagine
- Supporta la riproduzione in background

### Android
- Richiede Android 5.0 (API 21)+
- Utilizza ExoPlayer
- Supporta Chromecast
- Colori accentati personalizzati disponibili

### Rete
- Utilizza il lettore video HTML5
- Solo modalità incorporata
- Funzionalità native limitate

## Migliori pratiche

1. **Pulizia dei giocatori**: ferma sempre i giocatori quando hai finito
2. **Gestione degli errori**: racchiude le chiamate del giocatore in try-catch
3. **ID giocatore univoci**: utilizza ID univoci per più giocatori
4. **Controlla lo stato della riproduzione**: verifica lo stato prima delle operazioni
5. **Gestione delle risorse**: rilascia i giocatori per liberare memoria
