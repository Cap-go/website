## CI 集成

为了自动化您的工作，我们建议使用 GitHub Actions 将您的更新推送到 Capgo。查看我们的 [GitHub Actions 教程](https://capgo.app/blog/automatic-build-and-release-with-github-actions/)以获取更多信息。

不要忘记使用您的 Capgo API 密钥配置您的 CI 环境变量。
