## 这是什么?

Capgo 拥有广泛的 Web 应用来帮助您管理项目。此 Web 应用使用 Vue.js 构建并且是开源的。您可以在 [GitHub](https://github.com/Cap-go/capgo/tree/main/src/) 上找到源代码 \
该 Web 应用可在[此处](https://console.capgo.app/)访问。\
此 Web 应用允许您[管理频道](/docs/webapp/channels/)、[管理版本](/docs/webapp/bundles/)、[管理设备](/docs/webapp/devices/)、[检查日志](/docs/webapp/logs/)、[管理计费](/docs/webapp/settings/)和[管理您的帐户](/docs/webapp/settings/)。

## 如何使用它?

首先,您需要创建帐户或登录。这相对简单,可以通过单击屏幕中央的 `登录` 按钮来完成。登录后,您将被重定向到仪表板。这是 Web 应用的主页。从这里您可以导航到所有其他页面。

## 创建帐户

您必须单击登录表单底部的 `创建免费帐户`。然后,只需填写表单并按照说明操作即可。
