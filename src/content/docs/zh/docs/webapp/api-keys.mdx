## 我可以用 API 密钥做什么?

API 密钥可以重新生成、删除或添加新密钥。

## 如何查看所有 API 密钥?

您需要转到 [API 密钥页面](https://console.capgo.app/dashboard/apikeys/),在那里您将看到所有 API 密钥。

## 如何添加新的 API 密钥?

要添加新的 API 密钥,请单击小加号按钮

<figure><img src="/apikeys-add.webp" alt="添加 API 密钥" /><figcaption></figcaption></figure>

然后选择要授予新 API 密钥的权限。您可以选择:

 - 读取 - API 密钥将能够读取所有数据
 - 上传 - API 密钥将能够从 CLI 上传新版本并读取
 - 全部 - API 密钥将能够执行所有操作

<figure><img src="/apikeys-select-perm.webp" alt="选择权限" /><figcaption></figcaption></figure>

## 如何删除 API 密钥?

要删除 API 密钥,请单击小垃圾桶按钮。

<figure><img src="/apikeys-remove.webp" alt="删除 API 密钥" /><figcaption></figcaption></figure>

然后确认删除 API 密钥。

## 如何重新生成 API 密钥?

要重新生成 API 密钥,请单击小刷新按钮。

<figure><img src="/apikeys-regenerate.webp" alt="生成 API 密钥" /><figcaption></figcaption></figure>

然后确认重新生成 API 密钥。
