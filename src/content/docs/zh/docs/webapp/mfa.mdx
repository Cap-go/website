## How to login using 2FA

Next time you login to your account, you will see a window like this
<figure><img style="margin-left: auto; margin-right: auto" src="/mfa-required.webp" alt="MFA required" /><figcaption></figcaption></figure>

Please open the authenticator and copy your auth code

<figure><img style="margin-left: auto; margin-right: auto" src="/mfa-login.webp" alt="MFA login" /><figcaption></figcaption></figure>

:::danger
âš ï¸ Important Note: \
Press `verify` before the code refresh, otherwise the code will change, and the old one will no longer be valid
:::

Then input the code, into the login form and pres the `verify`

<figure><img style="margin-left: auto; margin-right: auto" src="/mfa-final-login-verify.webp" alt="MFA login" /><figcaption></figcaption></figure>

If you entered the correct code you should see the capgo dashboard

## How to disable 2FA

To disable 2FA please go to the [capgo's account settings](https://console.capgo.app/dashboard/settings/account/). There you should see a red button that should look like this:

<figure><img style="margin-left: auto; margin-right: auto" src="/mfa-disable-button.webp" alt="MFA disable button" /><figcaption></figcaption></figure>

Click it, and and you should see a screen like this

<figure><img style="margin-left: auto; margin-right: auto" src="/mfa-disable-2.webp" alt="MFA disabled" /><figcaption></figcaption></figure>

Just press the `disable` button, and that's it.
