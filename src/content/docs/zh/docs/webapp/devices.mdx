:::note[数据保留]
设备数据从最后一次设备活动起保留 **90 天**。在此期间内未连接到 Capgo 的设备将自动从列表中删除。活动设备在检查更新时会持续被跟踪。
:::

## 显示所有设备列表
首先，让我们看一下设备页面。您可以通过[点击您的应用](/docs/webapp/main-page)然后[点击设备标签](/docs/webapp/main-app-page)来访问它。

<figure><img style="margin-left: auto; margin-right: auto" src="/devices.webp" alt="设备列表" /><figcaption></figcaption></figure>

1. **设备标签** - 点击导航栏中的此标签以访问设备页面
2. **过滤器** - 点击按以下条件过滤设备：
   - **覆盖** - 仅显示具有[自定义频道](/docs/plugin/api/#setchannel)或自定义版本的设备
   - **自定义 ID** - 仅显示设置了自定义标识符的设备

## 配置设备

要配置特定设备，请在表格中点击它，然后您应该看到类似这样的内容：

<figure><img style="margin-left: auto; margin-right: auto" src="/device-specific.webp" alt="显示一个设备" /><figcaption></figcaption></figure>

1. **Information** - 查看设备详情，包括设备 ID、最后更新、平台、插件版本、版本、内置版本、操作系统版本，以及是否为生产应用
2. **Deployments** - 查看此设备的部署历史
3. **Logs** - 查看与此设备相关的日志
4. **频道覆盖** - 为此特定设备覆盖频道。如果设置，此设备将不使用公共（默认）频道，只接收所选频道的更新

### 自定义 ID

`自定义 ID` 是一个非常有用的字段，可帮助您将设备连接到您可以为自己的用户识别的 ID。

使用它可以轻松识别哪个设备属于哪个用户。

自定义 ID 只能从设备本身使用 `CapacitorUpdater.setCustomId({ customId: user });` 设置。
