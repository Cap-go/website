:::note[数据保留]
设备数据从最后一次设备活动起保留 **90 天**。在此期间内未连接到 Capgo 的设备将自动从列表中删除。活动设备在检查更新时会持续被跟踪。
:::

## 配置设备

要配置特定设备,请在表格中点击它,然后您应该看到类似这样的内容:

<figure><img style="margin-left: auto; margin-right: auto" src="/device-specific.webp" alt="显示一个设备" /><figcaption></figcaption></figure>

首先是 `自定义 ID`。这是一个非常有用的字段,可帮助您将设备连接到您可以为自己的用户识别的 ID。

使用它可以轻松识别哪个设备属于哪个用户。

您可以从 WebUI 或使用 `CapacitorUpdater.setCustomId({ customId: user });` 从设备设置它

接下来是强制版本。如果设置了此项,则此设备将**始终**接收此版本。它优先于频道。

最后是自定义频道。如果设置了此项,则此设备将不关心公共(默认)频道,只使用这个频道。
