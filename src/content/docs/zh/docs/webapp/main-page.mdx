## 主页面显示什么?

首先,让我们看一下主页面:
<figure><img src="/main-page.webp" alt="主页面截图" /><figcaption></figcaption></figure>

让我们从头开始。您看到的第一件事是**统计**。它包含特定于应用的统计信息。这些更新取决于应用的使用情况

您看到的第二件事是所有可用应用的列表。这将根据您创建的应用数量而变化。您可以单击每个应用以查看有关它的更多详细信息。

您看到的第三件事是 **API 密钥**按钮。此按钮将带您到 [API 密钥](/docs/webapp/api-keys/)页面。

接下来您看到的是 **测试 Capgo** 按钮。此按钮将根据您的名字和姓氏而变化,它将带您到[设置](/docs/webapp/settings/)或支持页面。

最后,您会看到一个添加另一个应用的按钮。此按钮将带您到一个页面,您可以在其中向 Capgo 添加新应用。
