## 插件对比

Capacitor 应用中常用的三个主要后台地理位置插件的简短对比。

| 插件 | 准确性 | 后台 | HTTP 上传 | 定价 |
|--------|----------|------------|-------------|---------|
| `@capacitor-community/background-geolocation`（社区版） | 不准确 | 是 | 否 | 免费 |
| `@capgo/background-geolocation`（本插件） | 准确 | 是 | 否 | 免费 |
| Transistorsoft（原版） | 准确 | 是 | 是 — 内置 HTTP 上传器到您的 API | 付费 |

**注意：**
- 社区插件轻量级且可在后台继续工作，但已知不如以下选项准确。
- 此 Capgo 插件经过优化，可提供准确的位置修复和可靠的后台操作，无需付费许可证。
- Transistorsoft 提供完整的商业解决方案，还包括用于将位置数据发送到后端的内置 HTTP 上传功能。
