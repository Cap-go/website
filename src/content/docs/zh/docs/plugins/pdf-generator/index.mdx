使用 Capgo 的 iOS 和 Android 原生渲染器将 HTML 或远程页面转换为可共享的 PDF。

<CardGrid stagger>
  <Card title="HTML 转 PDF" icon="pencil">
    输入 HTML 模板，在几秒钟内获得一个可共享的 PDF 文档。
  </Card>
  <Card title="远程捕获" icon="rocket">
    快照任何可访问的 URL 并保存或分发生成的 PDF。
  </Card>
  <Card title="灵活的输出" icon="star">
    根据您的工作流程返回 base64 数据或触发原生分享表。
  </Card>
  <Card title="文档控制" icon="setting">
    为每次渲染指定方向、页面大小和文件名。
  </Card>
</CardGrid>
