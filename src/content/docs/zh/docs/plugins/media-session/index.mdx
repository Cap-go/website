Media Session 插件将 Capacitor 与 Media Session API 桥接，使您的音频和视频播放器在任何地方都感觉原生。

<CardGrid stagger>
  <Card title="丰富的元数据" icon="star">
    为锁屏和通知控件提供专辑封面、艺术家和标题。
  </Card>
  <Card title="播放状态同步" icon="rocket">
    保持播放/暂停按钮和进度指示器与您的播放器对齐。
  </Card>
  <Card title="操作回调" icon="pencil">
    从 Capacitor 代码处理耳机按钮和操作系统媒体操作。
  </Card>
  <Card title="位置报告" icon="download">
    推送持续时间、位置和速率，以便流畅地进行滑动。
  </Card>
</CardGrid>
