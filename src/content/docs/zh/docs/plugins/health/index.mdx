Capgo Health 插件统一了 HealthKit 和 Health Connect 访问，因此您可以使用一个 Capacitor 集成来跟踪活动、生物识别指标和身体测量。

<CardGrid stagger>
  <Card title="跨平台健康数据" icon="rocket">
    开箱即用支持 Apple HealthKit 和 Android Health Connect。
  </Card>
  <Card title="细粒度权限" icon="approve-check">
    请求单独的读/写范围并响应授权更改。
  </Card>
  <Card title="一致的单位" icon="star">
    使用标准化的数据类型，如 `steps`、`heartRate` 和 `weight`。
  </Card>
  <Card title="实时写入" icon="rocket">
    当用户记录活动时，将样本保存回用户的健康存储。
  </Card>
</CardGrid>

按照入门指南启用平台功能、收集同意并开始同步指标。
