## 为什么要贡献？

首先，感谢您考虑为 capgo 开源项目做出贡献！正是像您这样的人让 capgo 开源项目成为如此优秀的工具。

以下是您可能想要考虑贡献的一些原因：

* 为 capgo 开源项目做出贡献是从 capgo 团队提供的[众多悬赏](https://console.algora.io/org/Capgo)中赚取收入的好方法
* 为 capgo 开源项目做出贡献是添加您想要看到的功能的好方法
* 为 capgo 开源项目做出贡献是修复您遇到的错误的好方法
* [Capgo 主许可证](https://github.com/Cap-go/capgo/blob/main/LICENSE)要求您开源您对其所做的任何更改。通过贡献您的代码，您可以保持您的更改开源并让其他人使用它

## 如何贡献

 * 首先，您需要 fork 您想要贡献的仓库
 * 其次，您需要提交并推送您的更改到该仓库
 * 最后，您需要打开一个 pull request
 * 就是这样！现在您只需要等待 capgo 团队审查您的 PR
