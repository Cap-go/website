## 概述

Capacitor Downloader 插件提供强大的文件下载功能，支持后台下载、进度跟踪和全面的下载管理。此插件可实现稳健的文件下载，具有暂停/恢复功能和详细的进度监控。

> ⚠️ **正在开发中**: 此插件目前正在开发中，尚未准备好用于生产环境。
