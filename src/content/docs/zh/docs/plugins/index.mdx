## ⭐ Capgo Cloud - 实时更新

<LinkCard
  title="Capacitor Updater"
  description="支持 Capgo Cloud 的核心插件 - 无需应用商店延迟即可向您的 Capacitor 应用提供即时更新。直接将修复和功能推送给您的用户。"

Updater 插件是 Capgo Cloud 的基础，使您能够：
- 🚀 无需应用商店审核即可立即部署更新
- 📱 更新您应用的 JavaScript、HTML、CSS 和资源
- 🎯 使用频道针对特定用户群体
- 📊 使用内置分析监控更新成功率
- 🔒 使用加密和代码签名保护更新

## 🚀 开始使用

每个插件都遵循类似的安装模式：

1. **使用您首选的包管理器安装插件**
2. **使用 `npx cap sync` 同步您的项目**
3. **如果需要，配置平台特定的设置**
4. **在您的代码中开始使用插件**

访问每个插件的文档以获取详细的设置说明和 API 参考。
