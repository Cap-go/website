## 您将获得什么

所有集成指南包括:

- **简单设置**:快速入门的基本配置
- **高级工作流**:包含预发布和生产环境的多环境部署
- **功能分支测试**:自动将功能分支部署到测试渠道
- **安全最佳实践**:安全的密钥管理和环境保护
- **监控**:部署状态的通知和日志记录

<Aside type="tip">

**初次接触 CI/CD?** 从适合您平台的简单配置开始,然后随着需求的增长逐步添加更高级的功能,如多渠道部署和自动化测试。

</Aside>

## 通用功能

每个集成都支持:

- **自动化构建**:在代码更改时触发部署
- **多渠道支持**:部署到不同的渠道(开发、预发布、生产)
- **拉取请求/合并请求测试**:在隔离环境中测试更改
- **加密支持**:使用 Capgo 的加密功能进行安全部署
- **环境保护**:生产环境的人工批准和受限访问
- **通知**:Slack、邮件和其他通知集成

## 前置要求

在设置任何集成之前,请确保您具备:

- 已配置应用的 Capgo 账户
- Git 存储库中的应用源代码
- 来自 [console.capgo.app/apikeys](https://console.capgo.app/apikeys) 的 Capgo API token
- 项目中已配置 Node.js 和 npm/yarn
