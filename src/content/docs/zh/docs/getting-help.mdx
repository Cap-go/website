## 通过 Discord 获取支持

Capgo 有一个官方的 [Discord 服务器](https://discord.capgo.app)。在那里获取技术支持可能是获得响应的最快方式之一。

以下是速成课程：

步骤 1 - 前往 `questions` 频道

<figure><img style="margin-left: auto; margin-right: auto" src="/discord-questions.webp" alt="在 discord 上提问" /><figcaption></figcaption></figure>

步骤 2 - 创建您的主题

<figure><img style="margin-left: auto; margin-right: auto" src="/discord-newquestion.webp" alt="在 discord 上创建问题" /><figcaption></figcaption></figure>

步骤 3 - 描述您的问题并选择相关标签

<figure><img style="margin-left: auto; margin-right: auto" src="/discord-new-post.webp" alt="在 discord 上创建帖子" /><figcaption></figcaption></figure>


步骤 4 - 分享您的安全账户 ID（可选）

这将允许 Capgo 工作人员查看您的账户。分享此 ID 是安全的，因为它被设计为可以公开分享。

要分享此内容，请前往 [Capgo 的设置](https://console.capgo.app/dashboard/settings/account/)。在那里请点击 `copy account id`。

<figure><img style="margin-left: auto; margin-right: auto" src="/share-secure-id.webp" alt="分享您的 ID 而不泄露您的信息" /><figcaption></figcaption></figure>

这将把安全账户 ID 复制到剪贴板。请在您的 Discord 帖子中包含它。

## 通过电子邮件获取支持

这是获取支持的最慢方式。请首先使用 Discord 服务器。

如果您需要通过电子邮件联系我们，请发送电子邮件至 support@capgo.app。
