Ahora que has completado la guía de inicio rápido, deberías tener una comprensión básica de los conceptos clave de Capgo. Los conceptos clave que has aprendido en esta guía son:

<Steps>
1. Agregar una aplicación a tu cuenta de Capgo

2. Integrar Capgo con tu pipeline de CI/CD

3. Activar las subidas de paquetes a Capgo con nuevos commits

4. Configurar tu aplicación para habilitar actualizaciones en vivo con el SDK de Capgo

5. Desplegar actualizaciones en vivo a tu aplicación desde el panel de control de Capgo
</Steps>
