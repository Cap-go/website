Ahora que has completado la guía de inicio rápido, debes tener una comprensión básica de los conceptos clave de Capgo. Los conceptos clave que has aprendido en esta guía son:

<Steps>
1. Añadir una aplicación a tu cuenta de Capgo

2. Integrar Capgo con tu pipeline de CI/CD

3. Activar cargas de bundles a Capgo en nuevos commits

4. Configurar tu aplicación para habilitar actualizaciones en vivo con el SDK de Capgo

5. Implementar actualizaciones en vivo en tu aplicación desde el panel de Capgo
</Steps>
