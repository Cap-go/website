:::note[Retención de Datos]
Los datos de dispositivos se conservan durante **90 días** desde la última actividad del dispositivo. Los dispositivos que no se hayan conectado a Capgo dentro de este período se eliminarán automáticamente de la lista. Los dispositivos activos se rastrean continuamente mientras verifican actualizaciones.
:::

## Mostrar la lista de todos los dispositivos
Primero, echemos un vistazo a la página de dispositivos. Puedes acceder a ella [haciendo clic en tu aplicación](/docs/webapp/main-page) y luego [haciendo clic en la pestaña de dispositivos](/docs/webapp/main-app-page).

<figure><img style="margin-left: auto; margin-right: auto" src="/devices.webp" alt="lista de dispositivos" /><figcaption></figcaption></figure>

1. **Pestaña de Dispositivos** - Haz clic en esta pestaña en la barra de navegación para acceder a la página de dispositivos
2. **Filtros** - Haz clic para filtrar dispositivos por:
   - **Sobrescribir** - Muestra solo dispositivos que tienen un [canal personalizado](/docs/plugin/api/#setchannel) o una versión personalizada
   - **ID Personalizado** - Muestra solo dispositivos con un identificador personalizado establecido

## Configurando un dispositivo

Para configurar un dispositivo específico, haz clic en él en la tabla y deberías ver algo como esto:

<figure><img style="margin-left: auto; margin-right: auto" src="/device-specific.webp" alt="mostrar un dispositivo" /><figcaption></figcaption></figure>

1. **Información** - Ver detalles del dispositivo incluyendo ID del Dispositivo, Última actualización, Plataforma, Versión del plugin, Versión, Versión builtin, Versión del SO, y si es una aplicación de Producción
2. **Deployments** - Ver el historial de despliegues para este dispositivo
3. **Logs** - Ver los registros asociados con este dispositivo
4. **Sobrescritura de canal** - Sobrescribe el canal para este dispositivo específico. Si está configurado, este dispositivo no usará el canal público (predeterminado) y solo recibirá actualizaciones del canal seleccionado

### ID Personalizado

El `ID Personalizado` es un campo muy útil que te ayuda a conectar el dispositivo con un ID que puedas reconocer para tus propios usuarios.

Úsalo para identificar fácilmente qué dispositivo pertenece a qué usuario.

El ID personalizado solo puede establecerse desde el propio dispositivo usando `CapacitorUpdater.setCustomId({ customId: user });`
