## ¿Qué puedo hacer con las claves api?

Una clave API puede ser regenerada, eliminada o se puede añadir una nueva

## ¿Cómo ver todas las claves api?

Necesitas ir a la [página de claves API](https://web.capgo.app/dashboard/apikeys/) y allí verás todas tus claves api

## ¿Cómo añadir una nueva clave api?

Para añadir una nueva clave API haz clic en el pequeño botón más

<figure><img src="/apikeys-add.webp" alt="add apikey" /><figcaption></figcaption></figure>

y luego selecciona los permisos que quieres dar a la nueva clave API. Puedes seleccionar:

 - Lectura - la clave API podrá leer todos los datos
 - Subir - la clave API podrá subir nuevas versiones desde la CLI y leer
 - Todo - La clave API podrá hacer todo

<figure><img src="/apikeys-select-perm.webp" alt="select perm" /><figcaption></figcaption></figure>

## ¿Cómo eliminar una clave api?

Para eliminar una clave API haz clic en el pequeño botón de papelera

<figure><img src="/apikeys-remove.webp" alt="remove apikey" /><figcaption></figcaption></figure>

Y luego confirma la eliminación de la clave API

## ¿Cómo regenerar una clave api?

Para regenerar una clave API haz clic en el pequeño botón de actualización

<figure><img src="/apikeys-regenerate.webp" alt="generate apikey" /><figcaption></figcaption></figure>

Y luego confirma la regeneración de la clave API
