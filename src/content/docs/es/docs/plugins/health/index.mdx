Importar { Card, CardGrid } De '@astrojs/starlight/components';

El Plugin Capgo Health unifica el acceso a HealthKit y Health Connect para que puedas rastrear actividad, métricas biométricas y mediciones corporales con una sola integración de Capacitor.

<CardGrid stagger>
  <Card title="Datos de salud multiplataforma" icon="rocket">
    Soporta Apple HealthKit y Android Health Connect desde el inicio.
  </Card>
  <Card title="Permisos granulares" icon="approve-Verificar">
    Solicita permisos separados de lectura/escritura y reacciona a cambios de autorización.
  </Card>
  <Card title="Unidades consistentes" icon="star">
    Trabaja con tipos de datos normalizados como `steps`, `heartRate` y `weight`.
  </Card>
  <Card title="Escritura en tiempo real" icon="rocket">
    Guarda muestras de vuelta en el almacén de salud del usuario cuando registre actividad.
  </Card>
</CardGrid>

Sigue la guía de inicio para habilitar capacidades de plataforma, recopilar consentimiento y comenzar a sincronizar métricas.
