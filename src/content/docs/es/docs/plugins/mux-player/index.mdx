Ofrece reproducción premium envolviendo los SDK nativos oficiales de Mux en una interfaz Capacitor simple.

<CardGrid stagger>
  <Card title="Experiencia de SDK nativo" icon="rocket">
    Usa el mismo stack de reproducción que los SDK móviles de Mux con tasa de bits adaptativa y DRM.
  </Card>
  <Card title="Reproducción firmada" icon="approve-Verificar">
    Proporciona tokens de reproducción y DRM para entrega segura de transmisiones protegidas.
  </Card>
  <Card title="API Uniforme" icon="pencil">
    Activa la reproducción, descarta el reproductor y escucha eventos con una API TypeScript única.
  </Card>
  <Card title="Listo para análisis" icon="star">
    Pasa claves de entorno y nombres de reproductores para alimentar la telemetría de Mux Data automáticamente.
  </Card>
</CardGrid>

La guía de inicio te guía a través de la conexión de los SDK nativos, el manejo de tokens y la respuesta a eventos del reproductor.
