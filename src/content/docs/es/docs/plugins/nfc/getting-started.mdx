## Configuración

### Configuración iOS

Agrega la descripción de uso NFC a tu `Info.plist`:

```xml
<key>NFCReaderUsageDescription</key>
<string>This app needs NFC access to read and write tags</string>
```

Habilita la capacidad Near Field Communication Tag Reading en tu proyecto Xcode.

### Configuración Android

Agrega el permiso NFC a tu `AndroidManifest.xml`:

```xml
<manifest>
  <uses-permission android:name="android.permission.NFC" />
  <uses-feature android:name="android.hardware.nfc" android:required="false" />
</manifest>
```

## Uso

### Comenzar Escaneo de Etiquetas NFC

```typescript
import { CapacitorNfc } from '@capgo/capacitor-nfc';

await CapacitorNfc.startScanning({
  invalidateAfterFirstRead: false, // Mantener sesión abierta (iOS)
  alertMessage: 'Hold a tag near the top of your device.',
});

const listener = await CapacitorNfc.addListener('nfcEvent', (event) => {
  console.log('Tag detected:', event.type);
  console.log('Tag ID:', event.tag?.id);
  console.log('NDEF message:', event.tag?.ndefMessage);
});
```

### Leer Etiqueta NFC

```typescript
await CapacitorNfc.addListener('nfcEvent', (event) => {
  if (event.tag?.ndefMessage) {
    event.tag.ndefMessage.forEach(record => {
      console.log('TNF:', record.tnf);
      console.log('Type:', record.type);
      console.log('Payload:', record.payload);

      // Decodificar registro de texto
      if (record.tnf === 1 && record.type[0] === 0x54) { // Registro de texto
        const langLen = record.payload[0] & 0x3f;
        const text = new TextDecoder().decode(
          new Uint8Array(record.payload.slice(langLen + 1))
        );
        console.log('Text:', text);
      }
    });
  }
});
```

### Escribir en Etiqueta NFC

```typescript
// Preparar un registro de texto
const encoder = new TextEncoder();
const langBytes = Array.from(encoder.encode('en'));
const textBytes = Array.from(encoder.encode('Hello NFC'));
const payload = [langBytes.length & 0x3f, ...langBytes, ...textBytes];

await CapacitorNfc.write({
  allowFormat: true,
  records: [
    {
      tnf: 0x01, // TNF Well-known
      type: [0x54], // 'T' para Texto
      id: [],
      payload,
    },
  ],
});

console.log('Tag written successfully');
```

### Escribir URL en Etiqueta NFC

```typescript
const url = 'https://capgo.app';
const urlBytes = Array.from(new TextEncoder().encode(url));

await CapacitorNfc.write({
  allowFormat: true,
  records: [
    {
      tnf: 0x01, // TNF Well-known
      type: [0x55], // 'U' para URI
      id: [],
      payload: [0x01, ...urlBytes], // 0x01 = https://
    },
  ],
});
```

### Borrar Etiqueta NFC

```typescript
await CapacitorNfc.erase();
console.log('Tag erased');
```

### Hacer Etiqueta de Solo Lectura

```typescript
await CapacitorNfc.makeReadOnly();
console.log('Tag is now read-only');
```

### Detener Escaneo

```typescript
await listener.remove();
await CapacitorNfc.stopScanning();
```

### Verificar Estado NFC

```typescript
const { status } = await CapacitorNfc.getStatus();
console.log('NFC status:', status);
// Valores posibles: 'NFC_OK', 'NO_NFC', 'NFC_DISABLED', 'NDEF_PUSH_DISABLED'

if (status === 'NFC_DISABLED') {
  // Abrir configuración del sistema
  await CapacitorNfc.showSettings();
}
```

### Android Beam (Compartir P2P)

```typescript
// Compartir datos vía Android Beam
const message = {
  records: [
    {
      tnf: 0x01,
      type: [0x54], // Texto
      id: [],
      payload: [/* payload de registro de texto */],
    },
  ],
};

await CapacitorNfc.share(message);

// Más tarde, detener compartir
await CapacitorNfc.unshare();
```

## Referencia de API

### startScanning(options?)

Comienza a escuchar etiquetas NFC.

```typescript
interface StartScanningOptions {
  invalidateAfterFirstRead?: boolean; // Solo iOS, predeterminado true
  alertMessage?: string; // Solo iOS
  androidReaderModeFlags?: number; // Solo Android
}

await CapacitorNfc.startScanning(options);
```

### stopScanning()

Detiene la sesión de escaneo NFC.

```typescript
await CapacitorNfc.stopScanning();
```

### write(options)

Escribe registros NDEF en la última etiqueta descubierta.

```typescript
interface WriteTagOptions {
  records: NdefRecord[];
  allowFormat?: boolean; // Predeterminado true
}

interface NdefRecord {
  tnf: number; // Type Name Format
  type: number[]; // Tipo de registro
  id: number[]; // ID de registro
  payload: number[]; // Payload de registro
}

await CapacitorNfc.write(options);
```

### erase()

Borra la última etiqueta descubierta.

```typescript
await CapacitorNfc.erase();
```

### makeReadOnly()

Hace la última etiqueta descubierta de solo lectura (permanente).

```typescript
await CapacitorNfc.makeReadOnly();
```

### share(options)

Comparte mensaje NDEF vía Android Beam (solo Android).

```typescript
await CapacitorNfc.share({ records: [...] });
```

### unshare()

Detiene compartir (solo Android).

```typescript
await CapacitorNfc.unshare();
```

### getStatus()

Obtiene el estado actual del adaptador NFC.

```typescript
const { status } = await CapacitorNfc.getStatus();
// Retorna: 'NFC_OK' | 'NO_NFC' | 'NFC_DISABLED' | 'NDEF_PUSH_DISABLED'
```

### showSettings()

Abre la configuración NFC del sistema.

```typescript
await CapacitorNfc.showSettings();
```

## Eventos

### nfcEvent

Se dispara cuando se descubre una etiqueta NFC.

```typescript
interface NfcEvent {
  type: 'tag' | 'ndef' | 'ndef-mime' | 'ndef-formattable';
  tag?: NfcTag;
}

interface NfcTag {
  id: number[];
  techTypes: string[];
  type: string | null;
  maxSize: number | null;
  isWritable: boolean | null;
  canMakeReadOnly: boolean | null;
  ndefMessage: NdefRecord[] | null;
}
```

### nfcStateChange

Se dispara cuando cambia la disponibilidad del adaptador NFC (solo Android).

```typescript
interface NfcStateChangeEvent {
  status: NfcStatus;
  enabled: boolean;
}
```

## Ejemplo Completo

```typescript
import { CapacitorNfc } from '@capgo/capacitor-nfc';

export class NfcService {
  private listener: any;

  async startReading() {
    // Verificar estado NFC
    const { status } = await CapacitorNfc.getStatus();

    if (status === 'NO_NFC') {
      throw new Error('NFC not available on this device');
    }

    if (status === 'NFC_DISABLED') {
      await CapacitorNfc.showSettings();
      return;
    }

    // Comenzar escaneo
    await CapacitorNfc.startScanning({
      invalidateAfterFirstRead: false,
      alertMessage: 'Ready to scan NFC tags',
    });

    // Escuchar etiquetas
    this.listener = await CapacitorNfc.addListener('nfcEvent', (event) => {
      this.handleNfcEvent(event);
    });
  }

  private handleNfcEvent(event: any) {
    console.log('NFC Event:', event.type);

    if (event.tag?.ndefMessage) {
      event.tag.ndefMessage.forEach(record => {
        this.processRecord(record);
      });
    }
  }

  private processRecord(record: any) {
    // Procesar registros de texto
    if (record.tnf === 1 && record.type[0] === 0x54) {
      const langLen = record.payload[0] & 0x3f;
      const text = new TextDecoder().decode(
        new Uint8Array(record.payload.slice(langLen + 1))
      );
      console.log('Text:', text);
    }

    // Procesar registros URI
    if (record.tnf === 1 && record.type[0] === 0x55) {
      const uriCode = record.payload[0];
      const uri = new TextDecoder().decode(
        new Uint8Array(record.payload.slice(1))
      );
      console.log('URI:', uri);
    }
  }

  async writeText(text: string) {
    const encoder = new TextEncoder();
    const langBytes = Array.from(encoder.encode('en'));
    const textBytes = Array.from(encoder.encode(text));
    const payload = [langBytes.length & 0x3f, ...langBytes, ...textBytes];

    await CapacitorNfc.write({
      allowFormat: true,
      records: [
        {
          tnf: 0x01,
          type: [0x54],
          id: [],
          payload,
        },
      ],
    });
  }

  async stopReading() {
    if (this.listener) {
      await this.listener.remove();
    }
    await CapacitorNfc.stopScanning();
  }
}
```

## Tipos de Registro NDEF

### TNF (Type Name Format)

- `0x00`: Vacío
- `0x01`: Well-known (ej., Texto, URI)
- `0x02`: Tipo de medio MIME
- `0x03`: URI absoluto
- `0x04`: Tipo externo
- `0x05`: Desconocido
- `0x06`: Sin cambios
- `0x07`: Reservado

### Tipos de Registro Comunes

- **Texto**: `type: [0x54]` ('T')
- **URI**: `type: [0x55]` ('U')
- **Smart Poster**: `type: [0x53, 0x70]` ('Sp')

### Prefijos URI

- `0x00`: (sin prefijo)
- `0x01`: `https://`
- `0x02`: `https://`
- `0x03`: `http://`
- `0x04`: `https://www.`

## Mejores Prácticas

1. **Verificar Estado NFC**: Siempre verifica que NFC esté disponible y habilitado
2. **Manejar Permisos**: Solicita permisos NFC apropiadamente
3. **Detener Escaneo**: Siempre detén el escaneo cuando termines para ahorrar batería
4. **Manejo de Errores**: Envuelve operaciones NFC en bloques try-catch
5. **Probar en Dispositivos**: Las funciones NFC no funcionan en simuladores/emuladores

## Notas de Plataforma

### iOS
- Requiere iOS 11.0+
- Usa el framework Core NFC
- Soporta lectura de etiquetas en segundo plano (iOS 13+)
- Limitado a leer etiquetas formateadas NDEF
- No puede escribir etiquetas en segundo plano
- Requiere NFCReaderUsageDescription en Info.plist

### Android
- Requiere Android 4.4 (API 19)+
- Usa Android NFC API
- Soporta lectura de etiquetas en primer y segundo plano
- Puede escribir en etiquetas
- Soporta Android Beam (P2P) en dispositivos con NFC
- Requiere permiso NFC en AndroidManifest.xml

### Web
- No soportado en plataforma web
