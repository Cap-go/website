Importar { Card, CardGrid } De '@astrojs/starlight/components';

Controla el sistema de interacción de texto de Apple para adaptar la experiencia de usuario en pantallas de kiosco, juegos e instalaciones interactivas.

<CardGrid stagger>
  <Card title="Específico de iOS" icon="rocket">
    Alterna la lupa y los controles de selección introducidos en iOS 15.
  </Card>
  <Card title="API simple" icon="pencil">
    Llama a un solo método para habilitar o deshabilitar el estado de interacción de texto.
  </Card>
  <Card title="Valores predeterminados seguros" icon="approve-Verificar">
    Recibe un resultado booleano para que puedas detectar versiones no compatibles.
  </Card>
  <Card title="Amigable para kioscos" icon="star">
    Previene copiar/pegar con pulsación prolongada en instalaciones controladas o señalización.
  </Card>
</CardGrid>

Usa la guía de primeros pasos para un patrón de integración mínimo y recordatorios para volver a habilitar la interacción antes de mostrar entradas de texto.
