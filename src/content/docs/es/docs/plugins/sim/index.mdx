Usa Capgo SIM para consultar datos del operador, índices de ranura y estados de permisos mientras permaneces dentro de Capacitor.

<CardGrid stagger>
  <Card title="Soporte Multi-SIM" icon="rocket">
    Enumera cada SIM instalada junto con índices de ranura e IDs de suscripción.
  </Card>
  <Card title="Ayudantes de permisos" icon="approve-Verificar">
    Verifica y solicita el permiso de telefonía requerido en Android.
  </Card>
  <Card title="Esquema consistente" icon="star">
    Trabaja con campos de operador normalizados como MCC, MNC, país ISO y números de teléfono.
  </Card>
  <Card title="Consciente de iOS" icon="pencil">
    Maneja las particularidades de la plataforma como datos limitados del operador en iOS 16.4+.
  </Card>
</CardGrid>

Obtén la guía de inicio para integrar patrones de permisos y consumo.
