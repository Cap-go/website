Te has estado preguntando si este Plugin alguna vez estará disponible para Cordova.

Hemos iniciado un repositorio de I+D para eso, pero es una gran cantidad de trabajo.

## Problemas

Sabemos que podemos hacerlo, pero para eso, tenemos que leer todo el código de la base de código de Cordova como lo hicimos con Capacitor, para entender cómo hacerlo funcionar con las características de Capgo.

La versión de Android es más fácil de hacer ya que ambos usan Java, pero iOS necesita una reescritura completa porque Swift aún no está bien soportado en Cordova

## Solución

Mientras tanto, esto es lo que puedes hacer:

* [Apóyanos](https://github.com/sponsors/cap-go) en GitHub y podemos priorizar eso.
Esto necesitará al menos 1 mes de trabajo.
* Contrátanos como [Consultor](https://capgo.app/consulting/), estamos acostumbrados a ayudar a grandes empresas a migrar a Capacitor, usualmente toma un mes, y el [beneficio](https://ionic.io/resources/articles/capacitor-vs-cordova-modern-hybrid-app-development) es enorme para tu equipo
