import { Card, CardGrid } from '@astrojs/starlight/components';

El Plugin AudioSession de Capgo te brinda control detallado sobre la capa AVAudioSession de iOS para que puedas manejar con elegancia auriculares, dispositivos Bluetooth e interrupciones inesperadas.

<CardGrid stagger>
  <Card title="Detección de ruta" icon="rocket">
    Detecta cuando los usuarios conectan o desconectan auriculares, AirPods, altavoces Bluetooth y más.
  </Card>
  <Card title="Anulación de altavoz" icon="setting">
    Fuerza la reproducción a través del altavoz integrado cuando necesites audio manos libres.
  </Card>
  <Card title="Manejo de interrupciones" icon="moon">
    Reacciona a eventos de interrupción como llamadas telefónicas o Siri para pausar y reanudar el audio de forma segura.
  </Card>
  <Card title="Seguridad de tipos" icon="pencil">
    Definiciones sólidas de TypeScript facilitan la creación de experiencias de audio resilientes.
  </Card>
</CardGrid>

Este Plugin se dirige a iOS donde se requiere el control de AVAudioSession. En otras plataformas, las llamadas se resuelven sin efecto, lo que te permite mantener una ruta de código compartida.
