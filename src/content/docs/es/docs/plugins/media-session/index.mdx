El plugin Media Session conecta Capacitor con la API de Media Session para que tus reproductores de audio y video se sientan nativos en todas partes.

<CardGrid stagger>
  <Card title="Metadatos enriquecidos" icon="star">
    Proporciona arte de álbum, artista y título para los controles de pantalla de bloqueo y notificaciones.
  </Card>
  <Card title="Sincronización de estado de reproducción" icon="rocket">
    Mantén los botones de reproducción/pausa y los indicadores de progreso alineados con tu reproductor.
  </Card>
  <Card title="Callbacks de acciones" icon="pencil">
    Maneja botones de auriculares y acciones multimedia del sistema operativo desde tu código Capacitor.
  </Card>
  <Card title="Reporte de posición" icon="download">
    Envía duración, posición y velocidad para que el desplazamiento funcione con fluidez.
  </Card>
</CardGrid>

Salta a la guía de inicio para conectar las actualizaciones de metadatos al ciclo de vida de tu reproductor.
