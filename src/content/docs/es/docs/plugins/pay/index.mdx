Importar { Card, CardGrid } De '@astrojs/starlight/components';

Capgo Pay conecta los últimos SDKs de Apple Pay y Google Pay para que puedas ofrecer una experiencia de checkout perfecta en todos los dispositivos.

<CardGrid stagger>
  <Card title="API Unificada" icon="pencil">
    Llama `isPayAvailable` y `requestPayment` una vez y deja que el Plugin maneje cada plataforma.
  </Card>
  <Card title="Configuración rápida" icon="rocket">
    Implementa hojas de pago completamente nativas sin cablear manualmente código específico de plataforma.
  </Card>
  <Card title="Pagos tokenizados" icon="approve-Verificar">
    Recibe tokens de pago listos para tus integraciones de pasarela o PSP.
  </Card>
  <Card title="Flujos personalizables" icon="star">
    Configura elementos de línea, campos de contacto y requisitos de facturación para ajustarse a tu checkout.
  </Card>
</CardGrid>

Sumérgete en la guía de inicio para listas de verificación de configuración y payloads de muestra para cada plataforma.
