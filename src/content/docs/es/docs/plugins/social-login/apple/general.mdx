### Introducción

En esta guía, aprenderás cómo configurar el inicio de sesión de Apple con Capacitor. Para hacer esto, necesitarás lo siguiente:

- una cuenta de Apple Developer
- Una computadora con macOS (solo iOS)
- Xcode instalado (solo iOS)
- Un backend personalizado (solo Android)
