## Introducción

En esta guía, aprenderá cómo configurar el inicio de sesión Google con el inicio de sesión social Capgo. Necesitará lo siguiente para configurar Google Iniciar sesión:

- Una cuenta Google

## Configuración general

:::note
Este paso es necesario independientemente de la plataforma que decida utilizar. 
:::

En esta parte, configurará la pantalla de inicio de sesión que muestra Google.

Si desea llamar a Google API, le recomiendo encarecidamente que consulte el OAuth 2.0 Playground de [Google](https://developers.google.com/oauthplayground). Desde allí puedes ver fácilmente qué API puedes llamar.

## Usando el acceso sin conexión con tu propio backend

Para utilizar el acceso en línea necesitará lo siguiente:

- Un servidor HTTP

En este ejemplo, utilizaré las siguientes tecnologías para proporcionar acceso sin conexión a mi aplicación:

- [Hono](https://hono.dev/)

- [Validador de Hono Zod](https://hono.dev/docs/guides/validation#with-zod)

- [Zod](https://zod.dev/)

- [Hono JWT](https://hono.dev/docs/helpers/jwt#jwt-authentication-helper)

- [LowDb](https://www.npmjs.com/package/lowdb) (una base de datos simple)

El código para este ejemplo se puede encontrar [aquí](https://github.com/WcaleNieWolny/capgo-social-login-backend-demo/blob/aac7a8c909f650a8c2cd7f88c97f5f3c594ce9ba/index.ts#L139-L287)

En cuanto al código del cliente, se ve así:

```mecanografiado
import { Capacitor } from '@capacitor/core';
import { GoogleLoginOfflineResponse, SocialLogin } from '@capgo/capacitor-social-login';
import { usePopoutStore } from '@/popoutStore'; // <-- specific to my app

const baseURL = "[redactado]";

función asíncrona fullLogin() {
  espere SocialLogin.initialize({
    google: {
      webClientId: '[redactado]',  
      iOSClientId: '[redactado]',
      iOSServerClientId: 'El mismo valor que webClientId',
      modo: 'fuera de línea' // <-- importante
    } 
  })
  respuesta constante = esperar SocialLogin.login({
    proveedor: 'google',
    opciones: {
      forceRefreshToken: verdadero // <-- importante
    }
  })

  si (respuesta.proveedor === 'google') {
    resultado constante = respuesta.resultado como GoogleLoginOfflineResponse
    const res = esperar a buscar(`${baseURL}/auth/google_offline`, {
      encabezados: {
        "Tipo de contenido": "aplicación/json"
      },
      cuerpo: JSON.stringify({
        serverAuthCode: resultado.serverAuthCode,
        plataforma: Capacitor.getPlatform()
      }),
      método: "POST"
    })

    si (res.estado! == 200) {
      popoutStore.popout("Error al iniciar sesión en Google", "verificar consola");
      volver
    }

    const {jwt} = esperar res.json();
    const información de usuario = esperar a buscar (`${baseURL}/auth/get_google_user`, {
      encabezados: {
        Autorización: `Bearer ${jwt}`
      }
    })
    si (información de usuario.status! == 200) {
      popoutStore.popout("Falló el inicio de sesión completo de Google (información de usuario)", "verificar consola");
      volver
    }
    popoutStore.popout("resolución de información de usuario", espera información de usuario.text());
  }
}
