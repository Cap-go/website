import { Card, CardGrid } from '@astrojs/starlight/components';

El Plugin Document Scanner brinda a tu aplicación Capacitor un flujo de escaneo con calidad nativa que funciona en dispositivos iOS y Android.

<CardGrid stagger>
  <Card title="Detección automática" icon="rocket">
    Detecta bordes de página, recorta automáticamente y permite a los usuarios ajustar antes de guardar.
  </Card>
  <Card title="Captura multipágina" icon="star">
    Captura una o muchas páginas en una sola sesión de escaneo con límites ajustables.
  </Card>
  <Card title="Salida personalizada" icon="Descargar">
    Devuelve rutas de archivo o cadenas base64 con la calidad de imagen que elijas.
  </Card>
  <Card title="API unificada" icon="pencil">
    Una interfaz TypeScript abarca las implementaciones de iOS y Android.
  </Card>
</CardGrid>

Usa la guía de inicio para habilitar permisos de cámara, configurar ajustes de calidad y exportar archivos escaneados a tu flujo de trabajo de almacenamiento.
