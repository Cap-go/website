Convierte HTML o páginas remotas en PDFs compartibles con el renderizador nativo de Capgo para iOS y Android.

<CardGrid stagger>
  <Card title="HTML a PDF" icon="pencil">
    Alimenta con plantillas HTML y obtén un documento PDF listo para compartir en segundos.
  </Card>
  <Card title="Captura remota" icon="rocket">
    Captura cualquier URL accesible y guarda o distribuye el PDF generado.
  </Card>
  <Card title="Salidas flexibles" icon="star">
    Devuelve datos en base64 o activa la hoja de compartir nativa según tu flujo de trabajo.
  </Card>
  <Card title="Controles de documento" icon="setting">
    Especifica orientación, tamaño de página y nombres de archivo para cada renderizado.
  </Card>
</CardGrid>
