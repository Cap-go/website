## Integración CI

Para automatizar tu trabajo, recomendamos usar GitHub Actions para enviar tus actualizaciones a Capgo. Consulta nuestro [tutorial de GitHub Actions](https://capgo.app/blog/automatic-build-and-release-with-github-actions/) para más información

No olvides configurar las variables de entorno de tu CI con tu clave API de Capgo
