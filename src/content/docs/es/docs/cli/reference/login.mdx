`npx @capgo/cli login [apikey]`

Este comando almacena tu clave API de Capgo para uso futuro

:::note
Puedes sobrescribir la clave API almacenada pasando `--apikey=<key>` a cualquier comando
:::

Opciones:

- `--local`: Almacena la clave API en el repositorio local y lo agrega a `gitignore`
