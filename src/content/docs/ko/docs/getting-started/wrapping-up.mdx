이제 빠른 시작 가이드를 완료했으니, Capgo의 핵심 개념에 대한 기본적인 이해를 갖게 되었을 것입니다! 이 가이드에서 학습한 핵심 개념들은 다음과 같습니다:

<Steps>
1 Capgo 계정에 앱 추가하기

2 CI/CD 파이프라인에 Capgo 통합하기

3 새로운 커밋에서 Capgo로 번들 업로드 트리거하기

4 Capgo SDK로 실시간 업데이트를 활성화하도록 앱 구성하기

5 Capgo 대시보드에서 앱에 실시간 업데이트 배포하기
</Steps>
