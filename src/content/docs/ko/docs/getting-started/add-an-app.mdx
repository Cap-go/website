### 수동 설정

init 명령어가 작동하지 않는 경우, 수동으로 앱을 추가할 수 있습니다

<Steps>

1 CLI를 계정에 연결하세요: <Code code={`npx @capgo/cli@latest login [APIKEY]`} lang="bash" />

2 이 명령어로 계정에 앱을 추가하세요: <Code code={`npx @capgo/cli@latest app add [APP_NAME]`} lang="bash" />

3 앱에 플러그인을 설치하세요: <Code code={`npm i @capgo/capacitor-updater`} lang="bash" />

4 앱에서 가능한 한 빨리 init 메서드를 호출하세요: <Code code={`import { CapacitorUpdater } from '@capgo/capacitor-updater';\nCapacitorUpdaternotifyAppReady();`} lang="typescript" />

5 실시간 업데이트 배포하기 <LinkCard href="/docs/getting-started/deploy/" title="실시간 업데이트 배포" description="앱에 실시간 업데이트를 배포하는 방법 알아보기" />

</Steps>
