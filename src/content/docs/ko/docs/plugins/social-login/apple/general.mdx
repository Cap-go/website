### 소개

이 가이드에서는 Capacitor를 사용하여 Apple 로그인을 구성하는 방법을 배웁니다. 이를 위해 다음이 필요합니다:

- Apple Developer 계정
- macOS를 실행하는 컴퓨터 (iOS 전용)
- 설치된 Xcode (iOS 전용)
- 커스텀 백엔드 (Android 전용)
