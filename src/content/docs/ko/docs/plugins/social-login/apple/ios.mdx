iOS에서 Apple 로그인을 설정하기 위해 필요한 것을 세분화해 보겠습니다.

<Steps>
1. 앱의 기능을 구성합니다.

    이를 위해 Xcode를 열고 `App`을 클릭하세요.
    ![App XCode](/social-login-assets/app-xcode.png)

2. 올바른 타겟을 선택했는지 확인하세요.

    ![XCode App Target](/social-login-assets/xcode-app-target.png)

3. `Sign in with Apple` 기능을 추가했는지 확인하세요.

    ![App XCode Add Capability](/social-login-assets/app-xcode-add-cap.png)

    ![App XCode Add Capability](/social-login-assets/sign-with-apple-cap.png)

    :::caution
    `Sign in with Apple` 기능이 보이지 않으면 [Account & Organizational Data Sharing](https://developer.apple.com/account/resources/services/cwa/configure/)을 구성하세요.
    :::

4. 앱에서 Apple 로그인을 초기화합니다.

    :::note
    저는 Vue를 프레임워크로 사용하고 있으며, 정확한 구현은 선택한 프레임워크에 따라 다릅니다.
    :::

    ```ts
    //  onMounted는 Vue 전용입니다
    onMounted(() => {
    SocialLogin.initialize({
        apple: {}
    })
    });
    ```

5. 로그인 프로세스를 시작하는 버튼을 만듭니다.

    해당 버튼은 다음 함수를 호출해야 합니다:

    ```ts
    async function loginApple() {
    const res = await SocialLogin.login({
        provider: 'apple',
        options: {}
    })
    ```

6. **<u>실제</u>** 기기에서 앱을 실행하고 테스트합니다.

    단계를 정확히 따랐다면 버튼을 클릭한 후 다음 화면이 표시됩니다.

    ![iOS 기기의 Apple 로그인 프롬프트](/social-login-assets/apple-sign-in-ios-final.png)
</Steps>

완료되었습니다! 모든 설정이 완료되었습니다.
