Capgo Capacitor 플러그인 컬렉션에 오신 것을 환영합니다! 우리는 Capacitor 앱에서 놀라운 네이티브 경험을 구축하는 데 도움이 되는 고품질의 잘 문서화된 플러그인 모음을 유지 관리합니다.

## ⭐ Capgo Cloud - 라이브 업데이트

<LinkCard
  title="Capacitor Updater"
  description="Capgo Cloud를 구동하는 핵심 플러그인 - 앱 스토어 지연 없이 Capacitor 앱에 즉시 업데이트를 제공합니다. 수정 사항과 기능을 사용자에게 직접 푸시하세요."
  href="/docs/plugins/updater/"
/>

Updater 플러그인은 Capgo Cloud의 기반으로, 다음을 가능하게 합니다:
- 🚀 앱 스토어 리뷰 없이 업데이트 즉시 배포
- 📱 앱의 JavaScript, HTML, CSS 및 자산 업데이트
- 🎯 채널을 통해 특정 사용자 세그먼트 타겟팅
- 📊 내장 분석으로 업데이트 성공 모니터링
- 🔒 암호화 및 코드 서명으로 안전한 업데이트

## 🚀 시작하기

각 플러그인은 유사한 설치 패턴을 따릅니다:

1. **플러그인 설치** 선호하는 패키지 매니저 사용
2. **프로젝트 동기화** `npx cap sync` 실행
3. **플랫폼별 설정 구성** 필요한 경우
4. **코드에서 플러그인 사용 시작**

자세한 설정 지침 및 API 참조는 각 플러그인의 문서를 방문하세요.
