import { Card, CardGrid } from '@astrojs/starlight/components';

Capgo의 AdMob 플러그인은 Google Mobile Ads SDK에 대한 일급 브리지를 제공하므로 Capacitor의 단순성을 희생하지 않고 네이티브 경험을 수익화할 수 있습니다.

<CardGrid stagger>
  <Card title="유연한 광고 형식" icon="star">
    통합 API로 배너, 전면, 보상형 및 앱 오픈 광고를 제공하세요.
  </Card>
  <Card title="요청 구성" icon="setting">
    어린이 대상 플래그, 콘텐츠 등급 및 추가 네트워크 매개변수로 광고 요청을 세밀하게 조정하세요.
  </Card>
  <Card title="라이프사이클 관리" icon="rocket">
    로드 상태를 추적하면서 요청 시 광고를 생성, 로드, 표시, 숨기기 및 삭제하세요.
  </Card>
  <Card title="개인정보 준비" icon="approve-check">
    ATT 프롬프트를 처리하고 추적 승인 상태를 모니터링하여 규정을 준수하세요.
  </Card>
</CardGrid>

시작 가이드를 사용하여 AdMob 앱 ID를 연결하고, 동의를 구성하고, Capacitor 프로젝트에서 직접 광고 인벤토리를 관리하세요.
