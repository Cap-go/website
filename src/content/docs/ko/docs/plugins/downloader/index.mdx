## 개요

Capacitor Downloader 플러그인은 백그라운드 다운로드, 진행률 추적 및 포괄적인 다운로드 관리를 지원하는 강력한 파일 다운로드 기능을 제공합니다. 이 플러그인은 일시 정지/재개 기능과 상세한 진행률 모니터링을 통해 강력한 파일 다운로드를 가능하게 합니다.

> ⚠️ **개발 중**: 이 플러그인은 현재 개발 중이며 아직 프로덕션 사용 준비가 되지 않았습니다.

## 개발 상태

이 플러그인은 react-native-background-downloader에서 영감을 받아 현재 개발 중입니다. 개발이 진행됨에 따라 기능이 변경될 수 있습니다.
