## 철학

Ionic 팀은 자체 우선순위와 릴리스 일정에 따라 Capacitor를 유지 관리합니다. 커뮤니티 기여는 병합되기까지 몇 달 또는 몇 년이 걸릴 수 있습니다. Capacitor+는 다른 접근 방식을 취합니다:

1. **포크에서 PR 병합** - 업스트림 큐에 갇힌 귀중한 PR이 적극적으로 병합됨
2. **지속적인 동기화** - 모든 업스트림 변경 사항이 자동으로 가져와지고 테스트됨
3. **빠른 릴리스** - CI를 통과하는 즉시 변경 사항이 npm에 게시됨
4. **커뮤니티 우선** - 귀하의 기여가 중요하며 우선순위가 부여됨
5. **투명성** - 모든 자동화가 오픈 소스이며 가시적임
