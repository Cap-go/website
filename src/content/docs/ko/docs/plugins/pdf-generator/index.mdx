import { Card, CardGrid } from '@astrojs/starlight/components';

Capgo의 iOS 및 Android용 네이티브 렌더러를 사용하여 HTML 또는 원격 페이지를 공유 가능한 PDF로 변환합니다.

<CardGrid stagger>
  <Card title="HTML에서 PDF로" icon="pencil">
    HTML 템플릿을 입력하고 몇 초 안에 공유할 준비가 된 PDF 문서를 얻습니다.
  </Card>
  <Card title="원격 캡처" icon="rocket">
    접근 가능한 모든 URL을 스냅샷하고 생성된 PDF를 저장하거나 배포합니다.
  </Card>
  <Card title="유연한 출력" icon="star">
    워크플로우에 따라 base64 데이터를 반환하거나 네이티브 공유 시트를 트리거합니다.
  </Card>
  <Card title="문서 제어" icon="setting">
    모든 렌더링에 대해 방향, 페이지 크기 및 파일 이름을 지정합니다.
  </Card>
</CardGrid>

플랫폼 참고 사항 및 사용 예제는 시작 가이드를 참조하세요.
