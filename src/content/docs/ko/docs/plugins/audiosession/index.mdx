import { Card, CardGrid } from '@astrojs/starlight/components';

Capgo의 AudioSession 플러그인은 iOS AVAudioSession 레이어에 대한 세밀한 제어를 제공하여 헤드폰, Bluetooth 장치 및 예상치 못한 중단을 우아하게 처리할 수 있습니다.

<CardGrid stagger>
  <Card title="라우트 인식" icon="rocket">
    사용자가 헤드셋, AirPods, Bluetooth 스피커 등을 연결하거나 분리할 때 감지합니다.
  </Card>
  <Card title="스피커 재정의" icon="setting">
    핸즈프리 오디오가 필요할 때 내장 스피커를 통해 재생을 강제합니다.
  </Card>
  <Card title="중단 처리" icon="moon">
    전화 통화 또는 Siri와 같은 중단 이벤트에 반응하여 오디오를 안전하게 일시 중지하고 재개합니다.
  </Card>
  <Card title="타입 안전성" icon="pencil">
    강력한 TypeScript 정의로 탄력적인 오디오 경험을 쉽게 구축할 수 있습니다.
  </Card>
</CardGrid>

이 플러그인은 AVAudioSession 제어가 필요한 iOS를 대상으로 합니다. 다른 플랫폼에서는 호출이 효과 없이 해결되므로 하나의 공유 코드 경로를 유지할 수 있습니다.
