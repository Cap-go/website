## 왜 기여하나요?

우선, capgo 오픈소스 프로젝트에 기여하는 것을 고려해 주셔서 감사합니다! capgo 오픈 소스 프로젝트를 훌륭한 도구로 만드는 것은 바로 당신과 같은 사람들입니다.

다음은 귀하가 기여하기를 원하는 몇 가지 이유입니다:

* capgo 오픈소스 프로젝트에 기여하는 것은 capgo 팀이 제공하는 [무수한 포상금](https://console.algora.io/org/Capgo)으로 돈을 벌 수 있는 좋은 방법입니다.
* capgo 오픈소스 프로젝트에 기여하는 것은 보고 싶은 기능을 추가하는 좋은 방법입니다.
* capgo 오픈소스 프로젝트에 기여하는 것은 발생한 버그를 수정하는 좋은 방법입니다.
* [Capgo 기본 라이선스](https://github.com/Cap-go/capgo/blob/main/LICENSE)에서는 변경 사항을 오픈 소스로 제공해야 합니다. 코드를 제공함으로써 변경 사항을 오픈 소스로 유지하고 다른 사람들이 이를 사용할 수 있도록 할 수 있습니다.

## 기여 방법

 * 우선, 기여하려는 저장소를 포크해야 합니다.
 * 두 번째로 변경 사항을 해당 저장소에 커밋하고 푸시해야 합니다.
 * 마지막으로 Pull Request를 열어야 합니다.
 * 그게 다야! 이제 capgo 팀이 귀하의 PR을 검토할 때까지 기다리시면 됩니다.
