## 이 튜토리얼에서는 무엇을 다룰까요?

이 튜토리얼에서는 자체 호스팅 capgo가 있는 개발 환경에서 CLI을 사용하는 방법을 보여줍니다.

## 시작하기

자체 호스팅 capgo와 함께 CLI을 사용하려면 앱 디렉터리에서 `capacitor.config.ts`을 편집하고 다음과 같이 설정합니다.

```ts
const config: CapacitorConfig = {
  appId: 'com.demo.app',
  appName: 'demoApp',
  webDir: 'dist',
  bundledWebRuntime: false,
  plugins: {
    CapacitorUpdater : {
      localHost: "http://localhost:5173",
      localWebHost: "http://localhost:5173",
      localSupa: "http://localhost:54321",
      localSupaAnon: "see_notes",
    },
  },
};
```

참고: `localSupaAnon`을(를) 얻으려면 [이 튜토리얼](/docs/plugin/self-hosted/local-dev/getting-started/)을 따르고 'anon key'를 `localSupaAnon`에 복사하세요.
