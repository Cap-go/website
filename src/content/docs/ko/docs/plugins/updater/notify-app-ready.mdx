`CapacitorUpdater.notifyAppReady()` 호출의 배치는 올바른 자동 업데이트 흐름에 매우 중요합니다.


올바르게 수행하지 않으면 앱이 최신 버전으로 업데이트되지 않습니다. 10초 이내에 `notifyAppReady()`를 호출하지 않는 각 버전은 유효하지 않은 것으로 표시되며 이전의 유효한 버전 또는 기본(내장) 버전으로 대체됩니다.
