## 기여하는 이유는?

우선, capgo 오픈소스 프로젝트에 기여하는 것을 고려해 주셔서 감사합니다! 여러분과 같은 분들이 capgo 오픈소스 프로젝트를 훌륭한 도구로 만들어 주고 있습니다.

다음은 기여해야 하는 이유입니다:

* capgo 오픈소스 프로젝트에 기여하는 것은 capgo 팀이 제공하는 [수많은 바운티](https://consolealgoraio/org/Capgo/)로부터 돈을 벌 수 있는 좋은 방법입니다
* capgo 오픈소스 프로젝트에 기여하는 것은 보고 싶은 기능을 추가할 수 있는 좋은 방법입니다
* capgo 오픈소스 프로젝트에 기여하는 것은 발견한 버그를 수정할 수 있는 좋은 방법입니다
* [Capgo 메인 라이선스](https://github.com/Cap-go/capgo/blob/main/LICENSE/)는 여러분이 변경한 사항을 오픈소스로 공개하도록 요구합니다. 코드를 기여함으로써, 여러분의 변경사항을 오픈소스로 유지하고 다른 사람들이 사용할 수 있도록 할 수 있습니다

## 기여하는 방법

* 우선, 기여하고자 하는 저장소를 포크해야 합니다
* 두 번째로 변경사항을 커밋하고 해당 저장소에 푸시해야 합니다
* 마지막으로, 풀 리퀘스트를 열어야 합니다
* 이게 전부입니다! 이제 capgo 팀이 PR을 검토하기를 기다리면 됩니다
