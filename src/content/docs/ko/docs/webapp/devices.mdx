:::note[데이터 보존]
디바이스 데이터는 마지막 디바이스 활동으로부터 **90일** 동안 보존됩니다. 이 기간 내에 Capgo에 연결하지 않은 디바이스는 목록에서 자동으로 제거됩니다. 활성 디바이스는 업데이트를 확인할 때 지속적으로 추적됩니다.
:::

## 모든 기기 목록 표시
먼저 기기 페이지를 살펴보겠습니다. [앱을 클릭](/docs/webapp/main-page)한 다음 [기기 탭을 클릭](/docs/webapp/main-app-page)하여 접근할 수 있습니다.

<figure><img style="margin-left: auto; margin-right: auto" src="/devices.webp" alt="기기 목록" /><figcaption></figcaption></figure>

1. **디바이스 탭** - 내비게이션 바에서 이 탭을 클릭하여 디바이스 페이지에 접근합니다
2. **필터** - 클릭하여 디바이스를 필터링합니다:
   - **오버라이드** - [사용자 정의 채널](/docs/plugin/api/#setchannel) 또는 사용자 정의 버전이 있는 디바이스만 표시합니다
   - **사용자 정의 ID** - 사용자 정의 식별자가 설정된 디바이스만 표시합니다

## 기기 구성하기

특정 기기를 구성하려면 테이블에서 해당 기기를 클릭하세요. 그러면 다음과 같은 화면이 표시됩니다:

<figure><img style="margin-left: auto; margin-right: auto" src="/device-specific.webp" alt="하나의 기기 표시" /><figcaption></figcaption></figure>

1. **Information** - 디바이스 ID, 마지막 업데이트, 플랫폼, 플러그인 버전, 버전, 빌트인 버전, OS 버전, 프로덕션 앱 여부 등 디바이스 세부 정보를 봅니다
2. **Deployments** - 이 디바이스의 배포 기록을 봅니다
3. **Logs** - 이 디바이스와 관련된 로그를 봅니다
4. **채널 오버라이드** - 이 특정 디바이스의 채널을 오버라이드합니다. 설정되면 이 디바이스는 공개(기본) 채널을 사용하지 않고 선택한 채널에서만 업데이트를 받습니다

### 사용자 정의 ID

`사용자 정의 ID`는 디바이스를 자신의 사용자를 위해 인식할 수 있는 ID에 연결하는 데 도움이 되는 매우 유용한 필드입니다.

어떤 디바이스가 어떤 사용자에게 속하는지 쉽게 식별하기 위해 사용하세요.

사용자 정의 ID는 `CapacitorUpdater.setCustomId({ customId: user });`를 사용하여 디바이스 자체에서만 설정할 수 있습니다.
