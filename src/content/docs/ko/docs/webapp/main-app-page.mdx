## 메인 페이지에서 무엇을 보여주나요?

먼저, 앱의 메인 페이지를 살펴보겠습니다:

<figure><img src="/main-app-pagewebp" alt="Main page screeshot" /><figcaption></figcaption></figure>

자세히 살펴보겠습니다.

먼저 통계입니다. 통계가 현재 재작업 중이어서 지금은 문서화할 수 없습니다. 완전히 작동되면 통계 섹션을 추가할 예정입니다.

두 번째로, 사용 가능한 모든 메뉴와 그 기능을 살펴보겠습니다.

<figure><img src="/app-multiple-sectionswebp" alt="Multiple sections screeshot" /><figcaption></figcaption></figure>

첫 번째 버튼은 [채널](/docs/webapp/channels/) 페이지로 이동합니다. 여기서 앱의 채널을 생성/삭제/구성할 수 있습니다.
두 번째 버튼은 [번들](/docs/webapp/bundles/) 페이지로 이동합니다. 여기서 앱의 모든 버전을 볼 수 있습니다.
세 번째 버튼은 [기기](/docs/webapp/devices/) 페이지로 이동합니다. 여기서 모든 기기를 볼 수 있습니다. 또한 여기서 기기별 특정 오버라이드를 설정할 수 있습니다.
네 번째 버튼은 [업데이트](/docs/webapp/logs/) 페이지로 이동합니다. 여기서 앱이 경험하고 있는 자세한 통계(로그)와 오류를 볼 수 있습니다.
