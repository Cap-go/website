## API 키로 무엇을 할 수 있나요?

API 키는 재생성하거나 삭제하거나 새로 추가할 수 있습니다

## 모든 API 키를 어떻게 볼 수 있나요?

[API 키 페이지](https://webcapgoapp/dashboard/apikeys/)로 이동하면 모든 API 키를 볼 수 있습니다

## 새로운 API 키를 어떻게 추가하나요?

새로운 API 키를 추가하려면 작은 플러스 버튼을 클릭하세요

<figure><img src="/apikeys-addwebp" alt="add apikey" /><figcaption></figcaption></figure>

그리고 새로운 API 키에 부여할 권한을 선택하세요. 다음과 같은 권한을 선택할 수 있습니다:

 - 읽기 - API 키로 모든 데이터를 읽을 수 있습니다
 - 업로드 - API 키로 CLI에서 새 버전을 업로드하고 읽을 수 있습니다
 - 모두 - API 키로 모든 작업을 수행할 수 있습니다

<figure><img src="/apikeys-select-permwebp" alt="select perm" /><figcaption></figcaption></figure>

## API 키를 어떻게 삭제하나요?

API 키를 삭제하려면 작은 휴지통 버튼을 클릭하세요

<figure><img src="/apikeys-removewebp" alt="remove apikey" /><figcaption></figcaption></figure>

그리고 API 키 삭제를 확인하세요

## API 키를 어떻게 재생성하나요?

API 키를 재생성하려면 작은 새로고침 버튼을 클릭하세요

<figure><img src="/apikeys-regeneratewebp" alt="generate apikey" /><figcaption></figcaption></figure>

그리고 API 키 재생성을 확인하세요
