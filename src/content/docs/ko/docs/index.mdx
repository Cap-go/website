## 🚀 Capgo 클라우드 - 간편한 실시간 업데이트

<CardGrid stagger>
  <Card title="즉각적인 업데이트" icon="rocket">
    앱 스토어 지연 없이 JavaScript, HTML, CSS 업데이트를 사용자에게 직접 배포합니다. 버그를 수정하고 며칠이 아닌 몇 분 만에 기능을 제공하세요.
  </Card>
  <Card title="3단계 통합" icon="pencil">
    `npx @capgo/cli@latest init [APIKEY]` 명령어만으로 시작하고 간단한 통합으로 즉시 업데이트를 푸시하세요.
  </Card>
  <Card title="고급 기능" icon="setting">
    A/B 테스팅을 위한 채널, 자동 롤백, 업데이트 분석, 암호화된 번들, 엔터프라이즈급 보안 기능을 제공합니다.
  </Card>
  <Card title="완전한 가이드" icon="open-book">
    포괄적인 문서에서 [빠른 설정](/docs/getting-started/quickstart/)부터 고급 배포 전략까지 모든 것을 배우세요.
  </Card>
</CardGrid>
