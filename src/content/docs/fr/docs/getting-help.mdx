## Support par Discord

Capgo possède un [serveur Discord](https://discordcom/invite/VnYRvBfgA6) officiel. Obtenir un support technique là-bas est probablement l'un des moyens les plus rapides pour obtenir une réponse.

Voici un guide rapide :

Étape 1 - allez dans le canal `questions`

<figure><img style="margin-left: auto; margin-right: auto" src="/discord-questionswebp" alt="Ask on discord" /><figcaption></figcaption></figure>

Étape 2 - créez votre fil de discussion

<figure><img style="margin-left: auto; margin-right: auto" src="/discord-newquestionwebp" alt="Create a question on discord" /><figcaption></figcaption></figure>

Étape 3 - Décrivez votre problème et sélectionnez les tags pertinents

<figure><img style="margin-left: auto; margin-right: auto" src="/discord-new-postwebp" alt="Create a post on discord" /><figcaption></figcaption></figure>

Étape 4 - Partagez votre identifiant de compte sécurisé (facultatif)

Cela permettra à l'équipe de Capgo d'examiner votre compte. Le partage de cet identifiant est sûr, car il a été conçu pour être partagé publiquement.

Pour le partager, rendez-vous dans les [paramètres de Capgo](https://webcapgoapp/dashboard/settings/account/). Cliquez ensuite sur `copier l'identifiant du compte`

<figure><img style="margin-left: auto; margin-right: auto" src="/share-secure-idwebp" alt="Share your id without leaking your info" /><figcaption></figcaption></figure>

Cela copiera l'identifiant de compte sécurisé dans le presse-papiers. Veuillez l'inclure dans votre message Discord.

## Support par email

C'est le moyen le plus lent d'obtenir de l'aide. Veuillez utiliser d'abord le serveur Discord.

Si vous devez nous contacter par email, veuillez envoyer un email à support@capgoapp
