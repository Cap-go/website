Maintenant que vous avez terminé le guide de démarrage rapide, vous devriez avoir une compréhension de base des concepts clés de Capgo ! Les concepts clés que vous avez appris dans ce guide sont :

<Steps>
1. Ajouter une application à votre compte Capgo

2. Intégrer Capgo à votre pipeline CI/CD

3. Déclencher des téléversements de bundles vers Capgo lors de nouveaux commits

4. Configurer votre application pour activer les mises à jour en direct avec le SDK Capgo

5. Déployer des mises à jour en direct vers votre application depuis le tableau de bord Capgo
</Steps>
