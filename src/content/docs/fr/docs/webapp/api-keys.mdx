## Que puis-je faire avec les clés api ?

Une clé API peut être régénérée, supprimée ou une nouvelle peut être ajoutée

## Comment voir toutes les clés api ?

Vous devez aller sur la [page des clés API](https://web.capgo.app/dashboard/apikeys/) et là vous verrez toutes vos clés api

## Comment ajouter une nouvelle clé api ?

Pour ajouter une nouvelle clé API, cliquez sur le petit bouton plus

<figure><img src="/apikeys-add.webp" alt="add apikey" /><figcaption></figcaption></figure>

puis sélectionnez les permissions que vous souhaitez donner à la nouvelle clé API. Vous pouvez sélectionner :

 - Lecture - la clé API pourra lire toutes les données
 - Upload - la clé API pourra télécharger de nouvelles versions depuis la CLI et lire
 - Tout - La clé API pourra tout faire

<figure><img src="/apikeys-select-perm.webp" alt="select perm" /><figcaption></figcaption></figure>

## Comment supprimer une clé api ?

Pour supprimer une clé API, cliquez sur le petit bouton corbeille

<figure><img src="/apikeys-remove.webp" alt="remove apikey" /><figcaption></figcaption></figure>

Puis confirmez la suppression de la clé API

## Comment régénérer une clé api ?

Pour régénérer une clé API, cliquez sur le petit bouton rafraîchir

<figure><img src="/apikeys-regenerate.webp" alt="generate apikey" /><figcaption></figcaption></figure>

Puis confirmez la régénération de la clé API
