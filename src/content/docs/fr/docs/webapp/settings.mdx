## Comment accéder à la page des paramètres

Cliquez d'abord sur **PRÉNOM NOM** Dans mon cas, c'est **test Capgo** Dans votre cas, ce sera votre prénom puis votre nom Ensuite, cliquez sur **Paramètres**

<figure><img src="/settings-go.webp" alt="open settings" /><figcaption></figcaption></figure>

## Modification des paramètres utilisateur

Pour modifier un paramètre utilisateur, il suffit de remplir le formulaire dans le compte puis de cliquer sur **Mettre à jour**

<figure><img src="/account-save.webp" alt="save change in account" /><figcaption></figcaption></figure>

Une confirmation devrait alors apparaître

<figure><img src="/account-updated.webp" alt="acount updated" /><figcaption></figcaption></figure>

## Modification du mot de passe

Pour changer le mot de passe, allez dans la page des paramètres et cliquez sur **Mot de passe** Puis remplissez le formulaire et cliquez sur **Mettre à jour**

<figure><img src="/update-passwd.webp" alt="update password" /><figcaption></figcaption></figure>

Lorsque le mot de passe ne respecte pas les règles de sécurité de Capgo, vous recevrez un message d'erreur

<figure><img src="/passwd-error.webp" alt="wrong password" /><figcaption></figcaption></figure>
