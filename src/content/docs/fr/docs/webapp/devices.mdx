## Afficher la liste de tous les appareils
Tout d'abord, jetons un coup d'œil à la page des appareils. Vous pouvez y accéder en [cliquant sur votre application](/docs/webapp/main-page) puis en [cliquant sur l'onglet des appareils](/docs/webapp/main-app-page)

## Afficher uniquement les appareils développeurs
Vous pourriez vouloir afficher uniquement les appareils modifiés (Appareils qui ont un [canal personnalisé](docs/plugin/api/#setchannel) ou une version personnalisée)
Pour cela, cliquez sur `Filters` puis sur `Override`. Cela devrait ressembler à ceci :

## Configuration d'un appareil

Pour configurer un appareil spécifique, cliquez dessus dans le tableau et vous devriez voir quelque chose comme ceci :

<figure><img style="margin-left: auto; margin-right: auto" src="/device-specificwebp" alt="show one device" /><figcaption></figcaption></figure>

D'abord l'`Custom ID`. Ce n'est pas vraiment utile, et pour l'instant cela ne fait rien. Vous pouvez l'ignorer

Ensuite la version forcée. Si elle est définie, cet appareil recevra **TOUJOURS** cette version. Elle prend la priorité sur le canal

Enfin, le canal personnalisé. S'il est défini, cet appareil ne tiendra pas compte du canal public (par défaut) et utilisera uniquement celui-ci
