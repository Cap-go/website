## Afficher uniquement les appareils développeur
Vous pourriez vouloir afficher uniquement les appareils modifiés (Appareils qui ont un [canal personnalisé](docs/plugin/api/#setchannel) ou une version personnalisée)
Pour cela, cliquez sur `Filtres` puis sur `Remplacer`. Cela devrait ressembler à ceci :

## Configuration d'un appareil

Pour configurer un appareil spécifique, cliquez dessus dans le tableau et vous devriez voir quelque chose comme ceci :

<figure><img style="margin-left: auto; margin-right: auto" src="/device-specificwebp" alt="show one device" /><figcaption></figcaption></figure>

D'abord l'`ID personnalisé`. Ce n'est pas vraiment utile, et pour l'instant cela ne fait rien. Vous pouvez l'ignorer

Ensuite la version forcée. Si celle-ci est définie, alors cet appareil recevra **TOUJOURS** cette version. Elle prime sur le canal

Enfin, le canal personnalisé. S'il est défini, alors cet appareil ne tiendra pas compte du canal public (par défaut) et utilisera uniquement celui-ci
