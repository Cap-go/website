## Qu'est-ce que le système d'organisation ?

Le système d'organisation est un système dans capgo qui vous permet de partager en toute sécurité vos applications avec les membres de votre équipe

### Q : Comment puis-je accéder aux informations de mon organisation ?

Pour accéder aux informations de votre organisation, veuillez aller dans [paramètres](/docs/webapp/settings/#how-to-get-to-the-settings-page) puis cliquez sur `Paramètres de l'organisation`

<figure><img style="margin-left: auto; margin-right: auto" src="/orgs-settingswebp" alt="Org settings" /><figcaption></figcaption></figure>

### Q : Comment puis-je changer l'organisation que je consulte ?

Pour voir les paramètres d'une autre organisation, veuillez cliquer sur le sélecteur d'organisation près de votre nom

<figure><img style="margin-left: auto; margin-right: auto" src="/org-selectorwebp" alt="Org selector" /><figcaption></figcaption></figure>

Si vous ne voyez pas cela, vous n'êtes probablement pas sur la page des paramètres

### Q : Comment puis-je voir les membres de mon organisation ?

Veuillez cliquer sur `membres`
<figure><img style="margin-left: auto; margin-right: auto" src="/org-show-memberswebp" alt="Org Show members" /><figcaption></figcaption></figure>

### Q : Comment puis-je inviter un utilisateur dans une organisation ?

Veuillez cliquer sur `Ajouter un membre`
<figure><img style="margin-left: auto; margin-right: auto" src="/orgs-add-memberwebp" alt="Org add member" /><figcaption></figcaption></figure>

Une fenêtre contextuelle apparaîtra - veuillez entrer l'adresse e-mail de l'utilisateur

<figure><img style="margin-left: auto; margin-right: auto" src="/invite-to-org-email-enterwebp" alt="Invite to org" /><figcaption></figcaption></figure>

Puis cliquez sur `inviter` Une autre fenêtre apparaîtra, vous demandant cette fois les permissions que l'utilisateur invité devrait avoir

<figure><img style="margin-left: auto; margin-right: auto" src="/select-perm-orgswebp" alt="Select perm for org" /><figcaption></figcaption></figure>

### Q : Comment fonctionne la facturation au sein des organisations ?

Toute personne ayant une permission de **super admin** peut gérer la facturation pour une organisation donnée
Les plans sont liés à une organisation et non à votre compte personnel

:::danger
⚠️ L'achat d'un plan n'affectera QUE les organisations que vous avez actuellement sélectionnées
:::
