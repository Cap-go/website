`npx @capgo/cli login [apikey]`

Cette commande stocke votre clé API Capgo pour une utilisation ultérieure

:::note
Vous pouvez remplacer la clé API stockée en passant `--apikey=<key>` à n'importe quelle commande
:::

Options:

- `--local`: Stocke la clé API dans le dépôt local et l'ajoute à `gitignore`
