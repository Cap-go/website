`npx @capgo/cli login [apikey]`

Cette commande enregistre votre clé API Capgo pour une utilisation ultérieure

:::note
Vous pouvez remplacer la clé API enregistrée en ajoutant `--apikey=<key>` à n'importe quelle commande
:::

Options:

- `--local`: Enregistre la clé API dans le dépôt local et l'ajoute au `gitignore`
