## Intégration CI

Pour automatiser votre travail, nous recommandons d'utiliser GitHub Actions pour pousser vos mises à jour vers Capgo. Consultez notre [tutoriel GitHub Actions](https://capgo.app/blog/automatic-build-and-release-with-github-actions/) pour plus d'informations

N'oubliez pas de configurer les variables d'environnement de votre CI avec votre clé API Capgo
