`npx @capgo/cli@latest init [apikey]`

Cette commande vous guide étape par étape pour :

- Ajouter votre application à Capgo
- Ajouter le code à votre application pour valider la mise à jour
- Construire votre application
- Téléverser votre application sur Capgo
- Vous aider à vérifier si la mise à jour fonctionne
