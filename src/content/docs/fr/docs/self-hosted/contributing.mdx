## Pourquoi contribuer ?

Tout d'abord, merci d'envisager de contribuer aux projets open source de capgo ! Ce sont des personnes comme vous qui font des projets open source capgo d'excellents outils.

Voici quelques raisons pour lesquelles vous devriez contribuer :

* Contribuer aux projets open source capgo est un excellent moyen de gagner de l'argent grâce aux [nombreuses primes](https://consolealgoraio/org/Capgo/) offertes par l'équipe capgo
* Contribuer aux projets open source capgo est un excellent moyen d'ajouter une fonctionnalité que vous aimeriez voir
* Contribuer aux projets open source capgo est un excellent moyen de corriger un bug que vous avez rencontré
* [La licence principale de Capgo](https://githubcom/Cap-go/capgo/blob/main/LICENSE/) vous oblige à rendre open source toutes les modifications que vous y apportez. En contribuant avec votre code, vous gardez vos modifications open source et permettez à d'autres de les utiliser

## Comment contribuer

* Tout d'abord, vous devez créer un fork du dépôt auquel vous souhaitez contribuer
* Ensuite, vous devez valider et pousser vos modifications vers ce dépôt
* Enfin, vous devez ouvrir une pull request
* C'est tout ! Maintenant, il vous suffit d'attendre que l'équipe capgo examine votre PR
