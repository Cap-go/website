4. **Opérations de base**
   ```typescript
   import { CapacitorDataStorageSqlite } from '@capgo/capacitor-data-storage-sqlite';

   // Définir une valeur

     key: 'username',

   // Obtenir une valeur
   const { value } = await CapacitorDataStorageSqlite.get({
     key: 'username'
   });
   console.log('Username:', value); // "john_doe"

   // Supprimer une valeur
   await CapacitorDataStorageSqlite.remove({
     key: 'username'
   });

   // Effacer toutes les données
   await CapacitorDataStorageSqlite.clear();

   // Vérifier si une clé existe
   const { result } = await CapacitorDataStorageSqlite.iskey({
     key: 'username'
   });
   console.log('Key exists:', result); // true ou false

   // Obtenir toutes les clés
   const { keys } = await CapacitorDataStorageSqlite.keys();
   console.log('All keys:', keys);

   // Obtenir toutes les valeurs
   const { values } = await CapacitorDataStorageSqlite.values();
   ```

5. **Utilisation avancée**
   ```typescript
   import { CapacitorDataStorageSqlite } from '@capgo/capacitor-data-storage-sqlite';

   export class StorageService {
     private dbName = 'app_storage';
     private isEncrypted = false;

     async initialize(encrypted = false) {
       this.isEncrypted = encrypted;

       // Ouvrir le stockage avec des options
       await CapacitorDataStorageSqlite.openStore({
         database: this.dbName,
         encrypted: encrypted,
         mode: encrypted ? 'encryption' : 'no-encryption',
         version: 1
       });
     }

     // Méthodes de stockage génériques
     async setObject<T>(key: string, data: T): Promise<void> {
       const value = JSON.stringify(data);
       await CapacitorDataStorageSqlite.set({ key, value });
     }

     async getObject<T>(key: string): Promise<T | null> {
       try {
         const { value } = await CapacitorDataStorageSqlite.get({ key });
         return value ? JSON.parse(value) : null;
       } catch (error) {
         console.error('Error getting object:', error);
         return null;
       }
     }

     // Opérations par lots
     async setMultiple(items: Record<string, any>): Promise<void> {
       for (const [key, value] of Object.entries(items)) {
         await CapacitorDataStorageSqlite.set({
           key,
           value: typeof value === 'string' ? value : JSON.stringify(value)
         });
       }
     }

     async getMultiple(keys: string[]): Promise<Record<string, any>> {
       const results: Record<string, any> = {};

       for (const key of keys) {
         try {
           const { value } = await CapacitorDataStorageSqlite.get({ key });
           results[key] = value;
         } catch (error) {
           results[key] = null;
         }
       }

       return results;
     }

     // Gestion des tables
     async getTables(): Promise<string[]> {
       const { tables } = await CapacitorDataStorageSqlite.tables();
       return tables;
     }

     async deleteTable(table: string): Promise<void> {
       await CapacitorDataStorageSqlite.deleteTable({ table });
     }

     // Fonctionnalité d'import/export
     async exportToJson(): Promise<any[]> {
       const { keys } = await CapacitorDataStorageSqlite.keys();
       const { values } = await CapacitorDataStorageSqlite.values();

       return keys.map((key, index) => ({
         key,
         value: values[index]
       }));
     }

     async importFromJson(data: Array<{ key: string; value: string }>): Promise<void> {
       // Effacer les données existantes
       await CapacitorDataStorageSqlite.clear();

       // Importer de nouvelles données
       for (const item of data) {
         await CapacitorDataStorageSqlite.set({
           key: item.key,
           value: item.value
         });
       }
     }

     // Filtrage et recherche
     async keysStartingWith(prefix: string): Promise<string[]> {
       const { keys } = await CapacitorDataStorageSqlite.keys();
       return keys.filter(key => key.startsWith(prefix));
     }

     async filterByPrefix(prefix: string): Promise<Array<{ key: string; value: string }>> {
       const { keys } = await CapacitorDataStorageSqlite.keys();
       const { values } = await CapacitorDataStorageSqlite.values();

       const filtered: Array<{ key: string; value: string }> = [];

       keys.forEach((key, index) => {
         if (key.startsWith(prefix)) {
           filtered.push({ key, value: values[index] });
         }
       });

       return filtered;
     }

     // Fermer la base de données lorsque terminé
     async close(): Promise<void> {
       await CapacitorDataStorageSqlite.closeStore({
         database: this.dbName
       });
     }
   }

   // Exemple d'utilisation
   const storage = new StorageService();
   await storage.initialize(true); // Utiliser le chiffrement

   // Stocker les données utilisateur
   await storage.setObject('user_profile', {
     id: 123,
     name: 'John Doe',
     email: 'john@example.com'
   });

   // Récupérer les données utilisateur
   const profile = await storage.getObject<any>('user_profile');
   console.log('User profile:', profile);
   ```

## Référence API

### Méthodes

#### `openStore(options: OpenStoreOptions)`
Ouvrir une base de données de stockage.

**Paramètres :**
- `options.database`: string - Nom de la base de données
- `options.encrypted`: boolean - Activer le chiffrement
- `options.mode`: string - 'encryption' ou 'no-encryption'
- `options.version`: number - Version de la base de données

#### `closeStore(options: CloseStoreOptions)`
Fermer la base de données de stockage.

#### `set(options: SetOptions)`
Stocker une paire clé-valeur.

**Paramètres :**
- `options.key`: string - Clé de stockage
- `options.value`: string - Valeur à stocker

#### `get(options: GetOptions)`
Récupérer une valeur par clé.

**Retourne :** `Promise<{ value: string }>`

#### `remove(options: RemoveOptions)`
Supprimer une paire clé-valeur.

#### `clear()`
Effacer toutes les données du stockage.

#### `iskey(options: IskeyOptions)`
Vérifier si une clé existe.

**Retourne :** `Promise<{ result: boolean }>`

#### `keys()`
Obtenir toutes les clés de stockage.

**Retourne :** `Promise<{ keys: string[] }>`

#### `values()`
Obtenir toutes les valeurs de stockage.

**Retourne :** `Promise<{ values: string[] }>`

#### `tables()`
Obtenir tous les noms de tables.

**Retourne :** `Promise<{ tables: string[] }>`

#### `deleteTable(options: DeleteTableOptions)`
Supprimer une table spécifique.

### Interfaces

```typescript
interface OpenStoreOptions {
  database: string;
  encrypted?: boolean;
  mode?: string;
  version?: number;
}

interface SetOptions {
  key: string;
  value: string;
}

interface GetOptions {
  key: string;
}

interface RemoveOptions {
  key: string;
}
```

## Notes sur les plateformes

### iOS
- Utilise SQLite3 avec SQLCipher optionnel pour le chiffrement
- Les données persistent entre les mises à jour de l'application
- Prend en charge iOS 11.0+

### Android
- Utilise SQLite avec SQLCipher optionnel
- Les données persistent entre les mises à jour de l'application
- Prend en charge Android 5.0 (API 21)+

## Bonnes pratiques

1. **Utiliser le chiffrement pour les données sensibles**
   ```typescript
   // Pour les données sensibles comme les jetons
   await openStore({
     database: 'secure_db',
     encrypted: true,
     mode: 'encryption'
   });
   ```

2. **Organiser les clés avec des préfixes**
   ```typescript
   // Utiliser des préfixes pour l'organisation
   await set({ key: 'user:123:profile', value: userData });
   await set({ key: 'cache:api:users', value: apiData });
   ```

3. **Gérer les données volumineuses avec précaution**
   ```typescript
   // Pour les objets volumineux, envisager la compression
   const compressed = compress(largeData);
   await set({ key: 'large_data', value: compressed });
   ```

4. **Nettoyage régulier**
   ```typescript
   // Supprimer les entrées de cache expirées
   const keys = await keys();
   for (const key of keys.keys) {
     if (key.startsWith('cache:') && isExpired(key)) {
       await remove({ key });
     }
   }
   ```

## Dépannage

**La base de données ne s'ouvre pas :**
- Vérifiez que le nom de la base de données est valide (alphanumérique, underscores)
- Assurez-vous qu'il n'y a pas de caractères spéciaux dans le nom de la base de données
- Vérifiez que le mode de chiffrement correspond à la base de données existante

**Les données ne persistent pas :**
- Assurez-vous que `openStore` est appelé avant les opérations
- Vérifiez les erreurs dans la console
- Vérifiez que les noms de clés sont des chaînes

**Problèmes de performance :**
- Évitez de stocker de très grandes valeurs
- Utilisez des opérations par lots lorsque possible
- Envisagez d'utiliser plusieurs bases de données pour différents types de données
