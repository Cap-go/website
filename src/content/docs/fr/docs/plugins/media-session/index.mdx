import { Card, CardGrid } from '@astrojs/starlight/components';

Le plugin Media Session fait le pont entre Capacitor et l'API Media Session pour que vos lecteurs audio et vidéo se sentent natifs partout.

<CardGrid stagger>
  <Card title="Métadonnées riches" icon="star">
    Fournissez une pochette d'album, un artiste et un titre pour les contrôles de l'écran de verrouillage et des notifications.
  </Card>
  <Card title="Synchronisation de l'état de lecture" icon="rocket">
    Maintenez les boutons lecture/pause et les indicateurs de progression alignés avec votre lecteur.
  </Card>
  <Card title="Callbacks d'action" icon="pencil">
    Gérez les boutons du casque et les actions multimédias du système d'exploitation depuis votre code Capacitor.
  </Card>
  <Card title="Rapport de position" icon="download">
    Transmettez la durée, la position et le débit pour que le défilement fonctionne de manière fluide.
  </Card>
</CardGrid>

Passez au guide de démarrage pour intégrer les mises à jour de métadonnées dans le cycle de vie de votre lecteur.
