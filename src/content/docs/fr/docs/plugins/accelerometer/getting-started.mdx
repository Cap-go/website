Ce guide vous guidera dans l'intégration du plugin Capacitor Accelerometer dans votre application.

## Installation

Installez le plugin en utilisant npm :

```bash
npm install @capgo/capacitor-accelerometer
npx cap sync
```

## Configuration de la plateforme

### iOS
Aucune configuration supplémentaire requise. L'accéléromètre est toujours disponible.

### Android
Aucune configuration supplémentaire requise. L'accéléromètre est toujours disponible.

### Web
Le plugin utilise l'API DeviceMotion. Nécessite HTTPS en production.

## Utilisation de base

### Importer le plugin

```typescript
import { Accelerometer } from '@capgo/capacitor-accelerometer';
```

### Démarrer la surveillance

```typescript
const startAccelerometer = async () => {
  await Accelerometer.start({
    interval: 100 // Intervalle de mise à jour en millisecondes
  });

  console.log('Accelerometer started');
};
```

### Écouter les événements d'accélération

```typescript
Accelerometer.addListener('accelerationChange', (data) => {
  console.log('X:', data.x);
  console.log('Y:', data.y);
  console.log('Z:', data.z);
  console.log('Timestamp:', data.timestamp);
});
```

### Obtenir la lecture actuelle

```typescript
const getCurrentAcceleration = async () => {
  const reading = await Accelerometer.getCurrentAcceleration();
  console.log('Current acceleration:', reading);
};
```

### Arrêter la surveillance

```typescript
const stopAccelerometer = async () => {
  await Accelerometer.stop();
  console.log('Accelerometer stopped');
};
```

## Exemple complet

Voici un exemple complet avec détection de secousses :

```typescript
import { Accelerometer } from '@capgo/capacitor-accelerometer';

class AccelerometerService {
  private listener: any;
  private lastX = 0;
  private lastY = 0;
  private lastZ = 0;
  private shakeThreshold = 15;

  async initialize() {
    await Accelerometer.start({ interval: 100 });

    this.listener = Accelerometer.addListener('accelerationChange', (data) => {
      this.handleAcceleration(data);
    });
  }

  handleAcceleration(data: any) {
    // Calculer le delta
    const deltaX = Math.abs(data.x - this.lastX);
    const deltaY = Math.abs(data.y - this.lastY);
    const deltaZ = Math.abs(data.z - this.lastZ);

    // Vérifier les secousses
    if (deltaX > this.shakeThreshold ||
        deltaY > this.shakeThreshold ||
        deltaZ > this.shakeThreshold) {
      this.onShake();
    }

    // Mettre à jour les dernières valeurs
    this.lastX = data.x;
    this.lastY = data.y;
    this.lastZ = data.z;

    // Mettre à jour l'interface utilisateur
    this.updateDisplay(data);
  }

  onShake() {
    console.log('Device shaken!');
    // Déclencher l'action de secousse
  }

  updateDisplay(data: any) {
    console.log(`X: ${data.x.toFixed(2)} m/s²`);
    console.log(`Y: ${data.y.toFixed(2)} m/s²`);
    console.log(`Z: ${data.z.toFixed(2)} m/s²`);

    // Calculer l'amplitude
    const magnitude = Math.sqrt(
      data.x * data.x +
      data.y * data.y +
      data.z * data.z
    );
    console.log(`Magnitude: ${magnitude.toFixed(2)} m/s²`);
  }

  async cleanup() {
    if (this.listener) {
      this.listener.remove();
    }
    await Accelerometer.stop();
  }
}

// Utilisation
const accelService = new AccelerometerService();
accelService.initialize();

// Nettoyer quand terminé
// accelService.cleanup();
```

## Comprendre les lectures

### Axes d'accélération
- **Axe X** : Gauche (-) à Droite (+)
- **Axe Y** : Bas (-) à Haut (+)
- **Axe Z** : Arrière (-) à Avant (+)

### Gravité
- Un appareil au repos affiche ~9,8 m/s² sur un axe (gravité)
- Le déplacement de l'appareil montre l'accélération en plus de la gravité

### Unités
- Mesuré en mètres par seconde carrée (m/s²)
- Gravité = 9,8 m/s²

## Cas d'utilisation courants

### Détection de secousses

```typescript
class ShakeDetector {
  private lastUpdate = 0;
  private lastX = 0;
  private lastY = 0;
  private lastZ = 0;

  detectShake(x: number, y: number, z: number): boolean {
    const currentTime = Date.now();

    if (currentTime - this.lastUpdate > 100) {
      const deltaX = Math.abs(x - this.lastX);
      const deltaY = Math.abs(y - this.lastY);
      const deltaZ = Math.abs(z - this.lastZ);

      this.lastUpdate = currentTime;
      this.lastX = x;
      this.lastY = y;
      this.lastZ = z;

      return deltaX + deltaY + deltaZ > 15;
    }

    return false;
  }
}
```

### Détection d'inclinaison

```typescript
class TiltDetector {
  getTiltAngles(x: number, y: number, z: number) {
    const roll = Math.atan2(y, z) * (180 / Math.PI);
    const pitch = Math.atan2(-x, Math.sqrt(y * y + z * z)) * (180 / Math.PI);

    return { roll, pitch };
  }

  isDeviceFlat(z: number): boolean {
    return Math.abs(z - 9.8) < 1.0;
  }

  isDeviceUpright(y: number): boolean {
    return Math.abs(y - 9.8) < 2.0;
  }
}
```

### Compteur de pas

```typescript
class StepCounter {
  private steps = 0;
  private lastMagnitude = 0;
  private threshold = 11;

  processAcceleration(x: number, y: number, z: number) {
    const magnitude = Math.sqrt(x * x + y * y + z * z);

    if (magnitude > this.threshold &&
        this.lastMagnitude < this.threshold) {
      this.steps++;
      console.log('Steps:', this.steps);
    }

    this.lastMagnitude = magnitude;
  }
}
```

## Meilleures pratiques

1. **Choisir des intervalles appropriés** : Équilibrer la réactivité et la durée de vie de la batterie
   - Jeux : 16-50ms
   - Fitness : 100-200ms
   - Général : 200-500ms

2. **Supprimer les écouteurs** : Toujours nettoyer lorsque terminé
3. **Filtrer le bruit** : Utiliser des moyennes mobiles pour des données plus fluides
4. **Considérer la batterie** : L'interrogation à haute fréquence draine la batterie
5. **Tester sur de vrais appareils** : Les simulateurs ne fournissent pas de données précises

## Conseils de performance

### Anti-rebond

```typescript
class AccelerometerDebouncer {
  private timeout: any;

  debounce(callback: Function, delay: number) {
    return (...args: any[]) => {
      clearTimeout(this.timeout);
      this.timeout = setTimeout(() => callback(...args), delay);
    };
  }
}
```

### Lissage des données

```typescript
class AccelerometerFilter {
  private alpha = 0.8;
  private filteredX = 0;
  private filteredY = 0;
  private filteredZ = 0;

  filter(x: number, y: number, z: number) {
    this.filteredX = this.alpha * x + (1 - this.alpha) * this.filteredX;
    this.filteredY = this.alpha * y + (1 - this.alpha) * this.filteredY;
    this.filteredZ = this.alpha * z + (1 - this.alpha) * this.filteredZ;

    return {
      x: this.filteredX,
      y: this.filteredY,
      z: this.filteredZ
    };
  }
}
```
