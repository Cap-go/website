import { Card, CardGrid } from '@astrojs/starlight/components';

Capgo Pay relie les derniers SDK Apple Pay et Google Pay afin que vous puissiez offrir une expérience de paiement transparente sur tous les appareils.

<CardGrid stagger>
  <Card title="API unifiée" icon="pencil">
    Appelez `isPayAvailable` et `requestPayment` une fois et laissez le plugin gérer chaque plateforme.
  </Card>
  <Card title="Configuration rapide" icon="rocket">
    Déployez des feuilles de paiement entièrement natives sans câbler manuellement du code spécifique à la plateforme.
  </Card>
  <Card title="Paiements tokenisés" icon="approve-check">
    Recevez des tokens de paiement prêts pour vos intégrations de passerelle ou PSP.
  </Card>
  <Card title="Flux personnalisables" icon="star">
    Configurez les éléments de ligne, les champs de contact et les exigences de facturation pour adapter votre paiement.
  </Card>
</CardGrid>

Plongez dans le guide de démarrage pour les listes de contrôle de configuration et les exemples de charges utiles pour chaque plateforme.
