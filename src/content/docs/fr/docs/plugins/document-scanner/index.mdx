import { Card, CardGrid } from '@astrojs/starlight/components';

Le plugin Document Scanner offre à votre application Capacitor un flux de numérisation de qualité native qui fonctionne sur les appareils iOS et Android.

<CardGrid stagger>
  <Card title="Détection automatique" icon="rocket">
    Détectez les bords de page, recadrez automatiquement et laissez les utilisateurs ajuster avant d'enregistrer.
  </Card>
  <Card title="Capture multi-pages" icon="star">
    Capturez une ou plusieurs pages dans une seule session de numérisation avec des limites ajustables.
  </Card>
  <Card title="Sortie personnalisée" icon="download">
    Retournez soit des chemins de fichiers, soit des chaînes base64 à la qualité d'image de votre choix.
  </Card>
  <Card title="API unifiée" icon="pencil">
    Une interface TypeScript couvre les implémentations iOS et Android.
  </Card>
</CardGrid>

Utilisez le guide de démarrage pour activer les permissions de caméra, configurer les paramètres de qualité et exporter les fichiers numérisés vers votre flux de stockage.
