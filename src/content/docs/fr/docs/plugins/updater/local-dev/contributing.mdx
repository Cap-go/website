## Pourquoi contribuer ?

Tout d'abord, merci d'envisager de contribuer aux projets open source de capgo ! Ce sont des personnes comme vous qui font des projets open source de capgo d'excellents outils.

Voici quelques raisons pour lesquelles vous pourriez envisager de contribuer :

* Contribuer aux projets open source de capgo est un excellent moyen de gagner de l'argent grâce aux [innombrables primes](https://console.algora.io/org/Capgo) offertes par l'équipe capgo
* Contribuer aux projets open source de capgo est un excellent moyen d'ajouter une fonctionnalité que vous aimeriez voir
* Contribuer aux projets open source de capgo est un excellent moyen de corriger un bug que vous avez rencontré
* [La licence principale de Capgo](https://github.com/Cap-go/capgo/blob/main/LICENSE) vous oblige à rendre open source toute modification que vous y apportez. En contribuant votre code, vous pouvez garder vos modifications open source et laisser les autres les utiliser

## Comment contribuer

 * Tout d'abord, vous devez forker le dépôt auquel vous souhaitez contribuer
 * Deuxièmement, vous devez commiter et pousser vos modifications vers ce dépôt
 * Enfin, vous devez ouvrir une pull request
 * C'est tout ! Maintenant, vous devez juste attendre que l'équipe capgo examine votre PR
