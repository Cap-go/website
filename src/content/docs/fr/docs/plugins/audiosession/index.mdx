import { Card, CardGrid } from '@astrojs/starlight/components';

Le plugin AudioSession de Capgo vous offre un contrôle précis sur la couche AVAudioSession d'iOS afin que vous puissiez gérer gracieusement les écouteurs, les appareils Bluetooth et les interruptions inattendues.

<CardGrid stagger>
  <Card title="Conscience de la route" icon="rocket">
    Détectez quand les utilisateurs branchent ou débranchent des casques, AirPods, enceintes Bluetooth, etc.
  </Card>
  <Card title="Forçage du haut-parleur" icon="setting">
    Forcez la lecture via le haut-parleur intégré lorsque vous avez besoin d'audio mains libres.
  </Card>
  <Card title="Gestion des interruptions" icon="moon">
    Réagissez aux événements d'interruption tels que les appels téléphoniques ou Siri pour mettre en pause et reprendre l'audio en toute sécurité.
  </Card>
  <Card title="Sécurité des types" icon="pencil">
    Des définitions TypeScript solides facilitent la création d'expériences audio résilientes.
  </Card>
</CardGrid>

Ce plugin cible iOS où le contrôle AVAudioSession est requis. Sur les autres plateformes, les appels se résolvent sans effet, vous permettant de conserver un seul chemin de code partagé.
