## Générer à partir de HTML

```typescript
import { PdfGenerator } from '@capgo/capacitor-pdf-generator';

const receiptHtml = `
  <html>
    <body>
      <h1>Capgo Store</h1>
      <p>Merci pour votre achat.</p>
    </body>
  </html>
`;

const pdf = await PdfGenerator.fromData({
  data: receiptHtml,
  documentSize: 'A4',
  orientation: 'portrait',
  type: 'base64',
  fileName: 'recu.pdf',
});

if (pdf.type === 'base64') {
  const link = document.createElement('a');
  link.href = `data:application/pdf;base64,${pdf.base64}`;
  link.download = 'recu.pdf';
  link.click();
}
```

## Capturer une URL distante

```typescript
await PdfGenerator.fromURL({
  url: 'https://docs.capgo.app/invoice?id=123',
  orientation: 'landscape',
  documentSize: 'A4',
  type: 'share', // ouvre la boîte de dialogue de partage native
  fileName: 'facture-123.pdf',
});
```

## Conseils

- Incluez du CSS en ligne dans votre chaîne HTML ou fournissez une `baseUrl` pour que les ressources relatives se résolvent correctement.
- Lors du retour de données base64 sur mobile, déplacez-les vers le système de fichiers (par exemple, Capacitor Filesystem) avant de les partager.
- L'orientation paysage est idéale pour les tableaux larges, tandis que le portrait convient à la plupart des documents.
- Envisagez d'utiliser des polices CSS à portée limitée (par exemple, polices système) pour un rendu cohérent sur toutes les plateformes.
