import { Card, CardGrid } from '@astrojs/starlight/components';

Le plugin AdMob de Capgo offre un pont de premier ordre vers le SDK Google Mobile Ads afin que vous puissiez monétiser les expériences natives sans sacrifier la simplicité de Capacitor.

<CardGrid stagger>
  <Card title="Formats d'annonces flexibles" icon="star">
    Diffusez des annonces bannière, interstitielles, récompensées et d'ouverture d'application avec une API unifiée.
  </Card>
  <Card title="Configuration des requêtes" icon="setting">
    Ajustez finement les requêtes d'annonces avec des drapeaux pour enfants, des évaluations de contenu et des paramètres réseau supplémentaires.
  </Card>
  <Card title="Gestion du cycle de vie" icon="rocket">
    Créez, chargez, affichez, masquez et détruisez les annonces à la demande tout en suivant l'état de chargement.
  </Card>
  <Card title="Prêt pour la confidentialité" icon="approve-check">
    Gérez les invites ATT et surveillez le statut d'autorisation de suivi pour rester conforme.
  </Card>
</CardGrid>

Utilisez le guide de démarrage pour configurer vos ID d'application AdMob, configurer le consentement et gérer l'inventaire publicitaire directement depuis votre projet Capacitor.
