## Qu'est-ce que la 2FA ?

2FA est une mesure de sécurité conçue pour empêcher un acteur malveillant de s'emparer de votre compte capgo.
Il y parvient en exigeant un code supplémentaire. Ce code est stocké sur votre téléphone mobile ou sur une clé matérielle 2FA. Cela change toutes les 30 secondes, ce qui rend impossible de le deviner

## Exigences pour ce guide
 - Un téléphone Android ou IOS
 - Connexion Internet

## Que couvrira ce guide ?

Ce guide montrera comment configurer 2FA à l'aide de `Google Authenticator`. D'autres applications existent pour remplir un objectif similaire, mais je ne peux pas aborder l'intégralité du sujet de la 2FA dans ce guide.

## Comment configurer 2FA ?

Téléchargez d’abord l’application `Google Authenticator`. Si vous êtes sur Android, vous pouvez le télécharger depuis le [Play Store](https://play.google.com/store/apps/details/?id=com.google.android.apps.authenticator2). Sur IOS, vous pouvez le télécharger depuis [App Store](https://apps.apple.com/us/app/google-authenticator/id388497605/)

Deuxièmement, veuillez accéder aux [paramètres du compte capgo](https://console.capgo.app/dashboard/settings/account/). Là, vous devriez voir un bouton vert qui devrait ressembler à ceci

<figure><img style="margin-left: auto; margin-right: auto" src="/button-mfa.webp" alt="Bouton MFA" /><figcaption></figcaption></figure>

Cliquez dessus, et une fois que vous l'avez fait, un code QR devrait apparaître

<figure><img style="margin-left: auto; margin-right: auto" src="/enable-mfa.webp" alt="Activer l'authentification multifacteur" /><figcaption></figcaption></figure>

:::danger
⚠️ Remarque importante : \
Ne partagez jamais ce code QR avec qui que ce soit, sinon vous pourriez être déconnecté de votre compte
:::

Ensuite, veuillez ouvrir le `Google Authenticator` sur votre téléphone. Une fois que vous l'avez fait, suivez ces étapes :

Cliquez sur le bouton plus

<figure><img style="margin-left: auto; margin-right: auto" src="/mfa-auth-plus.webp" alt="Authentification MFA plus" /><figcaption></figcaption></figure>

Après cela, veuillez cliquer sur le bouton de l'appareil photo

<figure><img style="margin-left: auto; margin-right: auto" src="/mfa-scan-qr.webp" alt="Scanner MFA QR" /><figcaption></figcaption></figure>

Cela ouvrira l'aperçu de la caméra. Veuillez scanner le code QR affiché sur votre PC. Après cela, vous devriez voir quelque chose comme ceci :

<figure><img style="margin-left: auto; margin-right: auto" src="/mfa-final-code.webp" alt="Code final MFA" /><figcaption></figcaption></figure>

Ensuite, retournez au PC et cliquez sur le bouton de vérification.

<figure><img style="margin-left: auto; margin-right: auto" src="/enable-mfa-verify.webp" alt="Vérifier l'AMF" /><figcaption></figcaption></figure>

Cela ouvrira une fenêtre pour saisir notre code 2FA. Dans mon cas, ce code est `095101` mais ce sera différent pour vous.\
Après avoir saisi ce code, veuillez cliquer sur le bouton `verify`

<figure><img style="margin-left: auto; margin-right: auto" src="/mfa-verify-final-code.webp" alt="Code de vérification MFA final" /><figcaption></figcaption></figure>

Si vous avez entré le bon code et appuyé sur `verify`, vous devriez voir une fenêtre contextuelle comme celle-ci

<figure><img style="margin-left: auto; margin-right: auto" src="/mfa-enabled.webp" alt="MFA activé" /><figcaption></figcaption></figure>

Félicitations !\
Vous avez activé l'authentification 2FA 🎉

## Comment se connecter en utilisant 2FA

La prochaine fois que vous vous connecterez à votre compte, vous verrez une fenêtre comme celle-ci
<figure><img style="margin-left: auto; margin-right: auto" src="/mfa-required.webp" alt="MFA requis" /><figcaption></figcaption></figure>

Veuillez ouvrir l'authentificateur et copier votre code d'authentification

<figure><img style="margin-left: auto; margin-right: auto" src="/mfa-login.webp" alt="Connexion MFA" /><figcaption></figcaption></figure>

:::danger
⚠️ Remarque importante : \
Appuyez sur `verify` avant l'actualisation du code, sinon le code changera et l'ancien ne sera plus valide
:::

Saisissez ensuite le code dans le formulaire de connexion et appuyez sur `verify`

<figure><img style="margin-left: auto; margin-right: auto" src="/mfa-final-login-verify.webp" alt="Connexion MFA" /><figcaption></figcaption></figure>

Si vous avez entré le bon code, vous devriez voir le tableau de bord capgo

## Comment désactiver 2FAPour désactiver 2FA, veuillez accéder aux [paramètres du compte capgo](https://console.capgo.app/dashboard/settings/account/). Là, vous devriez voir un bouton rouge qui devrait ressembler à ceci :

<figure><img style="margin-left: auto; margin-right: auto" src="/mfa-disable-button.webp" alt="Bouton de désactivation MFA" /><figcaption></figcaption></figure>

Cliquez dessus et vous devriez voir un écran comme celui-ci

<figure><img style="margin-left: auto; margin-right: auto" src="/mfa-disable-2.webp" alt="MFA désactivé" /><figcaption></figcaption></figure>

Appuyez simplement sur le bouton `disable` et c'est tout.
