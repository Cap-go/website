:::note[Conservation des Données]
Les données des appareils sont conservées pendant **90 jours** à compter de la dernière activité de l'appareil. Les appareils qui ne se sont pas connectés à Capgo pendant cette période seront automatiquement supprimés de la liste. Les appareils actifs sont continuellement suivis lorsqu'ils vérifient les mises à jour.
:::

## Afficher la liste de tous les appareils
Commençons par regarder la page des appareils. Vous pouvez y accéder en [cliquant sur votre application](/docs/webapp/main-page) puis en [cliquant sur l'onglet appareils](/docs/webapp/main-app-page).

<figure><img style="margin-left: auto; margin-right: auto" src="/devices.webp" alt="liste des appareils" /><figcaption></figcaption></figure>

1. **Onglet Appareils** - Cliquez sur cet onglet dans la barre de navigation pour accéder à la page des appareils
2. **Filtres** - Cliquez pour filtrer les appareils par :
   - **Substitution** - Affiche uniquement les appareils qui ont un [canal personnalisé](/docs/plugin/api/#setchannel) ou une version personnalisée
   - **ID Personnalisé** - Affiche uniquement les appareils avec un identifiant personnalisé défini

## Configurer un appareil

Pour configurer un appareil spécifique, cliquez dessus dans le tableau et vous devriez voir quelque chose comme ceci :

<figure><img style="margin-left: auto; margin-right: auto" src="/device-specific.webp" alt="afficher un appareil" /><figcaption></figcaption></figure>

1. **Information** - Affiche les détails de l'appareil incluant l'ID de l'appareil, Dernière mise à jour, Plateforme, Version du plugin, Version, Version builtin, Version de l'OS, et si c'est une application de Production
2. **Deployments** - Affiche l'historique des déploiements pour cet appareil
3. **Logs** - Affiche les journaux associés à cet appareil
4. **Substitution de canal** - Remplace le canal pour cet appareil spécifique. Si défini, cet appareil n'utilisera pas le canal public (par défaut) et ne recevra que les mises à jour du canal sélectionné

### ID Personnalisé

L'`ID Personnalisé` est un champ très utile qui vous aide à connecter l'appareil à un ID que vous pouvez reconnaître pour vos propres utilisateurs.

Utilisez-le pour identifier facilement quel appareil appartient à quel utilisateur.

L'ID personnalisé ne peut être défini que depuis l'appareil lui-même en utilisant `CapacitorUpdater.setCustomId({ customId: user });`
