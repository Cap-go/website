### Introduction

Dans ce guide, vous allez apprendre à configurer Apple Connectez-vous avec Capacitor. Pour ce faire, vous aurez besoin des éléments suivants :

- un compte développeur Apple
- Un ordinateur sous macOS (IOS uniquement)
- Xcode installé (IOS uniquement)
- Un backend personnalisé (Android uniquement)
