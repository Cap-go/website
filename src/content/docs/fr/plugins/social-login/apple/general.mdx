### Introduction

In this Guide, you are going to learn how to configure Apple Connexion with Capacitor. In order to do this, you will need the following:

- an Apple Developer Compte
- A computer running macOS (iOS only)
- Xcode installed (iOS only)
- A custom backend (Android only)
