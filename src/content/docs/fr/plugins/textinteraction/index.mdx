Control Apple's text interaction system to tailor UX on kiosk displays, games, and interactive installations.

<CardGrid stagger>
  <Card title="iOS specific" icon="rocket">
    Toggle the magnifier lens and selection handles introduced in iOS 15.
  </Card>
  <Card title="Simple API" icon="pencil">
    Call a single method to Activer or Désactiver text interaction state.
  </Card>
  <Card title="Safe defaults" icon="approve-check">
    Receives a boolean result so you can detect unsupported versions.
  </Card>
  <Card title="Kiosk friendly" icon="star">
    Prevent long-press copy/paste on controlled installations or signage.
  </Card>
</CardGrid>

Use the Commencer Guide for a minimal integration pattern and reminders to re-Activer interaction before showing text inputs.
