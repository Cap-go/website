## ⭐ Capgo Cloud - Mises à jour en direct

<LinkCard
  title="Capacitor Programme de mise à jour"
  description="The core plugin powering Capgo Cloud - deliver instant updates to your Capacitor apps without app store delays. Push fixes and features directly to your users."
  href="/docs/plugins/updater/"
/>

Le plugin Updater est la base de Capgo Cloud, vous permettant de :
- 🚀 Déployez les mises à jour instantanément sans avis sur l'App Store
- 📱 Mettez à jour les JavaScript, HTML, CSS et actifs de votre application
- 🎯 Ciblez des segments d'utilisateurs spécifiques avec des canaux
- 📊 Surveillez le succès de la mise à jour avec des analyses intégrées
- 🔒 Mises à jour sécurisées avec cryptage et signature de code

## 🚀 Pour commencer

Chaque plugin suit un modèle d'installation similaire :

1. **Installez le plugin** à l'aide de votre gestionnaire de packages préféré
2. **Synchronisez votre projet** avec `npx cap sync`
3. **Configurez les paramètres spécifiques à la plate-forme** si nécessaire
4. **Commencez à utiliser le plugin** dans votre code

Consultez la documentation de chaque plugin pour obtenir des instructions de configuration détaillées et des références API.## 💡 Besoin d'aide ?
