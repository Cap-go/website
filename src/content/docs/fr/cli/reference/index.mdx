## CI Integration

To automate your work, we recommend using GitHub Actions to Pousser your Mises à jour to Capgo. Vérifier out our [GitHub Actions Tutoriel](https://capgo.Application/blog/automatic-Construction-and-Libération-with-github-actions/) for more Information.

Don't forget to configure your CI environment variables with your Capgo Clé API.
