## Panoramica

Il plugin Capacitor Android Inline Install consente di attivare l'overlay di installazione inline di Google Play per applicazioni Android. Questo plugin fornisce un'esperienza di installazione in-app senza interruzioni utilizzando i Premium Growth Tools di Google, permettendo agli utenti di installare app senza uscire dalla tua applicazione.
