## Filosofia

Il team Ionic mantiene Capacitor con le proprie priorità e programma di rilascio. I contributi della community possono aspettare mesi o addirittura anni per essere integrati. Capacitor+ adotta un approccio diverso:

1. **Integra PR dai Fork** - PR preziose bloccate nella coda upstream vengono attivamente integrate
2. **Sync Continuo** - Ogni modifica upstream viene automaticamente prelevata e testata
3. **Rilasci Rapidi** - Le modifiche vengono pubblicate su npm non appena passano CI
4. **Community-First** - I tuoi contributi contano e vengono prioritizzati
5. **Trasparenza** - Tutta l'automazione è open source e visibile
