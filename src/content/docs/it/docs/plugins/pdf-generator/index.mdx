Converti HTML o pagine remote in PDF condivisibili con il renderer nativo di Capgo per iOS e Android.

<CardGrid stagger>
  <Card title="HTML in PDF" icon="pencil">
    Inserisci template HTML e ottieni un documento PDF pronto da condividere in pochi secondi.
  </Card>
  <Card title="Cattura remota" icon="rocket">
    Cattura qualsiasi URL accessibile e salva o distribuisci il PDF generato.
  </Card>
  <Card title="Output flessibili" icon="star">
    Restituisci dati base64 o attiva la schermata di condivisione nativa a seconda del tuo flusso di lavoro.
  </Card>
  <Card title="Controlli documento" icon="setting">
    Specifica orientamento, dimensione pagina e nomi file per ogni rendering.
  </Card>
</CardGrid>
