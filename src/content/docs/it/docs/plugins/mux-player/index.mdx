Offri una riproduzione premium avvolgendo gli SDK nativi ufficiali Mux in una semplice interfaccia Capacitor.

<CardGrid stagger>
  <Card title="Esperienza SDK nativa" icon="rocket">
    Usa lo stesso stack di riproduzione degli SDK mobili Mux con bitrate adattivo e DRM.
  </Card>
  <Card title="Riproduzione firmata" icon="approve-check">
    Fornisci token di riproduzione e DRM per la distribuzione sicura di stream protetti.
  </Card>
  <Card title="API uniforme" icon="pencil">
    Attiva la riproduzione, chiudi il player e ascolta gli eventi con un'unica API TypeScript.
  </Card>
  <Card title="Pronto per l'analisi" icon="star">
    Passa chiavi di ambiente e nomi di player per alimentare automaticamente la telemetria Mux Data.
  </Card>
</CardGrid>

La guida introduttiva illustra come collegare gli SDK nativi, gestire i token e rispondere agli eventi del player.
