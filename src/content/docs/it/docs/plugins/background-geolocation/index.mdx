## Confronto Plugin

Un breve confronto tra i tre principali plugin di background-geolocation comunemente usati nelle app Capacitor.

| Plugin | Accuratezza | Background | Upload HTTP | Prezzo |
|--------|----------|------------|-------------|---------|
| `@capacitor-community/background-geolocation` (Community) | Non accurato | Sì | No | Gratuito |
| `@capgo/background-geolocation` (questo plugin) | Accurato | Sì | No | Gratuito |
| Transistorsoft (originale) | Accurato | Sì | Sì — uploader HTTP integrato alla tua API | A pagamento |

**Note:**
- Il plugin Community è leggero e continua a funzionare in background, ma è noto per essere meno accurato delle opzioni sottostanti.
- Questo plugin Capgo è ottimizzato per fornire fix di posizione accurati e operazioni in background affidabili senza richiedere una licenza a pagamento.
- Transistorsoft offre una soluzione commerciale completa che include anche capacità di upload HTTP integrate per inviare dati di posizione al tuo backend.
