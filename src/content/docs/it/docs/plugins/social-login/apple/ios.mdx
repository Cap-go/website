import { Steps } from '@astrojs/starlight/components';

Analizziamo ciò di cui avrai bisogno per configurare l'accesso Apple su IOS.

<Steps>
1. Configura le funzionalità della tua app.

    Per fare ciò, apri Xcode, fai clic su `App`
    ![App XCode](/social-login-assets/app-xcode.png)

2. Assicurati di selezionare il target giusto.

    ![Destinazione app XCode](/social-login-assets/xcode-app-target.png)

3. Assicurati di aggiungere la funzionalità `Sign in with Apple`.

    ![Funzionalità di aggiunta App XCode](/social-login-assets/app-xcode-add-cap.png)

    ![Funzionalità di aggiunta App XCode](/social-login-assets/sign-with-apple-cap.png)

    :::attenzione
    Se non vedi la funzionalità `Sign in with Apple`, configura [Condivisione dati account e organizzazione](https://developer.apple.com/account/resources/services/cwa/configure/)
    :::

4. Inizializza l'accesso Apple nella tua app.

    :::nota
    Sto utilizzando Vue come framework, l'implementazione esatta varierà a seconda del framework scelto
    :::

    ```ts
    //  onMounted is vue specific
    onMounted(() => {
    SocialLogin.initialize({
        apple: {}
    })
    });
    ```

5. Crea un pulsante che avvierà il processo di accesso.

    Detto pulsante dovrebbe richiamare la seguente funzione:

    ```ts
    async function loginApple() {
    const res = await SocialLogin.login({
        provider: 'apple',
        options: {}
    })
    ```

6. Esegui la tua app su un dispositivo **<u>PHYSICAL</u>** e testala. 

    Se hai seguito attentamente i passaggi, dopo aver fatto clic sul pulsante vedrai la seguente schermata.

    ![Apple Richiesta di accesso sul dispositivo iOS](/social-login-assets/apple-sign-in-ios-final.png)
</Steps>

Questo è tutto! È tutto pronto.
