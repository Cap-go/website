## Integrazione CI

Per automatizzare il tuo lavoro, ti consigliamo di utilizzare GitHub Actions per inviare i tuoi aggiornamenti a Capgo. Consulta il nostro [tutorial su GitHub Actions](https://capgo.app/blog/automatic-build-and-release-with-github-actions/) per maggiori informazioni

Non dimenticare di configurare le variabili d'ambiente del tuo CI con la tua chiave API Capgo

## App Demo

Per un esempio completo di un'app Capgo con integrazione CI, dai un'occhiata alla nostra [app demo su GitHub](https://github.com/Cap-go/demo-app/)
