`npx @capgo/cli login [apikey]`

Questo comando memorizza la tua chiave API Capgo per usi futuri

:::note
Puoi sovrascrivere la chiave API memorizzata passando `--apikey=<key>` a qualsiasi comando
:::

Opzioni:

- `--local`: Memorizza la chiave API nel repository locale e la aggiunge a `gitignore`
