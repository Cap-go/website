`npx @capgo/cli login [apikey]`

Questo comando memorizza la tua API key di Capgo per usi futuri

:::note
Puoi sovrascrivere l'API key memorizzata passando `--apikey=<key>` a qualsiasi comando
:::

Opzioni:

- `--local`: Memorizza l'API key nel repository locale e la aggiunge al `gitignore`
