## Perché contribuire?

Prima di tutto, grazie per aver considerato di contribuire ai progetti open source di capgo! Sono le persone come te che rendono i progetti open source di capgo degli strumenti così eccezionali.

Ecco alcuni motivi per cui dovresti contribuire:

* Contribuire ai progetti open source di capgo è un ottimo modo per guadagnare denaro dalle [numerose bounty](https://consolealgoraio/org/Capgo/) offerte dal team capgo
* Contribuire ai progetti open source di capgo è un ottimo modo per aggiungere una funzionalità che vorresti vedere
* Contribuire ai progetti open source di capgo è un ottimo modo per correggere un bug che hai incontrato
* La [licenza principale di Capgo](https://githubcom/Cap-go/capgo/blob/main/LICENSE/) richiede di rendere open source qualsiasi modifica tu faccia. Contribuendo con il tuo codice, mantieni le tue modifiche open source e permetti ad altri di utilizzarle

## Come contribuire

1. Prima di tutto, devi fare il fork del repository al quale vuoi contribuire
2. In secondo luogo devi fare il commit e il push delle tue modifiche su quel repository
3. Infine, devi aprire una pull request
4. È tutto! Ora devi solo attendere che il team capgo esamini la tua PR
