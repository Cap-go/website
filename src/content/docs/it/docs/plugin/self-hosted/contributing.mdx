## Perché contribuire?

Innanzitutto, grazie per aver considerato di contribuire ai progetti open source di capgo! Sono le persone come te che rendono i progetti open source di capgo degli strumenti così straordinari.

Ecco alcuni motivi per cui potresti considerare di contribuire:

* Contribuire ai progetti open source di capgo è un ottimo modo per guadagnare denaro dalle [numerose ricompense](https://consolealgoraio/org/Capgo) offerte dal team capgo
* Contribuire ai progetti open source di capgo è un ottimo modo per aggiungere una funzionalità che vorresti vedere
* Contribuire ai progetti open source di capgo è un ottimo modo per correggere un bug che hai riscontrato
* La [licenza principale di Capgo](https://githubcom/Cap-go/capgo/blob/main/LICENSE) richiede di rendere open source qualsiasi modifica tu apporti. Contribuendo con il tuo codice, puoi mantenere le tue modifiche open source e permettere ad altri di utilizzarle

## Come contribuire

* Prima di tutto, devi fare il fork del repository a cui vuoi contribuire
* In secondo luogo devi committare e pushare le tue modifiche su quel repository
* Infine, devi aprire una pull request
* È tutto! Ora devi solo attendere che il team capgo revisioni la tua PR
