クイックスタートガイドを完了したことで、Capgoの主要な概念について基本的な理解が得られたはずです。このガイドで学んだ主要な概念は以下の通りです：

<Steps>
1 Capgoアカウントへのアプリの追加

2 CI/CDパイプラインとCapgoの統合 

3 新しいコミットでCapgoへのバンドルアップロードをトリガー

4 Capgo SDKでライブアップデートを有効にするためのアプリの設定

5 Capgoダッシュボードからアプリへのライブアップデートのデプロイ
</Steps>
