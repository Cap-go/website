## 組織

このエンドポイントでは、組織とそのメンバーを管理できます

### GET

`https://apicapgoapp/organization`

組織情報を取得します。パラメータで`orgId`が提供された場合は単一の組織を返し、そうでない場合はアクセス可能なすべての組織を返します

```typescript
interface Organization {
  id: string
  created_by: string
  created_at: string
  updated_at: string
  logo: string | null
  name: string
  management_email: string
  customer_id: string | null
}
```

戻り値:
```typescript
return Organization[] | Organization
```

### POST

`https://apicapgoapp/organization`

既存の組織を更新します。管理者権限が必要です

```typescript
interface OrganizationUpdate {
  orgId: string
  logo?: string
  name?: string
  management_email?: string
}
```

`OrganizationUpdate`型のリクエストボディが必要です

成功時、以下を返します：

```typescript
return { status: 'Organization updated', data: Organization }
```

### PUT

`https://apicapgoapp/organization`

新しい組織を作成します

```typescript
interface OrganizationCreate {
  name: string
}
```

`OrganizationCreate`型のリクエストボディが必要です

成功時、以下を返します：

```typescript
return { status: 'Organization created', id: string }
```

## APIキー

このエンドポイントでは、Capgo APIにアクセスするためのAPIキーを管理できます

### GET

`https://apicapgoapp/apikey`

アカウントのAPIキーを取得します。アクセス可能なすべてのAPIキーを返します

```typescript
interface ApiKey {
  created_at: string | null
  id: number
  key: string
  mode: 'read' | 'write' | 'upload' | 'all'
  name: string
  updated_at: string | null
  user_id: string
}
```

戻り値:
```typescript
return ApiKey[]
```

### POST

`https://apicapgoapp/apikey`

特定の組織の新しいAPIキーを作成します

クエリパラメータ:
```typescript
interface ApiKeyCreate {
  org_id: string
  mode: 'read' | 'write' | 'upload' | 'all'
}
```

戻り値:
```typescript
return { apikey: ApiKey }
```

### DELETE

`https://apicapgoapp/apikey/:id`

既存のAPIキーを削除します

パラメータ:
- `id`: 削除するAPIキー

成功時の戻り値:
```typescript
return { success: true }
```

失敗時の戻り値:
```typescript
return { error: string, supabaseError?: any }
```

## メンバー (/organization/members)

### GET 

組織メンバーを取得します

`{ orgId: string }`型のリクエストボディ（クエリパラメータ）が必要です

成功時、以下を返します:
```typescript
interface Member {
  uid: string;
  email: string;
  image_url: string;
  role: "invite_read" | "invite_upload" | "invite_write" | "invite_admin" | "invite_super_admin" | "read" | "upload" | "write" | "admin" | "super_admin";
}
return Member[]
```

### DELETE

`https://apicapgoapp/organization/members`

組織メンバーを削除します

```typescript
interface MemberDelete {
  orgId: string
  email: string
}
```

`MemberDelete`型のリクエストボディ（クエリパラメータ）が必要です

成功時、以下を返します:

```typescript
return { status: 'OK' }
```

失敗時、以下を返します:

```typescript
return { error: string, status: 'KO' }
```

## 統計情報

このエンドポイントでは、アプリと組織に関する様々な統計情報を取得できます

### GET /statistics/app/:app_id

特定のアプリの統計情報を取得します

クエリパラメータ:
```typescript
interface StatsQuery {
  from: Date
  to: Date
  graph?: 'mau' | 'storage' | 'bandwidth'
}
```

戻り値:
```typescript
interface Stats {
  date: string
  mau: number      // 月間アクティブユーザー
  storage: number  // ストレージ使用量（バイト）
  bandwidth: number // 帯域幅使用量（バイト）
}
```

`graph`パラメータが提供された場合、要求された指標のSVG画像の可視化を返します

### GET /statistics/user

ユーザーがアクセスできるすべての組織の集計統計を取得します

クエリパラメータ:
```typescript
interface StatsQuery {
  from: Date
  to: Date
  graph?: 'mau' | 'storage' | 'bandwidth'
}
```

戻り値:
`graph`が提供されない場合:
```typescript
interface Stats {
  date: string
  mau: number      // 月間アクティブユーザー
  storage: number  // ストレージ使用量（バイト）
  bandwidth: number // 帯域幅使用量（バイト）
}[]
```

`graph`パラメータが提供された場合、要求された指標のSVG画像の可視化を返します

### GET /statistics/org/:org_id

特定の組織の統計情報を取得します

クエリパラメータ:
```typescript
interface StatsQuery {
  from: Date
  to: Date
  graph?: 'mau' | 'storage' | 'bandwidth'
}
```

戻り値:
`graph`が提供されない場合:
```typescript
interface Stats {
  date: string
  mau: number      // 月間アクティブユーザー
  storage: number  // ストレージ使用量（バイト）
  bandwidth: number // 帯域幅使用量（バイト）
}[]
```

`graph`パラメータが提供された場合、要求された指標のSVG画像の可視化を返します

## チャンネル

このエンドポイントでは、アプリの利用可能なすべてのチャンネルを確認および変更できます

### POST

`https://apicapgoapp/channel`

チャンネルを作成または更新するために以下を送信します

```typescript
type disable_update = "major" | "minor" | "version_number" | "none"
interface ChannelSet {
  app_id: string
  channel: string
  version?: string
  public?: boolean
  disableAutoUpdateUnderNative?: boolean
  disableAutoUpdate?: disable_update
  ios?: boolean
  android?: boolean
  allow_device_self_set?: boolean
  allow_emulator?: boolean
  allow_dev?: boolean
}
```

以下を受信します:

```json
{ "status": "ok" }
```

### GET

`https://apicapgoapp/channel`

URLパラメータとして`app_id`を送信すると、最初の50チャンネルの配列を受信します\
`page=1`を送信することで次のチャンネルを取得できます

```typescript
interface Channel {
    id: number;
    created_at: string;
    name: string;
    app_id: string;
    version: {
        id: number,
        name: string
    };
    created_by: string;
    updated_at: string;
    public: boolean; // デフォルトかどうか
    disableAutoUpdateUnderNative: boolean;
    disableAutoUpdate: boolean;
    allow_emulator: boolean;
    allow_dev: boolean;
}[]
```

オプションでURLパラメータとして`channel`名を送信すると、1つのチャンネルを受信します:

```typescript
interface Channel {
    id: number;
    created_at: string;
    name: string;
    app_id: string;
    version: {
        id: number,
        name: string
    };
    created_by: string;
    updated_at: string;
    public: boolean; // デフォルトかどうか
    disableAutoUpdateUnderNative: boolean;
    disableAutoUpdate: boolean;
    allow_emulator: boolean;
    allow_dev: boolean;
}
```

### DELETE

`https://apicapgoapp/channel`

以下をクエリパラメータとして送信します

```typescript
interface Channel {
  channel: string
  app_id: string
}
```

以下を受信します:

```json
{ "status": "ok" }
```

## デバイス

このエンドポイントでは、アプリにリンクされたすべてのデバイスを確認および変更できます

### POST

`https://apicapgoapp/device`

以下を送信します

```typescript
interface DeviceLink {
  app_id: string
  device_id: string
  version_id?: string // バージョン名（より良い名前に移行予定）
  channel?: string // チャンネル名
}
```

以下を受信します:

```json
{ "status": "ok" }
```

### GET

`https://apicapgoapp/device`

URLパラメータとして`app_id`を送信すると、最初の50デバイスの配列を受信します`page=1`を送信することで次のものを取得できます

```typescript
interface Device{
    created_at?: string | undefined;
    updated_at?: string | undefined;
    device_id: string;
    custom_id: string;
    version: {
        id: number,
        name: string
    };
    app_id: string;
    platform?: "ios" | "android" | undefined;
    plugin_version: string;
    os_version?: string | undefined;
    version_build: string;
    is_prod: boolean;
    is_emulator: boolean;
}[]       
```

オプションでURLパラメーターとして`device_id`を送信すると、1つを受け取れます:

```typescript
interface Device {
    created_at?: string | undefined;
    updated_at?: string | undefined;
    device_id: string;
    version: {
        id: number,
        name: string
    };
    app_id: string;
    platform?: "ios" | "android" | undefined;
    plugin_version: string;
    os_version?: string | undefined;
}
```

### DELETE

このエンドポイントはチャンネルとバージョンオーバーライドのリンクを解除するためのものです。Capgoからデバイスを削除することはできません

`https://apicapgoapp/device`

これをクエリパラメーターとして送信

```typescript
interface Device {
  device_id: string
  app_id: string
}
```

これを受信:

```json
{ "status": "ok" }
```

## バンドル

このエンドポイントではアプリにリンクされているすべてのバンドルを確認・削除できます

### GET

`https://apicapgoapp/bundle`

URLパラメーターとして`app_id`を送信すると、最初の50バンドルの配列を受信します\
`page=1`を送信することで次のものを取得できます

```typescript
interface Bundle {
  app_id: string
  bucket_id: string | null
  checksum: string | null
  created_at: string | null
  deleted: boolean
  external_url: string | null
  id: number
  minUpdateVersion: string | null
  name: string
  native_packages: Json[] | null
  owner_org: string
  r2_path: string | null
  session_key: string | null
  storage_provider: string
  updated_at: string | null
  user_id: string | null
}[]
```

### DELETE

クラウド内の1つのバンドルを削除

`https://apicapgoapp/bundle`

これをクエリパラメーターとして送信

```typescript
interface Bundle {
  app_id: string
  version: string
}
```

これを受信:

```json
{ "status": "ok" }
```

あるいは以下を送信することもできます

```typescript
interface Bundle {
  app_id: string
}
```
