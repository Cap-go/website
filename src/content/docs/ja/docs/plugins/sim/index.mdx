import { Card, CardGrid } from '@astrojs/starlight/components';

Capgo SIMを使用して、Capacitor内でキャリアデータ、スロットインデックス、パーミッション状態をクエリします。

<CardGrid stagger>
  <Card title="マルチSIMサポート" icon="rocket">
    スロットインデックスとサブスクリプションIDとともに、インストールされているすべてのSIMを列挙。
  </Card>
  <Card title="パーミッションヘルパー" icon="approve-check">
    Androidで必要な電話パーミッションをチェックしてリクエスト。
  </Card>
  <Card title="一貫したスキーマ" icon="star">
    MCC、MNC、ISO国、電話番号などの正規化されたキャリアフィールドで作業。
  </Card>
  <Card title="iOS対応" icon="pencil">
    iOS 16.4以降の限定的なキャリアデータなどのプラットフォームのニュアンスに対応。
  </Card>
</CardGrid>

パーミッションと利用パターンを統合するには、スタートガイドをご覧ください。
