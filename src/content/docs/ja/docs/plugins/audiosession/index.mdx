import { Card, CardGrid } from '@astrojs/starlight/components';

CapgoのAudioSessionプラグインは、iOS AVAudioSessionレイヤーをきめ細かく制御できるため、ヘッドフォン、Bluetoothデバイス、予期しない中断を適切に処理できます。

<CardGrid stagger>
  <Card title="ルート認識" icon="rocket">
    ユーザーがヘッドセット、AirPods、Bluetoothスピーカーなどを接続または取り外したときを検出します。
  </Card>
  <Card title="スピーカーオーバーライド" icon="setting">
    ハンズフリーオーディオが必要な場合に、内蔵スピーカーを通じて強制的に再生します。
  </Card>
  <Card title="中断処理" icon="moon">
    電話やSiriなどの中断イベントに対応して、オーディオを安全に一時停止および再開します。
  </Card>
  <Card title="型安全性" icon="pencil">
    強力なTypeScript定義により、堅牢なオーディオエクスペリエンスを簡単に構築できます。
  </Card>
</CardGrid>

このプラグインは、AVAudioSessionの制御が必要なiOSを対象としています。他のプラットフォームでは、呼び出しは効果なく解決され、1つの共有コードパスを維持できます。
