## 概要

Capacitor Downloaderプラグインは、バックグラウンドダウンロード、進行状況追跡、包括的なダウンロード管理のサポートを備えた強力なファイルダウンロード機能を提供します。このプラグインは、一時停止/再開機能と詳細な進行状況監視を備えた堅牢なファイルダウンロードを可能にします。

> ⚠️ **開発中**: このプラグインは現在開発中であり、まだ本番環境での使用準備ができていません。

このプラグインはreact-native-background-downloaderにインスパイアされており、現在開発中です。開発が進むにつれて機能が変更される可能性があります。
