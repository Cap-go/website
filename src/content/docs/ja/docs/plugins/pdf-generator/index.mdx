import { Card, CardGrid } from '@astrojs/starlight/components';

iOS と Android 用の Capgo のネイティブレンダラーで、HTML またはリモートページを共有可能な PDF に変換します。

<CardGrid stagger>
  <Card title="HTML から PDF へ" icon="pencil">
    HTML テンプレートを入力して、数秒で共有可能な PDF ドキュメントを取得します。
  </Card>
  <Card title="リモートキャプチャ" icon="rocket">
    アクセス可能な任意の URL のスナップショットを作成し、生成された PDF を保存または配布します。
  </Card>
  <Card title="柔軟な出力" icon="star">
    ワークフローに応じて、base64 データを返すか、ネイティブ共有シートをトリガーします。
  </Card>
  <Card title="ドキュメント制御" icon="setting">
    各レンダリングの向き、ページサイズ、ファイル名を指定します。
  </Card>
</CardGrid>

プラットフォームノートと使用例については、はじめにガイドをご覧ください。
