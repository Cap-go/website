## プラグイン比較

Capacitorアプリで一般的に使用される3つの主要なバックグラウンド位置情報プラグインの簡単な比較。

| プラグイン | 精度 | バックグラウンド | HTTP アップロード | 価格 |
|--------|----------|------------|-------------|---------|
| `@capacitor-community/background-geolocation` (コミュニティ) | 不正確 | はい | いいえ | 無料 |
| `@capgo/background-geolocation` (このプラグイン) | 正確 | はい | いいえ | 無料 |
| Transistorsoft (オリジナル) | 正確 | はい | はい — API への組み込みHTTPアップローダー | 有料 |

**注意:**
- コミュニティプラグインは軽量で、バックグラウンドで動作し続けますが、以下のオプションよりも精度が低いことが知られています。
- このCapgoプラグインは、有料ライセンスを必要とせずに、正確な位置情報修正と信頼性の高いバックグラウンド動作を提供するように最適化されています。
- Transistorsoftは、バックエンドに位置情報データを送信するための組み込みHTTPアップロード機能も含む完全な商用ソリューションを提供しています。
