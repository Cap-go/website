## なぜ貢献するのか？

まず第一に、capgoのオープンソースプロジェクトへの貢献を検討していただきありがとうございます！capgoのオープンソースプロジェクトを素晴らしいツールにしているのは、あなたのような人々です。

貢献を検討する理由は次のとおりです:

* capgoのオープンソースプロジェクトに貢献することは、capgoチームが提供する[数え切れないほどの報奨金](https://console.algora.io/org/Capgo)からお金を稼ぐ素晴らしい方法です
* capgoのオープンソースプロジェクトに貢献することは、見たい機能を追加する素晴らしい方法です
* capgoのオープンソースプロジェクトに貢献することは、遭遇したバグを修正する素晴らしい方法です
* [Capgo mainのライセンス](https://github.com/Cap-go/capgo/blob/main/LICENSE)では、変更をオープンソース化する必要があります。コードを貢献することで、変更をオープンソースのままにして、他の人がそれを使用できるようにすることができます

## 貢献する方法

 * まず、貢献したいリポジトリをフォークする必要があります
 * 次に、そのリポジトリに変更をコミットしてプッシュする必要があります
 * 最後に、プルリクエストを開く必要があります
 * それだけです！あとはcapgoチームがあなたのPRをレビューするのを待つだけです
