`CapacitorUpdater.notifyAppReady()`呼び出しの配置は、正しい自動アップデートフローにとって重要です。


正しく行われない場合、アプリは最新バージョンにアップデートできません。10秒以内に`notifyAppReady()`を呼び出さない各バージョンは無効とマークされ、以前の有効なバージョンまたはデフォルト（組み込み）バージョンに置き換えられます。


このガイドでは、正しい自動アップデートフローを確保するために、アプリで`notifyAppReady()`呼び出しを適切に配置する方法を説明します。
