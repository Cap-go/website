import { Card, CardGrid } from '@astrojs/starlight/components';

公式Muxネイティブ SDKをシンプルなCapacitorインターフェースでラップすることで、プレミアム再生を提供します。

<CardGrid stagger>
  <Card title="ネイティブSDKエクスペリエンス" icon="rocket">
    アダプティブビットレートとDRMを備えたMuxモバイルSDKと同じ再生スタックを使用します。
  </Card>
  <Card title="署名済み再生" icon="approve-check">
    保護されたストリームの安全な配信のために、再生およびDRMトークンを提供します。
  </Card>
  <Card title="統一API" icon="pencil">
    1つのTypeScript APIで再生をトリガー、プレーヤーを閉じ、イベントをリスンします。
  </Card>
  <Card title="分析対応" icon="star">
    Mux Dataテレメトリーを自動的にフィードするために環境キーとプレーヤー名を渡します。
  </Card>
</CardGrid>

はじめにガイドでは、ネイティブSDKの接続、トークンの処理、プレーヤーイベントへの応答について説明します。
