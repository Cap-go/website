iOSでApple loginを設定するために必要なことを説明します。

<Steps>
1. アプリのcapabilityを設定します。

    これを行うには、Xcodeを開き、`App`をクリックしてください。
    ![App XCode](/social-login-assets/app-xcode.png)

2. 正しいtargetが選択されていることを確認してください。

    ![XCode App Target](/social-login-assets/xcode-app-target.png)

3. `Sign in with Apple` capabilityを追加してください。

    ![App XCode Add Capability](/social-login-assets/app-xcode-add-cap.png)

    ![App XCode Add Capability](/social-login-assets/sign-with-apple-cap.png)

    :::caution
    `Sign in with Apple` capabilityが表示されない場合は、[Account & Organizational Data Sharing](https://developer.apple.com/account/resources/services/cwa/configure/)を設定してください。
    :::

4. アプリでApple Loginを初期化します。

    :::note
    私はフレームワークとしてVueを使用しています。実際の実装は、選択したフレームワークによって異なります。
    :::

    ```ts
    //  onMountedはVue固有のものです
    onMounted(() => {
    SocialLogin.initialize({
        apple: {}
    })
    });
    ```

5. ログインプロセスを開始するボタンを作成します。

    そのボタンは次の関数を呼び出す必要があります:

    ```ts
    async function loginApple() {
    const res = await SocialLogin.login({
        provider: 'apple',
        options: {}
    })
    ```

6. **<u>実機</u>**でアプリを実行してテストしてください。

    手順を正しく実行した場合、ボタンをクリックした後に次の画面が表示されます。

    ![Apple Sign In prompt on iOS device](/social-login-assets/apple-sign-in-ios-final.png)
</Steps>

これで完了です！設定はすべて完了しました。
