## 哲学

Ionicチームは、独自の優先順位とリリーススケジュールでCapacitorをメンテナンスしています。コミュニティからの貢献は、マージされるまでに数ヶ月、場合によっては数年待つことがあります。Capacitor+は異なるアプローチを取ります:

1. **フォークからPRをマージ** - アップストリームキューで停滞している価値あるPRを積極的にマージ
2. **継続的な同期** - すべてのアップストリームの変更は自動的にプルされテストされます
3. **迅速なリリース** - 変更はCIをパスするとすぐにnpmに公開されます
4. **コミュニティファースト** - あなたの貢献が重要で優先されます
5. **透明性** - すべての自動化はオープンソースで可視化されています
