Capgo Capacitor Plugins コレクションへようこそ！Capacitor アプリで素晴らしいネイティブエクスペリエンスを構築するための高品質で充実したドキュメントを備えたプラグインスイートを提供しています。

## ⭐ Capgo Cloud - ライブアップデート

<LinkCard
  title="Capacitor Updater"
  description="Capgo Cloud を支えるコアプラグイン - アプリストアの遅延なしに Capacitor アプリへの即座のアップデートを提供します。修正と機能をユーザーに直接プッシュできます。"
  href="/docs/plugins/updater/"
/>

Updater プラグインは Capgo Cloud の基盤であり、以下を可能にします：
- 🚀 アプリストアのレビューなしで即座にアップデートをデプロイ
- 📱 アプリの JavaScript、HTML、CSS、アセットをアップデート
- 🎯 チャンネルで特定のユーザーセグメントをターゲット
- 📊 組み込み分析でアップデートの成功を監視
- 🔒 暗号化とコード署名で安全なアップデート

## 🚀 はじめに

各プラグインは同様のインストールパターンに従います：

1. **プラグインをインストール** お好みのパッケージマネージャーを使用
2. **プロジェクトを同期** `npx cap sync` を実行
3. **プラットフォーム固有の設定を構成** 必要に応じて
4. **プラグインの使用を開始** コード内で

詳細なセットアップ手順と API リファレンスについては、各プラグインのドキュメントをご覧ください。
