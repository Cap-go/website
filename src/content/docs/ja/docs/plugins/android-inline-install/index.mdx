## 概要

Capacitor Android Inline Installプラグインは、AndroidアプリケーションのGoogle PlayのInline Installオーバーレイをトリガーできるようにします。このプラグインは、GoogleのPremium Growth Toolsを使用してシームレスなアプリ内インストール体験を提供し、ユーザーがアプリを離れることなくアプリをインストールできるようにします。
