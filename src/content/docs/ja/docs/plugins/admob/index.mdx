import { Card, CardGrid } from '@astrojs/starlight/components';

CapgoのAdMobプラグインは、Capacitorのシンプルさを犠牲にすることなくネイティブエクスペリエンスを収益化できるように、Google Mobile Ads SDKへのファーストクラスのブリッジを提供します。

<CardGrid stagger>
  <Card title="柔軟な広告フォーマット" icon="star">
    統一されたAPIでバナー、インタースティシャル、リワード、アプリオープン広告を配信。
  </Card>
  <Card title="リクエスト構成" icon="setting">
    児童向けフラグ、コンテンツ評価、追加のネットワークパラメータで広告リクエストを微調整。
  </Card>
  <Card title="ライフサイクル管理" icon="rocket">
    読み込み状態を追跡しながら、オンデマンドで広告を作成、読み込み、表示、非表示、破棄。
  </Card>
  <Card title="プライバシー対応" icon="approve-check">
    ATTプロンプトを処理し、追跡認証ステータスを監視してコンプライアンスを維持。
  </Card>
</CardGrid>

はじめにガイドを使用して、AdMobアプリID、同意の設定、CapacitorプロジェクトからJavaScript直接で広告インベントリを管理する方法を学びましょう。
