import { Card, CardGrid } from '@astrojs/starlight/components';

Document ScannerプラグインはCapacitorアプリにiOSとAndroidデバイスの両方で動作するネイティブ品質のスキャンフローを提供します。

<CardGrid stagger>
  <Card title="自動検出" icon="rocket">
    ページのエッジを検出し、自動的にクロップし、保存前にユーザーが調整できるようにします。
  </Card>
  <Card title="複数ページキャプチャ" icon="star">
    調整可能な制限を持つ1つのスキャンセッションで1つまたは多くのページをキャプチャします。
  </Card>
  <Card title="カスタム出力" icon="download">
    選択した画質でファイルパスまたはbase64文字列のいずれかを返します。
  </Card>
  <Card title="統一API" icon="pencil">
    1つのTypeScriptインターフェースがiOSとAndroidの両方の実装にまたがります。
  </Card>
</CardGrid>

はじめるガイドを使用して、カメラパーミッションを有効にし、品質設定を構成し、スキャンしたファイルをストレージワークフローにエクスポートします。
