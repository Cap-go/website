Media SessionプラグインはCapacitorをMedia Session APIと橋渡しし、オーディオおよびビデオプレーヤーがどこでもネイティブに感じられるようにします。

<CardGrid stagger>
  <Card title="リッチメタデータ" icon="star">
    ロック画面と通知コントロールのためのアルバムアート、アーティスト、タイトルを提供します。
  </Card>
  <Card title="再生状態の同期" icon="rocket">
    再生/一時停止ボタンと進行状況インジケーターをプレーヤーと整合させます。
  </Card>
  <Card title="アクションコールバック" icon="pencil">
    CapacitorコードからヘッドセットボタンとOSメディアアクションを処理します。
  </Card>
  <Card title="位置レポート" icon="download">
    スクラビングがスムーズに機能するように、期間、位置、レートをプッシュします。
  </Card>
</CardGrid>

プレーヤーのライフサイクルにメタデータ更新を組み込むために、はじめにガイドにジャンプしてください。
