Capgo Photo Library は、アルバムのリスト、アセットのページング、サムネイルのリクエストを行いながら、パフォーマンスをチェックするのに役立ちます。

<CardGrid stagger>
  <Card title="認証ヘルパー" icon="approve-check">
    パーミッションをリクエストし、制限付きアクセス状態を適切に処理します。
  </Card>
  <Card title="ページング取得" icon="rocket">
    カーソルベースの読み込みとカスタマイズ可能な制限でギャラリーをページ分割します。
  </Card>
  <Card title="最適化されたサムネイル" icon="star">
    スムーズなスクロールリストのために、固定サイズでサムネイル URL を取得します。
  </Card>
  <Card title="メディア選択" icon="download">
    設定可能な選択制限でユーザーに写真やビデオを選択させます。
  </Card>
</CardGrid>

パーミッションの設定とギャラリーの例については、はじめにガイドをご覧ください。
