import { Card, CardGrid } from '@astrojs/starlight/components';

キオスクディスプレイ、ゲーム、インタラクティブインスタレーションでUXを調整するために、Appleのテキストインタラクションシステムを制御します。

<CardGrid stagger>
  <Card title="iOS固有" icon="rocket">
    iOS 15で導入された拡大鏡レンズと選択ハンドルを切り替えます。
  </Card>
  <Card title="シンプルなAPI" icon="pencil">
    単一のメソッドを呼び出してテキストインタラクション状態を有効または無効にします。
  </Card>
  <Card title="安全なデフォルト" icon="approve-check">
    サポートされていないバージョンを検出できるようにブール結果を受け取ります。
  </Card>
  <Card title="キオスクフレンドリー" icon="star">
    制御されたインストールまたはサイネージでの長押しコピー/ペーストを防止します。
  </Card>
</CardGrid>

最小限の統合パターンとテキスト入力を表示する前にインタラクションを再有効化するためのリマインダーについては、はじめにガイドを使用してください。
