## CI統合

作業を自動化するために、GitHub Actionsを使用してCapgoに更新をプッシュすることをお勧めします。詳細については、[GitHub Actionsチュートリアル](https://capgo.app/blog/automatic-build-and-release-with-github-actions/)をご覧ください。

CI環境変数にCapgo APIキーを設定することを忘れないでください。

## デモアプリ

CI統合を含むCapgoアプリの完全な例については、[GitHubのデモアプリ](https://github.com/Cap-go/demo-app/)をご確認ください。
