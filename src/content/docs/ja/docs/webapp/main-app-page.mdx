## メインページには何が表示されますか？

まず、アプリのメインページを見てみましょう：

<figure><img src="/main-app-pagewebp" alt="Main page screeshot" /><figcaption></figcaption></figure>

詳しく見ていきましょう

まず統計情報については、現在改修中のため、この時点では文書化できません。完全に運用可能になった時点で統計セクションを追加する予定です。

次に、利用可能なすべてのメニューとその機能を見ていきましょう

<figure><img src="/app-multiple-sectionswebp" alt="Multiple sections screeshot" /><figcaption></figcaption></figure>

最初のボタンで[チャンネル](/docs/webapp/channels/)ページに移動します。そこでアプリのチャンネルの作成/削除/設定ができます\
2番目のボタンで[バンドル](/docs/webapp/bundles/)ページに移動します。そこでアプリの全バージョンを確認できます
2番目のボタンで[デバイス](/docs/webapp/devices/)ページに移動します。そこですべてのデバイスを確認できます。また、デバイス固有の上書き設定もそこで行えます
2番目のボタンで[アップデート](/docs/webapp/logs/)ページに移動します。そこでアプリで発生している詳細な統計情報（ログ）とエラーを確認できます
