import { Card, CardGrid } from '@astrojs/starlight/components';

Deliver premium playback by wrapping the official Mux native SDKs in a simple Capacitor interface.

<CardGrid stagger>
  <Card title="Native SDK experience" icon="rocket">
    Use the same playback stack as the Mux mobile SDKs with adaptive bitrate and DRM.
  </Card>
  <Card title="Signed playback" icon="shield">
    Provide playback and DRM tokens for secure delivery of protected streams.
  </Card>
  <Card title="Uniform API" icon="code">
    Trigger playback, dismiss the player, and listen to events with one TypeScript API.
  </Card>
  <Card title="Analytics ready" icon="chart">
    Pass environment keys and player names to feed Mux Data telemetry automatically.
  </Card>
</CardGrid>

The getting started guide walks through hooking up the native SDKs, handling tokens, and responding to player events.
