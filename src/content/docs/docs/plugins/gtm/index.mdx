## Usage

```typescript
import { CapacitorGTM } from '@capgo/capacitor-gtm';

// Initialize GTM
await CapacitorGTM.initialize({
  containerId: 'GTM-XXXXXX'
});

// Track an event
await CapacitorGTM.trackEvent({
  event: 'purchase',
  parameters: {
    value: 99.99,
    currency: 'USD',
    transaction_id: '12345'
  }
});
```
