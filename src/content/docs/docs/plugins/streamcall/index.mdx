## Configuration

### iOS Configuration

Add camera and microphone permissions to your `Info.plist`:

```xml
<key>NSCameraUsageDescription</key>
<string>This app needs access to camera for video calls</string>
<key>NSMicrophoneUsageDescription</key>
<string>This app needs access to microphone for calls</string>
```

### Android Configuration

Add permissions to your `AndroidManifest.xml`:

```xml
<uses-permission android:name="android.permission.CAMERA" />
<uses-permission android:name="android.permission.RECORD_AUDIO" />
<uses-permission android:name="android.permission.INTERNET" />
```

## Usage

```typescript
import { StreamCall } from '@capgo/capacitor-streamcall';

// Initialize a call
const call = await StreamCall.initCall({
  roomId: 'room-123',
  userId: 'user-456'
});

// Start streaming
await StreamCall.startStream();

// End call
await StreamCall.endCall();
```
