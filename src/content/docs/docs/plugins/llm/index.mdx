## Usage

```typescript
import { CapacitorLLM } from '@capgo/capacitor-llm';

// Initialize the LLM
await CapacitorLLM.initialize({
  modelPath: 'path/to/model'
});

// Generate text
const result = await CapacitorLLM.generate({
  prompt: 'What is the meaning of life?',
  maxTokens: 100
});

console.log(result.text);
```
