The Capgo Health plugin unifies HealthKit and Health Connect access so you can track activity, biometric metrics, and body measurements with one Capacitor integration.

<CardGrid stagger>
  <Card title="Cross-platform health data" icon="rocket">
    Support Apple HealthKit and Android Health Connect out of the box.
  </Card>
  <Card title="Granular permissions" icon="approve-check">
    Request separate read/write scopes and react to authorization changes.
  </Card>
  <Card title="Consistent units" icon="star">
    Work with normalized data types like `steps`, `heartRate`, and `weight`.
  </Card>
  <Card title="Real-time writes" icon="rocket">
    Save samples back to the user's health store when they log activity.
  </Card>
</CardGrid>

Follow the getting started guide to enable platform capabilities, gather consent, and start syncing metrics.
