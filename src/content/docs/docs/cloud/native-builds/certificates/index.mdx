## What You'll Need

### For iOS

| Item | File Type | Purpose |
|------|-----------|---------|
| Signing Certificate | `.p12` | Authenticates you as a developer |
| Provisioning Profile | `.mobileprovision` | Links app, certificate, and devices |
| App Store Connect API Key | `.p8` | Automatic TestFlight/App Store submission |

### For Android

| Item | File Type | Purpose |
|------|-----------|---------|
| Signing Keystore | `.keystore` / `.jks` | Signs your app for release |
| Google Play Service Account | `.json` | Automatic Play Store uploads |
