## Filosofi

Tim Ionic memelihara Capacitor dengan prioritas dan jadwal rilis mereka sendiri. Kontribusi komunitas bisa menunggu berbulan-bulan atau bahkan bertahun-tahun untuk digabungkan. Capacitor+ mengambil pendekatan yang berbeda:

1. **Gabungkan PR dari Fork** - PR berharga yang terjebak di antrian upstream secara aktif digabungkan
2. **Sinkronisasi Berkelanjutan** - Setiap perubahan upstream secara otomatis ditarik dan diuji
3. **Rilis Cepat** - Perubahan dipublikasikan ke npm segera setelah lulus CI
4. **Komunitas Pertama** - Kontribusi Anda penting dan diprioritaskan
5. **Transparansi** - Semua otomasi bersifat open source dan terlihat
