import { Card, CardGrid } from '@astrojs/starlight/components';

Plugin AdMob Capgo memberikan jembatan kelas satu ke Google Mobile Ads SDK sehingga Anda dapat memonetisasi pengalaman native tanpa mengorbankan kesederhanaan Capacitor.

<CardGrid stagger>
  <Card title="Format iklan fleksibel" icon="star">
    Tampilkan iklan banner, interstitial, rewarded, dan app open dengan API terpadu.
  </Card>
  <Card title="Konfigurasi request" icon="setting">
    Sesuaikan request iklan dengan flag child-directed, rating konten, dan parameter jaringan tambahan.
  </Card>
  <Card title="Manajemen lifecycle" icon="rocket">
    Buat, muat, tampilkan, sembunyikan, dan hancurkan iklan sesuai permintaan sambil melacak status pemuatan.
  </Card>
  <Card title="Siap privasi" icon="approve-check">
    Tangani prompt ATT dan pantau status otorisasi tracking untuk tetap patuh.
  </Card>
</CardGrid>

Gunakan panduan memulai untuk menghubungkan ID aplikasi AdMob Anda, mengonfigurasi persetujuan, dan mengelola inventaris iklan langsung dari proyek Capacitor Anda.
