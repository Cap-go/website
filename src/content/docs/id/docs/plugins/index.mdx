Setiap plugin mengikuti pola instalasi yang serupa:

1. **Instal plugin** menggunakan package manager pilihan Anda
2. **Sinkronkan project Anda** dengan `npx cap sync`
3. **Konfigurasi pengaturan khusus platform** jika diperlukan

Kunjungi dokumentasi setiap plugin untuk instruksi setup terperinci dan referensi API.
