Capgo Pay bridges the latest Apple Pay and Google Pay SDKs so you can deliver one seamless checkout experience across devices.

<CardGrid stagger>
  <Card title="Unified API" icon="pencil">
    Call `isPayAvailable` and `requestPayment` once and let the plugin handle each platform.
  </Card>
  <Card title="Rapid setup" icon="rocket">
    Ship fully native payment sheets without manually wiring platform-specific code.
  </Card>
  <Card title="Tokenized payments" icon="approve-check">
    Receive payment tokens ready for your gateway or PSP integrations.
  </Card>
  <Card title="Customizable flows" icon="star">
    Configure line items, contact fields, and billing requirements to fit your checkout.
  </Card>
</CardGrid>

Dive into the getting started guide for setup checklists and sample payloads for each platform.
