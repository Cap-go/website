Kontrol sistem interaksi teks Apple untuk menyesuaikan UX pada display kiosk, game, dan instalasi interaktif.

<CardGrid stagger>
  <Card title="Khusus iOS" icon="rocket">
    Toggle lensa pembesar dan pegangan pemilihan yang diperkenalkan di iOS 15.
  </Card>
  <Card title="API Sederhana" icon="pencil">
    Panggil satu metode untuk mengaktifkan atau menonaktifkan status interaksi teks.
  </Card>
  <Card title="Default yang aman" icon="approve-check">
    Menerima hasil boolean sehingga Anda dapat mendeteksi versi yang tidak didukung.
  </Card>
  <Card title="Ramah kiosk" icon="star">
    Mencegah salin/tempel long-press pada instalasi atau signage terkontrol.
  </Card>
</CardGrid>

Gunakan panduan memulai untuk pola integrasi minimal dan pengingat untuk mengaktifkan kembali interaksi sebelum menampilkan input teks.
