### Pendahuluan

Dalam panduan ini, Anda akan belajar cara mengkonfigurasi Login Apple dengan Capacitor. Untuk melakukan ini, Anda akan memerlukan hal berikut:

- akun Apple Developer
- Komputer yang menjalankan macOS (hanya IOS)
- Xcode terinstal (hanya IOS)
- Backend khusus (hanya Android)
