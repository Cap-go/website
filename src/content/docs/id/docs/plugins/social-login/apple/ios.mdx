Mari kita uraikan apa yang Anda perlukan untuk mengatur login Apple di IOS.

<Steps>
1. Konfigurasi kemampuan aplikasi Anda.

    Untuk melakukan ini, silakan buka Xcode, klik pada `App`
    ![App XCode](/social-login-assets/app-xcode.png)

2. Pastikan Anda memilih target yang benar.

    ![XCode App Target](/social-login-assets/xcode-app-target.png)

3. Silakan pastikan Anda menambahkan kemampuan `Sign in with Apple`.

    ![App XCode Add Capability](/social-login-assets/app-xcode-add-cap.png)

    ![App XCode Add Capability](/social-login-assets/sign-with-apple-cap.png)

    :::caution
    Jika Anda tidak melihat kemampuan `Sign in with Apple`, konfigurasikan [Account & Organizational Data Sharing](https://developer.apple.com/account/resources/services/cwa/configure/)
    :::

4. Inisialisasi Login Apple di aplikasi Anda.

    :::note
    Saya menggunakan Vue sebagai framework saya, implementasi pastinya akan bervariasi tergantung pada framework pilihan Anda
    :::

    ```ts
    //  onMounted adalah spesifik vue
    onMounted(() => {
    SocialLogin.initialize({
        apple: {}
    })
    });
    ```

5. Buat tombol yang akan memulai proses login.

    Tombol tersebut harus memanggil fungsi berikut:

    ```ts
    async function loginApple() {
    const res = await SocialLogin.login({
        provider: 'apple',
        options: {}
    })
    ```

6. Jalankan aplikasi Anda di perangkat **<u>FISIK</u>** dan uji.

    Jika Anda mengikuti langkah-langkahnya dengan cermat, Anda akan melihat layar berikut setelah mengklik tombol Anda.

    ![Apple Sign In prompt on iOS device](/social-login-assets/apple-sign-in-ios-final.png)
</Steps>

Itu saja! Anda sudah siap.
