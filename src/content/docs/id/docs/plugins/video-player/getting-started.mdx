## Penggunaan Dasar

```typescript
import { CapacitorVideoPlayer } from '@capgo/capacitor-video-player';

// Initialize a video player
const playVideo = async () => {
  const result = await CapacitorVideoPlayer.initPlayer({
    mode: 'fullscreen',
    url: 'https://example.com/video.mp4',
    playerId: 'myPlayer',
    componentTag: 'div'
  });

  console.log('Player initialized:', result);
};

// Play the video
await CapacitorVideoPlayer.play({ playerId: 'myPlayer' });

// Pause the video
await CapacitorVideoPlayer.pause({ playerId: 'myPlayer' });

// Get current time
const { value } = await CapacitorVideoPlayer.getCurrentTime({ playerId: 'myPlayer' });
console.log('Current time:', value);

// Seek to position
await CapacitorVideoPlayer.setCurrentTime({
  playerId: 'myPlayer',
  seektime: 30 // seconds
});

// Set volume (0-1)
await CapacitorVideoPlayer.setVolume({
  playerId: 'myPlayer',
  volume: 0.5
});
```

## API Referensi

### initPlayer (pilihan)

Inisialisasi instance pemutar video.

```typescript
await CapacitorVideoPlayer.initPlayer({
  mode: 'fullscreen', // or 'embedded'
  url: 'https://example.com/video.mp4',
  playerId: 'player1',
  subtitle: 'https://example.com/subtitles.vtt',
  language: 'en',
  rate: 1.0,
  exitOnEnd: true,
  loopOnEnd: false,
  pipEnabled: true,
  showControls: true
});
```

### bermain (pilihan)

Putar videonya.

```typescript
await CapacitorVideoPlayer.play({ playerId: 'player1' });
```

### jeda (pilihan)

Jeda videonya.

```typescript
await CapacitorVideoPlayer.pause({ playerId: 'player1' });
```

### dapatkanDurasi(pilihan)

Dapatkan durasi video dalam hitungan detik.

```typescript
const { value } = await CapacitorVideoPlayer.getDuration({ playerId: 'player1' });
console.log('Duration:', value, 'seconds');
```

### dapatkanWaktu Saat Ini(pilihan)

Dapatkan posisi pemutaran saat ini dalam hitungan detik.

```typescript
const { value } = await CapacitorVideoPlayer.getCurrentTime({ playerId: 'player1' });
```

### setWaktu Saat Ini(pilihan)

Carilah waktu tertentu.

```typescript
await CapacitorVideoPlayer.setCurrentTime({
  playerId: 'player1',
  seektime: 60 // seconds
});
```

### setVolume(pilihan)

Atur volume (0,0 hingga 1,0).

```typescript
await CapacitorVideoPlayer.setVolume({
  playerId: 'player1',
  volume: 0.8
});
```

### dapatkanVolume(pilihan)

Dapatkan volume saat ini.

```typescript
const { value } = await CapacitorVideoPlayer.getVolume({ playerId: 'player1' });
```

### setMuted (pilihan)

Bisukan atau suarakan video.

```typescript
await CapacitorVideoPlayer.setMuted({
  playerId: 'player1',
  muted: true
});
```

### setRate(pilihan)

Atur kecepatan pemutaran.

```typescript
await CapacitorVideoPlayer.setRate({
  playerId: 'player1',
  rate: 1.5 // 1.5x speed
});
```

### hentikanSemua Pemain()

Hentikan semua pemain aktif.

```typescript
await CapacitorVideoPlayer.stopAllPlayers();
```

### keluarPemain()

Keluar dari pemutar video.

```typescript
await CapacitorVideoPlayer.exitPlayer();
```

## Contoh Lengkap

```typescript
import { CapacitorVideoPlayer } from '@capgo/capacitor-video-player';

export class VideoPlayerService {
  private playerId = 'mainPlayer';

  async initializePlayer(videoUrl: string, subtitleUrl?: string) {
    try {
      const result = await CapacitorVideoPlayer.initPlayer({
        mode: 'fullscreen',
        url: videoUrl,
        playerId: this.playerId,
        subtitle: subtitleUrl,
        language: 'en',
        rate: 1.0,
        exitOnEnd: true,
        loopOnEnd: false,
        pipEnabled: true,
        bkmodeEnabled: true,
        showControls: true,
        displayMode: 'all'
      });

      console.log('Player initialized:', result);
      return result;
    } catch (error) {
      console.error('Failed to initialize player:', error);
      throw error;
    }
  }

  async togglePlayPause() {
    const { value: isPlaying } = await CapacitorVideoPlayer.isPlaying({
      playerId: this.playerId
    });

    if (isPlaying) {
      await CapacitorVideoPlayer.pause({ playerId: this.playerId });
    } else {
      await CapacitorVideoPlayer.play({ playerId: this.playerId });
    }
  }

  async seekForward(seconds: number = 10) {
    const { value: currentTime } = await CapacitorVideoPlayer.getCurrentTime({
      playerId: this.playerId
    });

    await CapacitorVideoPlayer.setCurrentTime({
      playerId: this.playerId,
      seektime: currentTime + seconds
    });
  }

  async seekBackward(seconds: number = 10) {
    const { value: currentTime } = await CapacitorVideoPlayer.getCurrentTime({
      playerId: this.playerId
    });

    await CapacitorVideoPlayer.setCurrentTime({
      playerId: this.playerId,
      seektime: Math.max(0, currentTime - seconds)
    });
  }

  async setPlaybackSpeed(speed: number) {
    await CapacitorVideoPlayer.setRate({
      playerId: this.playerId,
      rate: speed
    });
  }

  async getProgress() {
    const { value: currentTime } = await CapacitorVideoPlayer.getCurrentTime({
      playerId: this.playerId
    });
    const { value: duration } = await CapacitorVideoPlayer.getDuration({
      playerId: this.playerId
    });

    return {
      currentTime,
      duration,
      percentage: (currentTime / duration) * 100
    };
  }

  async cleanup() {
    await CapacitorVideoPlayer.stopAllPlayers();
  }
}
```

## Catatan Platform

### iOS
- Membutuhkan iOS 10.0+
- Menggunakan AVPlayer asli
- Mendukung Gambar-dalam-Gambar
- Pemutaran latar belakang didukung

### Android
- Membutuhkan Android 5.0 (API 21)+
- Menggunakan ExoPlayer
- Mendukung Chromecast
- Warna aksen khusus tersedia

### Web
- Menggunakan pemutar video HTML5
- Hanya mode tertanam
- Fitur asli terbatas

## Praktik Terbaik

1. **Bersihkan pemain**: Selalu hentikan pemain setelah selesai
2. **Menangani kesalahan**: Menggabungkan panggilan pemain dalam try-catch
3. **ID pemain unik**: Gunakan ID unik untuk beberapa pemain
4. **Periksa status pemutaran**: Verifikasi status sebelum pengoperasian
5. **Manajemen sumber daya**: Lepaskan pemain untuk mengosongkan memori
