Anda mungkin bertanya-tanya apakah plugin ini akan tersedia untuk Cordova.

Kami telah memulai repositori R&D untuk itu, tetapi ini adalah jumlah pekerjaan yang sangat besar.

## Masalah

Kami tahu kami bisa melakukannya tetapi untuk itu, kami harus membaca semua kode dari codebase Cordova seperti yang kami lakukan untuk Capacitor, untuk memahami cara membuatnya bekerja dengan fitur Capgo.

Versi Android lebih mudah dilakukan karena keduanya menggunakan Java, tetapi iOS memerlukan penulisan ulang penuh karena Swift masih belum didukung dengan baik di Cordova

## Solusi

Sementara itu, inilah yang bisa Anda lakukan:

* [Dukung kami](https://github.com/sponsors/cap-go) di GitHub dan kami dapat memprioritaskannya.
Ini akan membutuhkan setidaknya 1 bulan kerja.
* Pekerjakan kami sebagai [Konsultan](https://capgo.app/consulting/), kami terbiasa membantu perusahaan besar bermigrasi ke Capacitor, biasanya memakan waktu sebulan, dan [manfaatnya](https://ionic.io/resources/articles/capacitor-vs-cordova-modern-hybrid-app-development) sangat besar untuk tim Anda
