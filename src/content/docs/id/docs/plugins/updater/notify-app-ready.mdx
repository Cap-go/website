Penempatan pemanggilan `CapacitorUpdater.notifyAppReady()` sangat penting untuk alur auto-update yang benar.


Jika tidak dilakukan dengan benar, aplikasi tidak akan dapat memperbarui ke versi terbaru. Setiap versi yang tidak memanggil `notifyAppReady()` dalam waktu 10 detik akan ditandai sebagai tidak valid dan akan diganti dengan versi valid sebelumnya atau versi default (bawaan).


Dalam panduan ini kami akan menunjukkan cara menempatkan pemanggilan `notifyAppReady()` dengan benar di aplikasi Anda untuk memastikan alur auto-update yang benar.
