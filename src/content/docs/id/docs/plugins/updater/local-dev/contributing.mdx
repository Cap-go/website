## Mengapa berkontribusi?

Pertama-tama, terima kasih telah mempertimbangkan untuk berkontribusi ke proyek open source capgo! Orang-orang seperti Andalah yang membuat proyek open source capgo menjadi alat yang luar biasa.

Berikut adalah beberapa alasan mengapa Anda mungkin ingin mempertimbangkan untuk berkontribusi:

* Berkontribusi ke proyek open source capgo adalah cara yang bagus untuk mendapatkan uang dari [berbagai bounty](https://console.algora.io/org/Capgo) yang ditawarkan oleh tim capgo
* Berkontribusi ke proyek open source capgo adalah cara yang bagus untuk menambahkan fitur yang ingin Anda lihat
* Berkontribusi ke proyek open source capgo adalah cara yang bagus untuk memperbaiki bug yang Anda temukan
* [Lisensi utama Capgo](https://github.com/Cap-go/capgo/blob/main/LICENSE) mewajibkan Anda untuk membuka sumber setiap perubahan yang Anda buat padanya. Dengan berkontribusi kode Anda, Anda dapat menjaga perubahan Anda tetap open source dan membiarkan orang lain menggunakannya

## Cara berkontribusi

 * Pertama-tama, Anda perlu fork repositori yang ingin Anda kontribusikan
 * Kedua, Anda perlu commit dan push perubahan Anda ke repositori tersebut
 * Terakhir, Anda perlu membuka pull request
 * Itu saja! Sekarang Anda hanya perlu menunggu tim capgo untuk meninjau PR Anda
