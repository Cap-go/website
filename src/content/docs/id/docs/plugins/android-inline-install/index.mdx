## Ringkasan

Plugin Capacitor Android Inline Install memungkinkan pemicuan overlay Instalasi Inline Google Play untuk aplikasi Android. Plugin ini menyediakan pengalaman instalasi dalam aplikasi yang mulus menggunakan Premium Growth Tools Google, memungkinkan pengguna untuk menginstal aplikasi tanpa meninggalkan aplikasi Anda.
