`npx @capgo/cli@latest init [apikey]`

Perintah ini akan membimbing Anda langkah demi langkah untuk:

- Menambahkan aplikasi Anda ke Capgo
- Menambahkan kode ke aplikasi Anda untuk memvalidasi pembaruan
- Membangun aplikasi Anda
- Mengunggah aplikasi Anda ke Capgo
- Membantu Anda memeriksa apakah pembaruan berfungsi
