## Integrasi CI

Untuk mengotomatiskan pekerjaan Anda, kami merekomendasikan menggunakan GitHub Actions untuk mendorong pembaruan Anda ke Capgo. Lihat [tutorial GitHub Actions](https://capgo.app/blog/automatic-build-and-release-with-github-actions/) kami untuk informasi lebih lanjut.

Jangan lupa untuk mengonfigurasi variabel lingkungan CI Anda dengan kunci API Capgo Anda.
