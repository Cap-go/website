`npx @capgo/cli login [apikey]`

Perintah ini menyimpan kunci API Capgo Anda untuk penggunaan di masa mendatang

:::note
Anda dapat mengganti kunci API yang tersimpan dengan menambahkan `--apikey=<key>` ke perintah apapun
:::

Opsi:

- `--local`: Menyimpan kunci API di repo lokal dan menambahkannya ke `gitignore`
