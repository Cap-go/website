`npx @capgo/cli login [apikey]`

Perintah ini menyimpan API key Capgo Anda untuk penggunaan di masa mendatang

:::note
Anda dapat mengganti API key yang tersimpan dengan menambahkan `--apikey=<key>` ke perintah apapun
:::

Options:

- `--local`: Simpan API key di repo lokal dan tambahkan ke `gitignore`
