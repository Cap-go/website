## Yang Akan Anda Dapatkan

Semua panduan integrasi mencakup:

- **Setup Sederhana**: Konfigurasi dasar untuk memulai dengan cepat
- **Workflow Lanjutan**: Deployment multi-environment dengan staging dan production
- **Testing Branch Fitur**: Deployment otomatis branch fitur ke channel test
- **Praktik Terbaik Keamanan**: Manajemen secret aman dan perlindungan environment
- **Monitoring**: Notifikasi dan logging untuk status deployment

<Aside type="tip">

**Baru dengan CI/CD?** Mulai dengan konfigurasi sederhana untuk platform Anda, kemudian secara bertahap tambahkan fitur yang lebih lanjutan seperti deployment multi-channel dan testing otomatis sesuai kebutuhan Anda.

</Aside>

## Fitur Umum

Setiap integrasi mendukung:

- **Build Otomatis**: Trigger deployment pada perubahan kode
- **Dukungan Multi-Channel**: Deploy ke channel yang berbeda (development, staging, production)
- **Testing Pull Request/Merge Request**: Uji perubahan dalam environment terisolasi
- **Dukungan Enkripsi**: Deployment aman dengan fitur enkripsi Capgo
- **Perlindungan Environment**: Persetujuan manual dan akses terbatas untuk production
- **Notifikasi**: Integrasi Slack, email, dan notifikasi lainnya

## Prasyarat

Sebelum mengatur integrasi apapun, pastikan Anda memiliki:

- Akun Capgo dengan aplikasi yang sudah dikonfigurasi
- Kode sumber aplikasi Anda dalam repositori Git
- Token API Capgo dari [console.capgo.app/apikeys](https://console.capgo.app/apikeys)
- Node.js dan npm/yarn yang dikonfigurasi dalam proyek Anda
