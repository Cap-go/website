Sekarang setelah Anda menyelesaikan panduan quickstart, Anda seharusnya memiliki pemahaman dasar tentang konsep-konsep kunci Capgo! Konsep-konsep kunci yang telah Anda pelajari dalam panduan ini adalah:

<Steps>
1. Menambahkan aplikasi ke akun Capgo Anda

2. Mengintegrasikan Capgo dengan pipeline CI/CD Anda

3. Memicu unggahan bundle ke Capgo pada commit baru

4. Mengkonfigurasi aplikasi Anda untuk mengaktifkan pembaruan langsung dengan SDK Capgo

5. Menerapkan pembaruan langsung ke aplikasi Anda dari dashboard Capgo
</Steps>
