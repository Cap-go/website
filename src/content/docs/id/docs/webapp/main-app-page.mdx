## Apa yang ditampilkan halaman utama?

Pertama, mari kita lihat halaman utama aplikasi:

<figure><img src="/main-app-pagewebp" alt="Main page screeshot" /><figcaption></figcaption></figure>

Mari kita lihat lebih detail

Pertama statistik. Statistik sedang dalam proses pengerjaan ulang sehingga belum bisa didokumentasikan saat ini. Kami akan menambahkan bagian statistik ketika sudah sepenuhnya beroperasi.

Kedua, mari kita lihat semua menu yang tersedia dan fungsinya

<figure><img src="/app-multiple-sectionswebp" alt="Multiple sections screeshot" /><figcaption></figcaption></figure>

Tombol pertama akan membawa Anda ke halaman [channels](/docs/webapp/channels/) Di sana Anda dapat membuat/menghapus/mengkonfigurasi channel aplikasi\
Tombol kedua akan membawa Anda ke halaman [bundles](/docs/webapp/bundles/) Di sana Anda dapat melihat semua versi aplikasi
Tombol kedua akan membawa Anda ke halaman [device](/docs/webapp/devices/) Di sana Anda dapat melihat semua perangkat. Anda juga dapat mengatur pengaturan khusus perangkat dari sana
Tombol kedua akan membawa Anda ke halaman [updates](/docs/webapp/logs/) Di sana Anda dapat melihat statistik detail (log) dan kesalahan yang dialami aplikasi Anda
