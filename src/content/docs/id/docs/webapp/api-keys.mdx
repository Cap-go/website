## Apa yang bisa saya lakukan dengan kunci API?

Kunci API dapat dibuat ulang, dihapus atau ditambahkan yang baru

## Bagaimana melihat semua kunci API?

Anda perlu membuka [halaman kunci API](https://webcapgoapp/dashboard/apikeys/) dan di sana Anda akan melihat semua kunci API Anda

## Bagaimana menambahkan kunci API baru?

Untuk menambahkan kunci API baru klik tombol plus kecil

<figure><img src="/apikeys-addwebp" alt="add apikey" /><figcaption></figcaption></figure>

lalu pilih izin yang ingin Anda berikan ke kunci API baru. Anda dapat memilih:

 - Read - kunci API akan dapat membaca semua data
 - Upload - kunci API akan dapat mengunggah versi baru dari CLI dan membaca
 - All - Kunci API akan dapat melakukan semuanya

<figure><img src="/apikeys-select-permwebp" alt="select perm" /><figcaption></figcaption></figure>

## Bagaimana menghapus kunci API?

Untuk menghapus kunci API klik tombol sampah kecil

<figure><img src="/apikeys-removewebp" alt="remove apikey" /><figcaption></figcaption></figure>

Lalu konfirmasi penghapusan kunci API

## Bagaimana membuat ulang kunci API?

Untuk membuat ulang kunci API klik tombol refresh kecil

<figure><img src="/apikeys-regeneratewebp" alt="generate apikey" /><figcaption></figcaption></figure>

Lalu konfirmasi pembuatan ulang kunci API
