## Apa yang bisa saya lakukan dengan kunci api?

Kunci API dapat dibuat ulang, dihapus atau bisa ditambahkan yang baru

## Bagaimana cara melihat semua kunci api?

Anda perlu mengunjungi [halaman kunci API](https://webcapgo.app/dashboard/apikeys/) dan di sana Anda akan melihat semua kunci api Anda

## Bagaimana cara menambahkan kunci api baru?

Untuk menambahkan kunci API baru klik pada tombol plus kecil

<figure><img src="/apikeys-add.webp" alt="add apikey" /><figcaption></figcaption></figure>

dan kemudian pilih izin yang ingin Anda berikan ke kunci API baru. Anda dapat memilih:

 - Read - kunci API akan dapat membaca semua data
 - Upload - kunci API akan dapat mengunggah versi baru dari CLI dan membaca
 - All - Kunci API akan dapat melakukan semuanya

<figure><img src="/apikeys-select-perm.webp" alt="select perm" /><figcaption></figcaption></figure>

## Bagaimana cara menghapus kunci api?

Untuk menghapus kunci API klik pada tombol sampah kecil

<figure><img src="/apikeys-remove.webp" alt="remove apikey" /><figcaption></figcaption></figure>

Dan kemudian konfirmasi penghapusan kunci API

## Bagaimana cara membuat ulang kunci api?

Untuk membuat ulang kunci API klik pada tombol refresh kecil

<figure><img src="/apikeys-regenerate.webp" alt="generate apikey" /><figcaption></figcaption></figure>

Dan kemudian konfirmasi pembuatan ulang kunci API
