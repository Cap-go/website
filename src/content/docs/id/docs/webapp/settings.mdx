## Cara menuju halaman pengaturan

Pertama klik pada **NAMA_DEPAN NAMA_BELAKANG** Dalam kasus saya ini adalah **test Capgo** Dalam kasus Anda, ini akan menjadi nama depan dan nama belakang Anda Kemudian klik pada **Pengaturan**

<figure><img src="/settings-gowebp" alt="open settings" /><figcaption></figcaption></figure>

## Mengubah pengaturan pengguna

Untuk mengubah pengaturan pengguna, Anda cukup mengisi formulir di akun dan kemudian klik **Perbarui**

<figure><img src="/account-savewebp" alt="save change in account" /><figcaption></figcaption></figure>

Kemudian konfirmasi akan muncul

<figure><img src="/account-updatedwebp" alt="acount updated" /><figcaption></figcaption></figure>

## Mengubah kata sandi

Untuk mengubah kata sandi, buka halaman pengaturan dan klik **Kata Sandi** Kemudian isi formulir dan klik **Perbarui**

<figure><img src="/update-passwdwebp" alt="update password" /><figcaption></figcaption></figure>

Ketika kata sandi tidak mengikuti aturan keamanan kata sandi capgo, Anda akan mendapatkan pesan kesalahan

<figure><img src="/passwd-errorwebp" alt="wrong password" /><figcaption></figcaption></figure>
