## Cara menuju halaman pengaturan

Pertama klik pada **NAMA_DEPAN NAMA_BELAKANG** Dalam kasus saya ini adalah **test Capgo** Dalam kasus Anda, ini akan menjadi nama depan dan nama belakang Anda Kemudian klik pada **Settings**

<figure><img src="/settings-go.webp" alt="open settings" /><figcaption></figcaption></figure>

## Mengubah pengaturan pengguna

Untuk mengubah pengaturan pengguna, Anda cukup mengisi formulir di akun dan kemudian klik **Update**

<figure><img src="/account-save.webp" alt="save change in account" /><figcaption></figcaption></figure>

Kemudian konfirmasi akan muncul

<figure><img src="/account-updated.webp" alt="acount updated" /><figcaption></figcaption></figure>

## Mengubah kata sandi

Untuk mengubah kata sandi, pergi ke halaman pengaturan dan klik **Password** Kemudian isi formulir dan klik **Update**

<figure><img src="/update-passwd.webp" alt="update password" /><figcaption></figcaption></figure>

Ketika kata sandi tidak mengikuti aturan keamanan kata sandi capgo, Anda akan mendapatkan pesan kesalahan

<figure><img src="/passwd-error.webp" alt="wrong password" /><figcaption></figcaption></figure>
