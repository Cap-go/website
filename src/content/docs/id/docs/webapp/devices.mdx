:::note[Penyimpanan Data]
Data perangkat disimpan selama **90 hari** sejak aktivitas perangkat terakhir. Perangkat yang tidak terhubung ke Capgo dalam periode ini akan secara otomatis dihapus dari daftar. Perangkat aktif terus dilacak saat mereka memeriksa pembaruan.
:::

## Tampilkan daftar semua perangkat
Pertama, mari kita lihat halaman perangkat. Anda dapat mengaksesnya dengan [mengklik aplikasi Anda](/docs/webapp/main-page) dan kemudian [mengklik tab perangkat](/docs/webapp/main-app-page).

<figure><img style="margin-left: auto; margin-right: auto" src="/devices.webp" alt="daftar perangkat" /><figcaption></figcaption></figure>

1. **Tab Perangkat** - Klik tab ini di bilah navigasi untuk mengakses halaman perangkat
2. **Filter** - Klik untuk memfilter perangkat berdasarkan:
   - **Override** - Menampilkan hanya perangkat yang memiliki [channel kustom](/docs/plugin/api/#setchannel) atau versi kustom
   - **ID Kustom** - Menampilkan hanya perangkat dengan identifier kustom yang diatur

## Mengkonfigurasi perangkat

Untuk mengkonfigurasi perangkat tertentu, klik perangkat tersebut di tabel dan Anda akan melihat tampilan seperti ini:

<figure><img style="margin-left: auto; margin-right: auto" src="/device-specific.webp" alt="tampilkan satu perangkat" /><figcaption></figcaption></figure>

1. **Information** - Melihat detail perangkat termasuk ID Perangkat, Pembaruan terakhir, Platform, Versi plugin, Versi, Versi builtin, Versi OS, dan apakah ini aplikasi Produksi
2. **Deployments** - Melihat riwayat deployment untuk perangkat ini
3. **Logs** - Melihat log yang terkait dengan perangkat ini
4. **Override channel** - Mengganti channel untuk perangkat spesifik ini. Jika diatur, perangkat ini tidak akan menggunakan channel publik (default) dan hanya akan menerima pembaruan dari channel yang dipilih

### ID Kustom

`ID Kustom` adalah bidang yang sangat berguna yang membantu Anda menghubungkan perangkat ke ID yang dapat Anda kenali untuk pengguna Anda sendiri.

Gunakan untuk mengidentifikasi dengan mudah perangkat mana milik pengguna mana.

ID Kustom hanya dapat diatur dari perangkat itu sendiri menggunakan `CapacitorUpdater.setCustomId({ customId: user });`
