## Pilih antara Otomatis dan Manual

:::danger
‚ö†Ô∏è Pembaruan OTA(Over the Air) hanya bekerja pada perubahan HTML, CSS, JS\
Saat Anda memperbarui kode native (plugin Capacitor), Anda harus mengirim ulang ke app store
:::

Dalam mode otomatis, sebagian logika ditangani oleh kode Native, pembaruan diputuskan di sisi server, ini lebih aman dan memungkinkan pembaruan yang lebih terperinci, penyebaran parsial ke satu perangkat atau grup dan lainnya

Dalam mode manual, semua logika ditangani oleh JS, ada hal baik dan buruk di kedua skenario

### Persiapkan bundle Anda

Untuk mengirim pembaruan ke aplikasi Anda, Anda perlu mengzipnya
Cara terbaik untuk memastikan zip Anda baik adalah menggunakan Capgo CLI untuk kompresi

`npx @capgo/cli@latest bundle zip` akan membuat zip Anda siap diunggah ke backend Anda
