## So gelangen Sie zur Einstellungsseite

Zuerst klicken Sie auf **VORNAME NACHNAME** In meinem Fall ist das **test Capgo** In Ihrem Fall wird dies Ihr Vorname und dann Ihr Nachname sein. Dann klicken Sie auf **Einstellungen**

<figure><img src="/settings-gowebp" alt="Einstellungen öffnen" /><figcaption></figcaption></figure>

## Ändern der Benutzereinstellungen

Um eine Benutzereinstellung zu ändern, füllen Sie einfach das Formular im Konto aus und klicken Sie dann auf **Aktualisieren**

<figure><img src="/account-savewebp" alt="Änderung im Konto speichern" /><figcaption></figcaption></figure>

Dann sollte eine Bestätigung erscheinen

<figure><img src="/account-updatedwebp" alt="Konto aktualisiert" /><figcaption></figcaption></figure>

## Passwort ändern

Um das Passwort zu ändern, gehen Sie zur Einstellungsseite und klicken Sie auf **Passwort** Füllen Sie dann das Formular aus und klicken Sie auf **Aktualisieren**

<figure><img src="/update-passwdwebp" alt="Passwort aktualisieren" /><figcaption></figcaption></figure>

Wenn das Passwort nicht den Sicherheitsregeln von Capgo für Passwörter entspricht, erhalten Sie eine Fehlermeldung

<figure><img src="/passwd-errorwebp" alt="falsches Passwort" /><figcaption></figcaption></figure>
