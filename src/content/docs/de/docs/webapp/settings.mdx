## So gelangen Sie zur Einstellungsseite

Klicken Sie zuerst auf **VORNAME NACHNAME**. In meinem Fall ist das **test Capgo**. In Ihrem Fall wird das Ihr Vorname und dann Ihr Nachname sein. Klicken Sie dann auf **Einstellungen**

<figure><img src="/settings-go.webp" alt="open settings" /><figcaption></figcaption></figure>

## Benutzereinstellungen ändern

Um Benutzereinstellungen zu ändern, füllen Sie einfach das Formular im Konto aus und klicken Sie dann auf **Aktualisieren**

<figure><img src="/account-save.webp" alt="save change in account" /><figcaption></figcaption></figure>

Dann sollte eine Bestätigung erscheinen

<figure><img src="/account-updated.webp" alt="acount updated" /><figcaption></figcaption></figure>

## Passwort ändern

Um das Passwort zu ändern, gehen Sie zur Einstellungsseite und klicken Sie auf **Passwort**. Füllen Sie dann das Formular aus und klicken Sie auf **Aktualisieren**

<figure><img src="/update-passwd.webp" alt="update password" /><figcaption></figcaption></figure>

Wenn das Passwort nicht den Capgo-Passwort-Sicherheitsregeln entspricht, erhalten Sie eine Fehlermeldung

<figure><img src="/passwd-error.webp" alt="wrong password" /><figcaption></figcaption></figure>
