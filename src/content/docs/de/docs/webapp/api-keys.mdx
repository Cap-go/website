## Was kann ich mit den API-Schlüsseln machen?

Ein API-Schlüssel kann regeneriert, entfernt oder ein neuer hinzugefügt werden.

## Wie sehe ich alle API-Schlüssel?

Sie müssen zur [API-Schlüssel-Seite](https://webcapgoapp/dashboard/apikeys/) gehen und dort sehen Sie alle Ihre API-Schlüssel.

## Wie füge ich einen neuen API-Schlüssel hinzu?

Um einen neuen API-Schlüssel hinzuzufügen, klicken Sie auf die kleine Plus-Taste.

<figure><img src="/apikeys-addwebp" alt="API-Schlüssel hinzufügen" /><figcaption></figcaption></figure>

Wählen Sie dann die Berechtigungen aus, die Sie dem neuen API-Schlüssel geben möchten. Sie können auswählen:

 - 1. Lesen - der API-Schlüssel kann alle Daten lesen
 - 2. Hochladen - der API-Schlüssel kann neue Versionen über die CLI hochladen und lesen
 - 3. Alle - der API-Schlüssel kann alles tun

<figure><img src="/apikeys-select-permwebp" alt="Berechtigung auswählen" /><figcaption></figcaption></figure>

## Wie entferne ich einen API-Schlüssel?

Um einen API-Schlüssel zu entfernen, klicken Sie auf die kleine Mülltaste.

<figure><img src="/apikeys-removewebp" alt="API-Schlüssel entfernen" /><figcaption></figcaption></figure>

Bestätigen Sie dann die Entfernung des API-Schlüssels.

## Wie regeneriere ich einen API-Schlüssel?

Um einen API-Schlüssel zu regenerieren, klicken Sie auf die kleine Aktualisierungstaste.

<figure><img src="/apikeys-regeneratewebp" alt="API-Schlüssel generieren" /><figcaption></figcaption></figure>

Bestätigen Sie dann die Regenerierung des API-Schlüssels.
