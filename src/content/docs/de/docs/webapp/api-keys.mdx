## Was kann ich mit den API-Schlüsseln tun?

Ein API-Schlüssel kann neu generiert, entfernt oder ein neuer hinzugefügt werden

## Wie kann ich alle API-Schlüssel sehen?

Sie müssen zur [API-Schlüssel-Seite](https://web.capgo.app/dashboard/apikeys/) gehen und dort sehen Sie alle Ihre API-Schlüssel

## Wie füge ich einen neuen API-Schlüssel hinzu?

Um einen neuen API-Schlüssel hinzuzufügen, klicken Sie auf den kleinen Plus-Button

<figure><img src="/apikeys-add.webp" alt="add apikey" /><figcaption></figcaption></figure>

und wählen Sie dann die Berechtigungen aus, die Sie dem neuen API-Schlüssel geben möchten. Sie können auswählen:

 - Lesen - der API-Schlüssel kann alle Daten lesen
 - Upload - der API-Schlüssel kann neue Versionen über die CLI hochladen und lesen
 - Alles - der API-Schlüssel kann alles tun

<figure><img src="/apikeys-select-perm.webp" alt="select perm" /><figcaption></figcaption></figure>

## Wie entferne ich einen API-Schlüssel?

Um einen API-Schlüssel zu entfernen, klicken Sie auf den kleinen Papierkorb-Button

<figure><img src="/apikeys-remove.webp" alt="remove apikey" /><figcaption></figcaption></figure>

Und bestätigen Sie dann das Entfernen des API-Schlüssels

## Wie generiere ich einen API-Schlüssel neu?

Um einen API-Schlüssel neu zu generieren, klicken Sie auf den kleinen Aktualisieren-Button

<figure><img src="/apikeys-regenerate.webp" alt="generate apikey" /><figcaption></figcaption></figure>

Und bestätigen Sie dann die Neugenerierung des API-Schlüssels
