## Was zeigt die Hauptseite?

Zuerst werfen wir einen Blick auf die Hauptseite der App:

<figure><img src="/main-app-pagewebp" alt="Screenshot der Hauptseite" /><figcaption></figcaption></figure>

Lassen Sie uns das genauer ansehen.

1. Die Statistiken Die Statistiken werden derzeit überarbeitet, daher können sie zu diesem Zeitpunkt nicht dokumentiert werden. Wir werden einen Statistikbereich hinzufügen, wenn sie vollständig betriebsbereit sind.

2. Lassen Sie uns alle verfügbaren Menüs und deren Funktionen ansehen.

<figure><img src="/app-multiple-sectionswebp" alt="Screenshot mehrerer Abschnitte" /><figcaption></figcaption></figure>

Der erste Button bringt Sie zur [Kanälen](/docs/webapp/channels/) Seite. Dort können Sie die Kanäle der App erstellen/löschen/konfigurieren.  
Der zweite Button bringt Sie zur [Bundles](/docs/webapp/bundles/) Seite. Dort können Sie alle Versionen der App sehen.  
Der dritte Button bringt Sie zur [Gerät](/docs/webapp/devices/) Seite. Dort können Sie alle Geräte sehen. Außerdem können Sie spezifische Überschreibungen für Geräte von dort aus festlegen.  
Der vierte Button bringt Sie zur [Updates](/docs/webapp/logs/) Seite. Dort können Sie die detaillierten Statistiken (Protokolle) und Fehler sehen, die Ihre App erlebt.
