## Nur Entwicklergeräte anzeigen
Möglicherweise möchten Sie nur die überschriebenen Geräte anzeigen (Geräte, die einen [benutzerdefinierten Kanal](docs/plugin/api/#setchannel) oder eine benutzerdefinierte Version haben)
Klicken Sie dazu auf `Filter` und dann auf `Override`. Es sollte dann etwa so aussehen:

## Ein Gerät konfigurieren

Um ein bestimmtes Gerät zu konfigurieren, klicken Sie in der Tabelle darauf. Sie sollten dann etwa Folgendes sehen:

<figure><img style="margin-left: auto; margin-right: auto" src="/device-specificwebp" alt="show one device" /><figcaption></figcaption></figure>

Zunächst die `Custom ID`. Diese ist nicht wirklich nützlich und macht derzeit nichts. Sie können sie ignorieren.

Als Nächstes die erzwungene Version. Wenn diese eingestellt ist, wird dieses Gerät **IMMER** diese Version erhalten. Sie hat Vorrang vor dem Kanal.

Zuletzt der benutzerdefinierte Kanal. Wenn dieser eingestellt ist, wird dieses Gerät sich nicht um den öffentlichen (Standard-) Kanal kümmern und stattdessen nur diesen verwenden.
