## Was zeigt die Hauptseite?

Zuerst werfen wir einen Blick auf die Hauptseite:
<figure><img src="/main-pagewebp" alt="Screenshot der Hauptseite" /><figcaption></figcaption></figure>

Lassen Sie uns am Anfang beginnen. Das erste, was Sie sehen, sind die **Statistiken**. Sie enthalten appspezifische Statistiken. Diese Aktualisierungen hängen von der Nutzung der App ab.

Das zweite, was Sie sehen, ist die Liste aller verfügbaren Apps. Diese wird sich ändern, abhängig davon, wie viele Apps Sie erstellt haben. Sie können auf jede App klicken, um weitere Details darüber zu sehen.

Das dritte, was Sie sehen, ist die **API-Schlüssel**-Schaltfläche. Diese Schaltfläche führt Sie zur [API-Schlüssel](/docs/webapp/api-keys/) Seite.

Das nächste, was Sie sehen, ist die **Test Capgo**-Schaltfläche. Diese Schaltfläche wird sich abhängig von Ihrem Vorname und Nachnamen ändern und Sie zur [Einstellungen](/docs/webapp/settings/) oder Support-Seite führen.

Zuletzt sehen Sie eine Schaltfläche, um eine weitere App hinzuzufügen. Diese Schaltfläche führt Sie zu einer Seite, auf der Sie eine neue App zu Capgo hinzufügen können.
