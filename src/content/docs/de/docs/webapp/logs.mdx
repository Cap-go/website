## Alle Protokolle anzeigen
Zuerst schauen wir uns die Protokollseite an. Sie können darauf zugreifen, indem Sie [auf Ihre App klicken](/docs/webapp/main-page) und dann [auf die Registerkarte „Updates“ klicken](/docs/webapp/main-app-page).
Von dort sollten Sie eine Seite sehen, die so aussieht:

## Weitere Details zu einem Protokoll erhalten

Wenn Sie auf ein Protokoll klicken, gelangen Sie zu einer [Seite der Geräte](/docs/webapp/devices/)
