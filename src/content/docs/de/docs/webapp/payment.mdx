## Worum geht es hier?

Diese Seite soll einige Fragen zum Zahlungssystem in capgo beantworten.

#### F: Wie kann ich meinen capgo-Plan upgraden?
A: Sie können Ihren capgo-Plan upgraden, indem Sie zu [den Einstellungen](/docs/webapp/settings/#how-to-get-to-the-settings-page) gehen und auf die Schaltfläche **Pläne** klicken.

<figure><img src="/plans-buttonwebp" alt="Wählen Sie den Plan" /><figcaption></figcaption></figure>

Dann können Sie den Plan auswählen, der am besten zu Ihren Bedürfnissen passt, und auf **Abonnieren** klicken.

<figure><img src="/plans-subscribewebp" alt="Plan abonnieren" /><figcaption></figcaption></figure>

Dann öffnet sich eine Stripe-Seite. Dort können Sie Ihre Zahlungsinformationen eingeben.

<figure><img src="/plans-stripewebp" alt="Stripe-Portal" /><figcaption></figcaption></figure>

#### F: Sind Zahlungen sicher?
A: Ja, Zahlungen werden vollständig von Stripe verwaltet. Capgo hat niemals Zugriff auf Ihre Kreditkartendetails. Stripe nimmt die Sicherheit sehr ernst. [Erfahren Sie mehr über die Sicherheitsrichtlinien von Stripe](https://stripecom/docs/security/)

#### F: Wird capgo meinen Plan automatisch upgraden, wenn ich das Limit überschreite?
A: Nein, capgo wird Ihren Plan niemals ändern.

#### F: Wird capgo mir eine E-Mail senden, wenn mein Plan nahe an seinen Grenzen ist?
A: Ja, capgo wird Ihnen eine E-Mail senden, in der Sie über die Nutzung informiert werden.

#### F: Wird der Plan, den ich kaufe, die Organisationen beeinflussen, zu denen ich eingeladen bin?
A: Nein, der Plan wird nur die Organisation betreffen, die Sie derzeit ausgewählt haben. Bitte beziehen Sie sich auf [die Dokumentation zur Organisation](/docs/webapp/organization-system/#q-how-is-billing-done-within-orgs).

#### F: Was, wenn ich einen maßgeschneiderteren Plan benötige?
A: [Bitte kontaktieren Sie den Support von capgo direkt](/docs/getting-help#support-by-chat).

#### F: Wie sieht die Rückerstattungsrichtlinie für capgo aus?
A: Eine Rückerstattungsrichtlinie finden Sie [hier](https://capgoapp/return/).
