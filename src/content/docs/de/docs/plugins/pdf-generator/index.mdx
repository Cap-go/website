import { Card, CardGrid } from '@astrojs/starlight/components';

Konvertieren Sie HTML oder entfernte Seiten in teilbare PDFs mit Capgos nativem Renderer für iOS und Android.

<CardGrid stagger>
  <Card title="HTML zu PDF" icon="pencil">
    HTML-Vorlagen eingeben und in Sekunden ein teilbares PDF-Dokument erhalten.
  </Card>
  <Card title="Entfernte Erfassung" icon="rocket">
    Jede zugängliche URL erfassen und das generierte PDF speichern oder verteilen.
  </Card>
  <Card title="Flexible Ausgaben" icon="star">
    Base64-Daten zurückgeben oder das native Teilen-Sheet je nach Workflow auslösen.
  </Card>
  <Card title="Dokumentsteuerung" icon="setting">
    Ausrichtung, Seitengröße und Dateinamen für jedes Rendering festlegen.
  </Card>
</CardGrid>

Gehen Sie zum Leitfaden für die ersten Schritte für Plattformhinweise und Verwendungsbeispiele.
