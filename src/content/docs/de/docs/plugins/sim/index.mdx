import { Card, CardGrid } from '@astrojs/starlight/components';

Verwenden Sie Capgo SIM, um Betreiberdaten, Slot-Indizes und Berechtigungsstatus abzufragen, während Sie in Capacitor bleiben.

<CardGrid stagger>
  <Card title="Multi-SIM-Unterstützung" icon="rocket">
    Zählen Sie jede installierte SIM zusammen mit Slot-Indizes und Abonnement-IDs auf.
  </Card>
  <Card title="Berechtigungshelfer" icon="approve-check">
    Überprüfen und fordern Sie die auf Android erforderliche Telefonieberechtigung an.
  </Card>
  <Card title="Konsistentes Schema" icon="star">
    Arbeiten Sie mit normalisierten Betreiber-Feldern wie MCC, MNC, ISO-Land und Telefonnummern.
  </Card>
  <Card title="iOS-bewusst" icon="pencil">
    Behandeln Sie Plattformpecüliarheiten wie begrenzte Betreiberdaten auf iOS 16.4+.
  </Card>
</CardGrid>

Schauen Sie sich den Leitfaden für Erste Schritte an, um Berechtigungen und Verbrauchsmuster zu integrieren.
