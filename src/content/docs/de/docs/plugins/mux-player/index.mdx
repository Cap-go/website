import { Card, CardGrid } from '@astrojs/starlight/components';

Bieten Sie Premium-Wiedergabe, indem Sie die offiziellen Mux-nativen SDKs in eine einfache Capacitor-Schnittstelle einbinden.

<CardGrid stagger>
  <Card title="Native SDK-Erfahrung" icon="rocket">
    Verwenden Sie denselben Wiedergabe-Stack wie die Mux Mobile SDKs mit adaptiver Bitrate und DRM.
  </Card>
  <Card title="Signierte Wiedergabe" icon="approve-check">
    Stellen Sie Wiedergabe- und DRM-Token für sichere Bereitstellung geschützter Streams bereit.
  </Card>
  <Card title="Einheitliche API" icon="pencil">
    Starten Sie die Wiedergabe, schließen Sie den Player und hören Sie auf Ereignisse mit einer TypeScript-API.
  </Card>
  <Card title="Analytics-bereit" icon="star">
    Übergeben Sie Umgebungsschlüssel und Playernamen, um Mux Data-Telemetrie automatisch zu speisen.
  </Card>
</CardGrid>

Der Leitfaden für die ersten Schritte führt durch das Einrichten der nativen SDKs, die Handhabung von Token und die Reaktion auf Player-Ereignisse.
