import { Card, CardGrid } from '@astrojs/starlight/components';

Das Media Session Plugin verbindet Capacitor mit der Media Session API, damit sich Ihre Audio- und Video-Player überall nativ anfühlen.

<CardGrid stagger>
  <Card title="Umfangreiche Metadaten" icon="star">
    Stellen Sie Album-Cover, Künstler und Titel für Sperrbildschirm- und Benachrichtigungssteuerungen bereit.
  </Card>
  <Card title="Wiedergabestatus-Synchronisation" icon="rocket">
    Halten Sie Play/Pause-Tasten und Fortschrittsanzeigen mit Ihrem Player synchron.
  </Card>
  <Card title="Action-Callbacks" icon="pencil">
    Behandeln Sie Headset-Tasten und OS-Medienaktionen aus Ihrem Capacitor-Code.
  </Card>
  <Card title="Positionsmeldung" icon="download">
    Übertragen Sie Dauer, Position und Rate, damit Scrubbing flüssig funktioniert.
  </Card>
</CardGrid>

Gehen Sie zum Leitfaden für den Einstieg, um Metadaten-Updates in Ihren Player-Lebenszyklus einzubinden.
