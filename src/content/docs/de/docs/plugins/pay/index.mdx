import { Card, CardGrid } from '@astrojs/starlight/components';

Capgo Pay verbindet die neuesten Apple Pay- und Google Pay-SDKs, sodass Sie ein nahtloses Checkout-Erlebnis über alle Geräte hinweg bieten können.

<CardGrid stagger>
  <Card title="Einheitliche API" icon="pencil">
    Rufen Sie `isPayAvailable` und `requestPayment` einmal auf und lassen Sie das Plugin jede Plattform handhaben.
  </Card>
  <Card title="Schnelle Einrichtung" icon="rocket">
    Liefern Sie vollständig native Zahlungssheets ohne manuelles Verdrahten plattformspezifischen Codes.
  </Card>
  <Card title="Tokenisierte Zahlungen" icon="approve-check">
    Erhalten Sie Zahlungstoken, die für Ihre Gateway- oder PSP-Integrationen bereit sind.
  </Card>
  <Card title="Anpassbare Abläufe" icon="star">
    Konfigurieren Sie Einzelposten, Kontaktfelder und Abrechnungsanforderungen, um Ihren Checkout anzupassen.
  </Card>
</CardGrid>

Tauchen Sie in den Leitfaden für die ersten Schritte ein, um Checklisten für die Einrichtung und Beispiel-Payloads für jede Plattform zu erhalten.
