### Einführung

In diesem Leitfaden lernen Sie, wie Sie Apple Login mit Capacitor konfigurieren. Dafür benötigen Sie Folgendes:

- ein Apple Developer-Konto
- Einen Computer mit macOS (nur iOS)
- Xcode installiert (nur iOS)
- Ein benutzerdefiniertes Backend (nur Android)
