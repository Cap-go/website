# Installation in Nuxt 2

Erstelle eine Plugin-Datei `capacitor-updaterjs` im `plugins` Verzeichnis

```js

import { CapacitorUpdater } from '@capgo/capacitor-updater'

export default ({ app }) => {
  if (processclient) {
    windowonNuxtReady(() => {
      CapacitorUpdaternotifyAppReady()
    })
  }
}
```
