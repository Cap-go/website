`npx @capgo/cli@latest init [apikey]`

Dieser Befehl führt Sie Schritt für Schritt durch:

1. Fügen Sie Ihre App zu Capgo hinzu
2. Fügen Sie den Code zu Ihrer App hinzu, um das Update zu validieren
3. Bauen Sie Ihre App
4. Laden Sie Ihre App zu Capgo hoch
5. Helfen Sie Ihnen zu überprüfen, ob das Update funktioniert
