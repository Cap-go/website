`npx @capgo/cli@latest init [apikey]`

Dieser Befehl führt Sie Schritt für Schritt durch:

- Ihre App zu Capgo hinzufügen
- Den Code zur Update-Validierung in Ihre App einbauen
- Ihre App bauen
- Ihre App zu Capgo hochladen
- Ihnen bei der Überprüfung helfen, ob das Update funktioniert
