`npx @capgo/cli login [apikey]`

Dieser Befehl speichert Ihren Capgo API-Schlüssel für zukünftige Verwendung

:::note
Sie können den gespeicherten API-Schlüssel überschreiben, indem Sie `--apikey=<key>` zu einem beliebigen Befehl hinzufügen
:::

Optionen:

- `--local`: Speichert den API-Schlüssel im lokalen Repository und fügt ihn zur `gitignore` hinzu
