`npx @capgo/cli login [apikey]`

Dieser Befehl speichert Ihren Capgo-API-Schlüssel zur späteren Verwendung.

:::note
Sie können den gespeicherten API-Schlüssel überschreiben, indem Sie `--apikey=<key>` an einen beliebigen Befehl übergeben.
:::

Optionen:

- `--local`: Speichern Sie den API-Schlüssel im lokalen Repository und fügen Sie ihn zu `gitignore` hinzu.
