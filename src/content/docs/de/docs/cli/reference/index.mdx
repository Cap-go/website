## CI-Integration

Um Ihre Arbeit zu automatisieren, empfehlen wir die Verwendung von GitHub Actions, um Ihre Updates an Capgo zu übertragen. Schauen Sie sich unser [GitHub Actions-Tutorial](https://capgoapp/blog/automatic-build-and-release-with-github-actions/) für weitere Informationen an.

Vergessen Sie nicht, Ihre CI-Umgebungsvariablen mit Ihrem Capgo-API-Schlüssel zu konfigurieren.
