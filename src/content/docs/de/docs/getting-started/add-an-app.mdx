## Live-Updates sind 3 Schritte entfernt

### Geführte Einrichtung

<Steps>

1. Erstellen Sie Ihr Konto unter [https://capgoapp/register](https://capgoapp/register)
![Anmelde-Screenshot](/signupwebp "Anmelde-Screenshot")

2. Verwenden Sie die Init-Befehle, um loszulegen <Code code={`npx @capgo/cli@latest init [APIKEY]`} lang="bash" /> Sie werden mit einer Reihe von Fragen konfrontiert. Geben Sie die erforderlichen Antworten ein, um die automatische Einrichtung abzuschließen<Aside type="tip">Wenn Sie diese Schritte befolgen, sind Sie im Handumdrehen bereit. Wenn Sie während des Prozesses weitere Unterstützung benötigen, ist unser Support-Team [hier, um zu helfen](mailto:support@capgoapp). Viel Erfolg beim Onboarding!</Aside>

3. Stellen Sie ein Live-Update bereit <LinkCard href="/docs/getting-started/deploy/" title="Stellen Sie ein Live-Update bereit" description="Erfahren Sie, wie Sie ein Live-Update für Ihre App bereitstellen" />

</Steps>

### Manuelle Einrichtung

Falls der init-Befehl für Sie nicht funktioniert, können Sie eine App manuell hinzufügen

<Steps>

1. Verbinden Sie die CLI mit Ihrem Konto: <Code code={`npx @capgo/cli@latest login [APIKEY]`} lang="bash" />

2. Fügen Sie die App mit diesem Befehl zu Ihrem Konto hinzu: <Code code={`npx @capgo/cli@latest app add [APP_NAME]`} lang="bash" />

3. Installieren Sie das Plugin in Ihrer App: <Code code={`npm i @capgo/capacitor-updater`} lang="bash" />

4. Rufen Sie die Init-Methode so früh wie möglich in Ihrer App auf: <Code code={`import { CapacitorUpdater } from '@capgo/capacitor-updater';\nCapacitorUpdaternotifyAppReady();`} lang="typescript" />

5. Stellen Sie ein Live-Update bereit <LinkCard href="/docs/getting-started/deploy/" title="Stellen Sie ein Live-Update bereit" description="Erfahren Sie, wie Sie ein Live-Update für Ihre App bereitstellen" />

</Steps>
