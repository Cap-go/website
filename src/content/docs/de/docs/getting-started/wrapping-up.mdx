Jetzt, wo du den Schnellstartleitfaden abgeschlossen hast, solltest du ein grundlegendes Verständnis der Kernkonzepte von Capgo haben! Die Kernkonzepte, die du in diesem Leitfaden gelernt hast, sind:

<Steps>
1. Eine App zu deinem Capgo-Konto hinzufügen

2. Capgo in deine CI/CD-Pipeline integrieren 

3. Bundles-Uploads zu Capgo bei neuen Commits auslösen

4. Deine App so konfigurieren, dass sie Live-Updates mit dem Capgo SDK ermöglicht

5. Live-Updates von deinem Capgo-Dashboard an deine App bereitstellen
</Steps>
